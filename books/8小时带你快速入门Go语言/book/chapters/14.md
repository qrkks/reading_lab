# 异常处理

## 向上抛

```go
import "fmt"
import "errors"

func div(a, b int) (int, error) {
	if b == 0 {
		// return 0, fmt.Errorf("division by zero")
		return 0, errors.New("division by zero")
		// 包依赖：需要导入 import "errors"
		// 功能：创建一个实现了 error 接口的错误实例
		// 参数字符串 "division by zero"：错误文本描述，后续打印 err 时会输出这段文字
		// 底层原理：error 是 Go 内置接口，只定义一个方法 Error() string；errors.New 返回的结构体自带该方法，满足接口规范
	}
	return a / b, nil
}
func server() (res int, err error) {
	res, err = div(2, 0)
	if err != nil {
		// 把错误向上传递
		return
	}

	// 执行其他的逻辑
	res++
	res += 2
	return
}
func main() {
	res, err := server()
	if err != nil {
		fmt.Println(err)
		return
	}
	fmt.Println(res)
}

// 模拟 div 除法函数，除数为 0 返回错误

main()
```

具名参数

```go
import "fmt"

func add(a, b int) (res int) {
	res = a + b
	return
}

fmt.Println(add(1, 2))
```

```go
import "fmt"

func add(a, b int) (int, error) {
	if a+b == 3 {
		return 0, fmt.Errorf("不能等于 3")
	}
	return a + b, nil
}
func run() {
	result, err := add(1, 2)
	if err != nil {
		fmt.Println("计算失败：", err)
		return
	}

	fmt.Println("结果：", result)
}

// fmt.Println(add(1, 2))

run()
```

## 中断程序

```go
import "fmt"
import "log"

func divide(a, b int) int {
	if b == 0 {
		panic("除数不能为 0")
		// log.Fatalln("错误了")
	}

	return a / b
}

fmt.Println(divide(10, 0))
```

## 恢复程序（异常捕获）

```go
import "fmt"
import "runtime/debug"

func divide(a, b int) int {
	defer func() {
		err := recover()
		fmt.Println(err)
		// debug.PrintStack()
	}()
	if b == 0 {
		panic("除数不能为 0")
	}

	return a / b
}

fmt.Println(divide(10, 0))
```
