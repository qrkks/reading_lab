## 闭包

[go 一般用什么命名法？](../appendices/7.md#go-一般用什么命名法)

```go
import (
	"fmt"
	"time"
)

func awaitAdd(second int) func(...int) {
	time.Sleep(time.Duration(second) * time.Second)
	return func(nums ...int) {
		sum := 0
		for _, j := range nums {
			sum += j
		}
		fmt.Println(sum)
	}
}

awaitAdd(2)(1, 2, 3)
```

## 值传递和引用传递

```go
import "fmt"

var name = "张三"

func value(v string) {
	fmt.Printf("值传递的函数内的 name 内存地址：%p\n", &v)
}
func pointer(p *string) {
	fmt.Printf("指针传递的函数内的 name 内存地址：%p\n", p)
}

fmt.Printf("外部 name 内存地址：%p\n", &name)

value(name)
pointer(&name)
```
