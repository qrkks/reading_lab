通项公式：

```python
import sympy as sp

# 定义符号
p, n, k = sp.symbols("p n k", nonnegative=True, integer=True)

# 第一部分：k 从 0 到 n-1，中途遇红灯停下的所有概率
sum_stop = sp.Sum(p * (1 - p)**k, (k, 0, n - 1))
# 第二部分：全部绿灯走完，没停下的概率
term_all_green = (1 - p)**n

# 总概率和
total = sum_stop + term_all_green

# 打印表达式
print("总概率表达式：")
sp.pprint(total)

# 化简验证恒等于 1
total_simplified = sp.simplify(total.doit())
print("\n化简结果：")
sp.pprint(total_simplified)
# total_simplified
```

### $\sum_{k=0}^{n} P\{X = k\} = \sum_{k=0}^{n} \binom{n}{k} p^k q^{n-k} = (p+q)^n = 1$

$$
(p+q)^n
$$

```python
from sympy import symbols, binomial, expand, summation

n, p, k = symbols("n p k", integer=True, nonnegative=True)
q = 1 - p

# 二项式定理直接展开求和项的闭式
closed_form = (p + q)**n
print("闭式表达式 (p+q)^n：", closed_form)
print("代入 q=1-p 化简：", expand(closed_form.subs(q, 1-p)))
```

$$
\sum_{k=0}^{n} \binom{n}{k} p^k q^{n-k}
$$

```python
from sympy import symbols, binomial, Sum, simplify, And
n = symbols("n", integer=True, nonnegative=True)
p = symbols("p", real=True, nonnegative=True)
k = symbols("k", integer=True)

term = binomial(n, k) * p**k * (1-p)**(n - k)
s = Sum(term, (k, 0, n))
res = s.doit()

# 手动带入二项分布的前提假设 0 ≤ p ≤ 1
final = simplify(res, assumptions=And(0 <= p, p <= 1))
print(final) # 输出 1
```
