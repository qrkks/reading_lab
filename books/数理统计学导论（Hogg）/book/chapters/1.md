## 随机变量

### 随机变量 X 的变化示例

下面用 Python 代码模拟抛两次硬币的场景，把**两个阶段的 `X`** 分别实现出来，你就能直观看到它们的区别。

#### 场景设定

- 原始试验：抛 2 次硬币，结果用字符串表示（`H`=正面，`T`=反面）
- 随机变量定义：`X` = 正面出现的次数

#### 1. 阶段一：原始随机变量（函数映射）

此时的 `X` 是**从原始样本空间到整数的函数**，定义域是所有抛硬币的结果，值域是 `{0,1,2}`。

```python
## 原始样本空间（所有可能的抛硬币结果）
omega = ["HH", "HT", "TH", "TT"]


## 阶段一的 X：从结果映射到正面次数的函数
def X_original(outcome: str) -> int:
    """输入抛硬币结果（如"HT"），输出正面次数"""
    return outcome.count("H")


## 测试：对每个结果调用 X_original
for outcome in omega:
    count = X_original(outcome)
    print(f"结果 {outcome} → 正面次数 = {count}")
```

运行输出：

```text
结果 HH → 正面次数 = 2
结果 HT → 正面次数 = 1
结果 TH → 正面次数 = 1
结果 TT → 正面次数 = 0
```

此时的 `X_original` 就是我们之前说的“函数本身”，它的定义域是 `omega`，值域是 `{0,1,2}`。

#### 2. 阶段二：分布视角下的随机变量

此时我们不再关心原始结果，只关注 `X` 的取值和对应的概率，把原来的值域 `D = {0,1,2}` 当成新的样本空间。

```python
from collections import defaultdict

## 阶段二的 X：直接表示取值和概率的分布
## 1. 先计算每个取值的概率（频率）
counts = defaultdict(int)
total = len(omega)
for outcome in omega:
    count = X_original(outcome)
    counts[count] += 1

## 转换为概率分布：{取值：概率}
X_distribution = {k: v / total for k, v in counts.items()}

print("\n阶段二的 X 分布：")
for value, prob in sorted(X_distribution.items()):
    print(f"X = {value} 的概率：{prob}")
```

运行输出：

```text
阶段二的 X 分布：
X = 0 的概率：0.25
X = 1 的概率：0.5
X = 2 的概率：0.25
```

此时的 `X_distribution` 就是我们研究的对象，它的“样本空间”是 `{0,1,2}`，也就是原来的值域。

#### 关键区别总结

| 阶段 | `X` 的含义                   | 定义域/样本空间                         | 值域/取值范围              |
| :--- | :--------------------------- | :-------------------------------------- | :------------------------- |
| 一   | 函数映射（把结果变成数字）   | 原始样本空间 `["HH", "HT", "TH", "TT"]` | `{0,1,2}`                  |
| 二   | 分布模型（直接看数字的概率） | 新样本空间 `{0,1,2}`                    | 它自己（取值就是 `0,1,2`） |

这就是为什么教材里说“它的定义域 D 成为关注的样本空间”——这里的 D 其实是阶段一的值域，在阶段二被当成了新的样本空间，而此时的 `X` 也变成了描述这个空间上概率分布的模型。

### 例 1.2.1

#### 一、基础前提

掷两枚公平骰子，第一枚点数$i$、第二枚点数$j$，样本空间：
$$C=\{(i,j)\mid 1\le i\le6,\ 1\le j\le6\}$$
总共有 $6\times6=36$ 种等可能结果，**每一组 $(i,j)$ 概率都是 $\dfrac{1}{36}$**。

随机变量 $X$ = 两枚骰子点数之和：
$$X(i,j)=i+j$$
$X$ 的取值范围 $D=\{2,3,4,\dots,12\}$。

#### 二、核心计算逻辑

对每一个和 $x$，数出满足 $i+j=x$ 的 $(i,j)$ 组合数量 $N(x)$，则概率质量函数：
$$p_X(x) = \frac{\text{符合条件的组合数 }N(x)}{36}$$

##### 逐个拆解每个$x$的组合数

1. $x=2$：$i+j=2$
   仅 $(1,1)$ → $N=1$，$p_X(2)=\dfrac{1}{36}$
2. $x=3$：$i+j=3$
   $(1,2),(2,1)$ → $N=2$，$p_X(3)=\dfrac{2}{36}$
3. $x=4$：$i+j=4$
   $(1,3),(2,2),(3,1)$ → $N=3$，$p_X(4)=\dfrac{3}{36}$
4. $x=5$：$i+j=5$
   $(1,4),(2,3),(3,2),(4,1)$ → $N=4$，$p_X(5)=\dfrac{4}{36}$
5. $x=6$：$i+j=6$
   $(1,5),(2,4),(3,3),(4,2),(5,1)$ → $N=5$，$p_X(6)=\dfrac{5}{36}$
6. $x=7$：$i+j=7$
   $(1,6),(2,5),(3,4),(4,3),(5,2),(6,1)$ → $N=6$，$p_X(7)=\dfrac{6}{36}$
7. $x=8$：$i+j=8$
   $(2,6),(3,5),(4,4),(5,3),(6,2)$ → $N=5$，$p_X(8)=\dfrac{5}{36}$
8. $x=9$：$i+j=9$
   $(3,6),(4,5),(5,4),(6,3)$ → $N=4$，$p_X(9)=\dfrac{4}{36}$
9. $x=10$：$i+j=10$
   $(4,6),(5,5),(6,4)$ → $N=3$，$p_X(10)=\dfrac{3}{36}$
10. $x=11$：$i+j=11$
    $(5,6),(6,5)$ → $N=2$，$p_X(11)=\dfrac{2}{36}$
11. $x=12$：$i+j=12$
    $(6,6)$ → $N=1$，$p_X(12)=\dfrac{1}{36}$

#### 三、规律总结

1. 组合数先递增到 $x=7$（最大值 6），再对称递减；
2. 左右对称：$x=2$ 和 $x=12$ 数量都是 1，$x=3$ 和 $x=11$ 都是 2……
3. 校验：所有概率分子相加 $1+2+3+4+5+6+5+4+3+2+1=36$，$\dfrac{36}{36}=1$，满足概率归一性。

#### 四、SymPy 代码批量验证

```python
import sympy as sp

#### 枚举所有骰子组合
pairs = [(i, j) for i in range(1, 7) for j in range(1, 7)]
print(pairs)
count = {}
for i, j in pairs:
    s = i + j
    count[s] = count.get(s, 0) + 1

#### 输出 pmf
for x in sorted(count.keys()):
    prob = sp.Rational(
        count[x], 36
    )  # sp.Rational(a, b) = 精确有理数对象，用来存储分数 \(\dfrac{a}{b}\)，全程保留分子分母、不自动转浮点数小数，避免精度丢失。
    print(f"x={x}, p_X({x}) = {prob}")
```

运行输出和表格完全一致。

### 定义 1.5.2（累积分布函数）

#### SymPy 中 PMF（概率质量函数）、CDF（累积分布函数）完整实现分两种场景：

1. 自定义离散随机变量（你之前骰子例题）
2. SymPy 内置标准分布（二项、泊松、均匀离散等）

#### 一、先区分两个符号对应写法

- PMF：$p_X(x) = P(X=x)$ → 单点概率
- CDF：$F_X(x) = P(X\le x)$ → 累积概率

##### 场景 1：自定义离散分布（你那道骰子题 $p_X(x)=\dfrac{4-x}{6},\ x\in\{1,2,3\}$）

###### 完整代码

```python
import sympy as sp

x = sp.Symbol("x", integer=True)

#### 1. 定义 PMF 表达式 p_X(x)
pmf_expr = (4 - x) / 6
#### 定义域：X 只能取 1,2,3
support = sp.FiniteSet(1, 2, 3)

#### 代入单点求 PMF 值
print("p_X(1) =", pmf_expr.subs(x, 1))
print("p_X(2) =", pmf_expr.subs(x, 2))
print("p_X(3) =", pmf_expr.subs(x, 3))


#### 2. 手动构造 CDF F_X(t) = sum_{k ≤ t} p_X(k)
def cdf_F(t):
    total = 0
    for k in support:
        if k <= t:
            total += pmf_expr.subs(x, k)
    return total


print("\nF_X(1) =", cdf_F(1))
print("F_X(2) =", cdf_F(2))
print("F_X(3) =", cdf_F(3))
print("F_X(1.5) =", cdf_F(1.5))
```

输出：

```
p_X(1) = 1/2
p_X(2) = 1/3
p_X(3) = 1/6

F_X(1) = 1/2
F_X(2) = 5/6
F_X(3) = 1
F_X(1.5) = 1/2
```

##### 场景 2：SymPy 内置标准离散分布（推荐，自带 `.pmf()` / `.cdf()` 方法）

内置分布统一接口：

- `.pmf(k)`：返回 $P(X=k)$
- `.cdf(t)`：返回 $P(X\le t)$

###### 示例：离散均匀分布 X~UniformDiscrete(1,3)

```python
from __future__ import annotations  # 顶部加，开启字符串类型注解

from sympy import FiniteSet, Rational
from sympy.stats import DiscreteUniform, density, cdf
from typing import Dict, Union

# 定义离散均匀随机变量，取值 {1,2,3}
X: DiscreteUniform = DiscreteUniform("X", FiniteSet(1, 2, 3))
# 变量名和符号完全不需要一致，两者互不绑定，只是为了可读性建议统一
print("X:", X)
# dis-：分开、分离（apart）
# cernere：区分、筛选（to separate, distinguish）
# 本义：彼此分开、独立可区分、不连续粘连
# 数学含义
# 离散的：变量只能取孤立、可数分开的数值，中间值取不到。
# 例：1,2,3；不能取 1.5、2.2。
# uni- = unus 单一、同一个
# formis = forma 形态、形式
# 字面：同一形态、一模一样
# 数学含义
# 均匀的：所有结果拥有完全相等的概率 / 权重。
# Discrete Uniform 合起来直译
# 离散均匀（分布）
# 数学标准定义：
# 随机变量取值为一组离散、互相分离的数字，且每一个取值出现概率完全相等。

# PMF：
pmf_obj: DiscreteUniformDistribution = density(X)
# 1）density(X) 返回对象，不直接返回字典
# 源码里 DiscreteUniformDistribution 是一个封装类，自带属性 / 方法：
# .set：取值集合 {1,2,3}
# .p：单点概率值 1/3
# .dict：缓存属性，调用才生成完整字典 {1:1/3,2:1/3,3:1/3}
# 支持 pmf_obj[2]、pmf_obj(x) 函数式求值，兼容符号变量、小数判断
# 还能算期望、方差、熵等分布特征值
# 设计目的：通用抽象层，统一离散 / 连续分布接口
# 连续分布（正态、均匀连续）的 density() 也会返回分布对象，能统一调用 .pdf(x)；
# 如果直接返回字典，连续分布没有有限取值，字典方案完全行不通。
# 所以离散有限场景也统一返回对象，保持 API 一致，字典只是它的一个附属缓存属性。
# 你打印对象只会显示类名 DiscreteUniformDistribution(1,2,3)，想要字典要手动 .dict：
print("pmf_obj: ", pmf_obj)
print("pmf_obj.dict: ", pmf_obj.dict)
print("p_X(1) =", pmf_obj[1])
print("p_X(2) =", pmf_obj[2])
print("p_X(3) =", pmf_obj.dict[3])

# CDF：
cdf_dict: Dict[int, Rational] = cdf(X)
# 2) cdf(X) 直接返回原生字典，没有封装类
# 对有限离散均匀做了特殊简化实现：
# CDF 只关心有限支撑点上的累积概率，不存在符号函数、连续区间计算需求；
# 开发者偷懒直接预计算好所有整数点累积概率，打包成 {取值：累积概率} 字典返回，省去一层封装类。
# 这套逻辑只针对有限离散变量；
# 如果是泊松、正态这类无限 / 连续分布，cdf() 不会返回字典，而是返回符号表达式 / 可调用函数。
print("\ncdf_dict: ", cdf_dict)
print("F_X(1) =", cdf_dict[1])
print("F_X(2) =", cdf_dict[2])
print("F_X(3) =", cdf_dict[3])

# from typing import get_type_hints
# print(get_type_hints(pmf_obj))
# print(get_type_hints(cdf_dict))
```

```python
from sympy import FiniteSet, Rational
from sympy.stats import DiscreteUniform, density, cdf
from typing import Dict, Union, List, Callable

# 定义离散均匀随机变量 X，取值 {1,2,3}
X: DiscreteUniform = DiscreteUniform("X", FiniteSet(1, 2, 3))

# PMF：字典 {取值：概率分数}
pmf_dict: Dict[int, Rational] = density(X)
print("p_X(1) =", pmf_dict[1])
print("p_X(2) =", pmf_dict[2])
print("p_X(3) =", pmf_dict[3])

# CDF 原生输出字典，仅支持整数键
cdf_dict: Dict[int, Rational] = cdf(X)
print("\n整数查询：")
print("F_X(1) =", cdf_dict[1])
print("F_X(2) =", cdf_dict[2])
print("F_X(3) =", cdf_dict[3])

# 封装通用 CDF 函数：支持 int/float 小数输入，自带类型标注
support_vals: List[int] = [1, 2, 3]


def calc_cdf(
    t: Union[int, float], pmf_map: Dict[int, Rational], vals: List[int]
) -> Rational:
    """计算 F_X(t) = P(X ≤ t)，兼容小数输入"""
    total_prob: Rational = Rational(0)
    for num in vals:
        if num <= t:
            total_prob += pmf_map[num]
    return total_prob


# 小数测试
print("\n小数查询：")
print("F_X(1.2) =", calc_cdf(1.2, pmf_dict, support_vals))
print("F_X(2.8) =", calc_cdf(2.8, pmf_dict, support_vals))
print("F_X(10) =", calc_cdf(10, pmf_dict, support_vals))
```

```python
from sympy import FiniteSet, Rational, Basic
from sympy.stats import FiniteRV, density, cdf
from typing import Dict, List, Union

# 自定义不等概率分布：{1:1/2, 2:1/3, 3:1/6}
dist_map: Dict[int, Rational] = {
    1: Rational(3, 6),
    2: Rational(2, 6),
    3: Rational(1, 6),
}

# 随机变量用 Basic 注解
X: Basic = FiniteRV("X", dist_map)

# PMF 对象，同样 Basic
pmf_obj: Basic = density(X)
# CDF 原生字典
cdf_dict: Dict[int, Rational] = cdf(X)

support: List[int] = [1, 2, 3]
p1: Rational = pmf_obj[1]
p2: Rational = pmf_obj[2]
p3: Rational = pmf_obj[3]

print(f"p_X(1) = {p1}, p_X(2) = {p2}, p_X(3) = {p3}")
print("CDF 字典：", cdf_dict)


# 通用 CDF 函数，支持小数输入
def get_cdf(t: Union[int, float]) -> Rational:
    total: Rational = Rational(0)
    for val in support:
        if val <= t:
            total += pmf_obj[val]
    return total


print("F_X(2.5) =", get_cdf(2.5))
```

###### 其他常用离散分布通用语法

```python
from sympy.stats import Binomial, Poisson, density, cdf

#### 二项分布 X~B(n=5,p=0.2)
Xb = Binomial("Xb", 5, 0.2)
print("二项 p_X(2) =", density(Xb)(2))
# 修复 cdf 调用
print("二项 F_X(2) =", cdf(Xb)[2])
print("双参 二项 F_X(2) =", cdf(Xb, 2))

#### 泊松分布 X~Poisson(λ=3)
Xp = Poisson("Xp", 3)
print("泊松 p_X(1) =", density(Xp)(1))
print("泊松 F_X(1) =", cdf(Xp)(1))
print("双参 泊松 F_X(1) =", cdf(Xp, 1))
```

##### 三、关键对应关系（记忆）

| 数学记号     | SymPy 实现方式  | 作用                          |
| ------------ | --------------- | ----------------------------- |
| $p_X(x)$ PMF | `density(X)(k)` | 随机变量恰好等于 k 的概率     |
| $F_X(x)$ CDF | `cdf(X)(t)`     | 随机变量小于等于 t 的累积概率 |

##### 四、补充连续变量对比（避免混淆）

连续无 PMF，只有 PDF 概率密度，同样用 `density(X)`；CDF 依旧 `cdf(X)`：

```python
from sympy.stats import Uniform

Y = Uniform("Y", 0, 1)
pdf = density(Y)
print("f_Y(0.5) =", pdf(0.5))
print("F_Y(0.5) =", cdf(Y)(0.5))
```

##### 一句话总结

1. 自定义公式：PMF 直接代数表达式代入求值；CDF 循环求和；
2. 内置分布：统一 `density(X)` 得到质量/密度函数，`cdf(X)` 得到累积分布函数。
