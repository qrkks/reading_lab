# 概率论的基本概念

## 频率与概率

班里n个同学生日至少两人相同的概率。

```python
import math
def birthday(n):
    p = math.perm(365,n)/(365**n)
    return 1-p
# `perm` 是 **permutation** 的缩写，中文意思是**排列**。
# - 和它对应的组合数 `comb`，是 **combination** 的缩写，表示组合数 \(C(n,k)\)，不考虑顺序。
# - 所以你在 Python 里看到的 `math.perm()` 函数，就是专门用来计算排列数的。

d = {}
for i in range(60):
    d.update({i+1: round(birthday(i+1), 3)})

print(d)
```
