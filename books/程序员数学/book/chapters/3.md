# 上升到三维世界

## 在三维空间中绘制向量

### Matplotlib 绘制空白三维画布（空 3D 区域）完整解释

#### 1. 核心原理

Matplotlib 本身没有单独内置 `draw3d()` 函数，**创建三维坐标系的标准方式是 `fig.add_subplot(projection='3d')`**：

- 普通二维图：`add_subplot()` 默认投影 `rectilinear`（2D 直角）
- 三维图：传入参数 `projection='3d'`，生成 `Axes3D` 三维坐标轴对象，这就是一块空白三维绘图区域；
- 所谓“空三维区域” = 只创建 3D 坐标轴、网格、x/y/z 三轴，不绘制任何点/线/曲面。

#### 2. 最简可运行代码（纯空白 3D 画布）

```python
import matplotlib.pyplot as plt

# 1. 创建画布容器
fig = plt.figure(figsize=(8, 6))

# 2. 生成空白三维绘图区域（核心语句，替代所谓 draw3d）
ax = fig.add_subplot(111, projection="3d")

# 可选美化：显示网格、坐标轴标签
ax.set_xlabel("X 轴")
ax.set_ylabel("Y 轴")
ax.set_zlabel("Z 轴")
ax.grid(True, alpha=0.3)

# 展示空白 3D 窗口
plt.show()
```

运行后弹出窗口，只有三维坐标轴网格，没有任何图形，就是**空的三维区域**。

#### 3. 逐行拆解关键逻辑

##### （1）`fig = plt.figure()`

创建一张空白画布，所有 2D/3D 图形都画在这个画布上。

##### （2）`ax = fig.add_subplot(111, projection='3d')`

- `111`：画布只分 1 行 1 列，当前用第 1 块绘图区；
- `projection='3d'` 关键参数：告诉 matplotlib 把这块区域渲染成三维坐标系，返回 `Axes3D` 对象 `ax`；
- `ax` 就是你操作三维空间的手柄：后续画点、线、曲面都调用 `ax.plot3D()` / `ax.scatter3D()`。

##### （3）空区域的定义

只要不执行：

- `ax.scatter3D()` 三维散点
- `ax.plot3D()` 三维线条
- `ax.plot_surface()` 三维曲面
  坐标轴内就只有网格框架，属于**空白三维区域**。

#### 4. 自定义封装 draw3d() 函数（一键生成空白 3D 区）

如果你想要题目说的 `draw3d()` 一键创建空三维区域，可以自己封装：

```python
import matplotlib.pyplot as plt


def draw3d():
    """一键生成空白三维绘图区域，返回画布 +3D 坐标轴"""
    fig = plt.figure(figsize=(7, 7))
    ax = fig.add_subplot(projection="3d")
    # 基础坐标轴配置
    ax.set_xlabel("X")
    ax.set_ylabel("Y")
    ax.set_zlabel("Z")
    ax.set_box_aspect([1, 1, 1])  # 三轴等比例，防止拉伸变形
    ax.grid(True)
    return fig, ax


# 调用：生成空白 3D 区域
fig, ax = draw3d()
plt.show()
```

调用 `draw3d()` 直接得到空白三维空间，后续可以在 `ax` 上绘制向量、三维点、立体图形。

#### 5. 补充：3D 绘图基础 API（在空白区域上加图形）

空白 3D 区创建完成后，可在 `ax` 上绘图：

```python
fig, ax = draw3d()
# 在空白 3D 区域画一条从原点到 (1,1,1) 的线段
ax.plot3D([0, 1], [0, 1], [0, 1], color="red")
# 画三维散点
ax.scatter3D(1, 1, 1, s=50, color="blue")
plt.show()
```

#### 6. 关键知识点总结

1. Matplotlib**无原生 `draw3d()`**，`projection='3d'` 是创建三维区域的标准方法；
2. 空白三维区域 = 仅初始化 3D 坐标轴，不绘制任何几何图形；
3. 封装自定义 `draw3d()` 可以一键生成干净的三维画布，方便向量、立体几何绘图；
4. 三维变换（3D 旋转矩阵、三维点距离）全部在这个 `Axes3D` 对象上可视化。
