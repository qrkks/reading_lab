# 离散分布

## 离散型随机变量

### 条形图与概率直方图

原文：通常使用描绘 $X$ 的 pmf 图可以帮助我们更好地理解特定的概率分布。注意，当 $f(x)>0$ 时，pmf 的图是简单的点集 $\{[x,f(x)]:x\in S\}$，其中 $S$ 是 $X$ 的空间。可以使用两种类型的图更好地视觉评估 pmf：条形图和概率直方图。随机变量 $X$ 的概率质量函数 $f(x)$ 的条形图是在 $S$ 中的每个 $x$ 处绘制一个具有从 $(x, 0)$ 到 $[x,f(x)]$ 的垂直线段的图。如果 $x$ 只能假设整数值，则概率质量函数 $f(x)$ 的概率直方图是一个以空间 $S$ 中的每个 $x$ 为中心、高度为 $f(x)$、宽度为 $1$ 的矩形。因此，每个矩形的面积等于相应的概率 $f(x)$，并且概率直方图的总面积是 $1$.

#### 一、图结构说明

左子图：**普通条形图**（定性分类：黑、白、灰三种球，横轴是无顺序文字标签）
右子图：**离散概率直方图**（定量数值：骰子点数 1~6，横轴是有序实数）

#### 二、视觉上一眼能看出的区别

1. **横轴与柱子间距**
   - 条形图：横轴是分类文字，柱子之间留有空白，代表类别互相独立、无数值连续关系；
   - 概率直方图：横轴是数字，柱子宽度固定为 1，左右紧紧贴在一起，代表实数轴连续排布。
2. **纵轴含义**
   - 条形图纵轴：样本频数（6、3、1），全部加起来是 10，不需要等于 1；
   - 概率直方图纵轴：单点概率 $f(x)=P(X=x)=\frac16$，6 根柱子高度相加严格等于 1。
3. **数学意义**
   直方图每根矩形面积 = 宽度 (1) × 高度 (概率) = 该取值的概率；条形图没有“面积代表概率”这个规则。

#### 四、文字总结核心差异

1. **条形图**：用于**分类定性数据**，横轴无数值大小，柱子分隔，纵轴是频数/频率；
2. **概率直方图**：用于**离散定量随机变量**，横轴是实数、有序连续，柱子无缝贴合，纵轴是理论概率，全部高度和为 1，用来可视化分布律 $f(x)$。

### Python 绘制 PMF（概率质量函数）工具汇总

PMF 针对**离散型随机变量**（二项、泊松、几何、离散自定义分布），核心绘图库：`matplotlib` + `scipy.stats`、`seaborn`、`plotly`（交互式）。

#### 一、基础组合：matplotlib + scipy.stats（最常用）

`scipy.stats` 内置所有标准离散分布，自带 `.pmf(x)` 计算概率，matplotlib 画竖线/柱状图。

##### 示例 1：二项分布 PMF

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import binom

### 分布参数 n 次试验，p 成功概率
n, p = 10, 0.5
x = np.arange(0, n+1)  ### 离散取值 0~10
pmf_vals = binom.pmf(x, n, p)

### 绘制 PMF：竖线 + 散点（标准离散分布画法）
plt.stem(x, pmf_vals, basefmt=" ")  ### stem 图是 PMF 标准绘图
plt.title("Binomial PMF n=10, p=0.5")
plt.xlabel("k")
plt.ylabel("P(X=k)")
plt.grid(alpha=0.3)
plt.show()
```

- `plt.stem()`：离散 PMF 推荐，竖线代表单点概率；
- 也可用 `plt.bar(x, pmf_vals)` 柱状图展示。

##### 示例 2：自定义离散分布 PMF

### 例 2. 1-4

#### 完全可以实现，下面是**可直接运行、完美复刻教材图**的完整代码

功能对应原图所有要素：

1. 模拟 1000 次掷两个四面骰子，生成样本
2. 灰色填充直方图 = 样本相对频率（观测）
3. 黑色虚线轮廓柱状 = 理论 PMF 概率（叠加在直方图上方）
4. X 轴 2~8，Y 轴概率 0~0.25，和教材坐标轴一致
5. 严格匹配公式 $f(x)=(4-|x-5|)/16$

```python
import numpy as np
import matplotlib.pyplot as plt

### ---------------------- 1. 理论 PMF 数据（四面骰子两次总和） ----------------------
x_vals = np.array([2, 3, 4, 5, 6, 7, 8])
### 理论概率 f(x) = (4 - |x-5|)/16
theory_p = (4 - np.abs(x_vals - 5)) / 16 # 直接用公式

### ---------------------- 2. 模拟 1000 次实验，生成样本数据 ----------------------
np.random.seed(42)  ### 固定随机种子，每次运行结果一致
n_sim = 1000
### 单次四面骰子：1~4
d1 = np.random.randint(1, 5, size=n_sim)
d2 = np.random.randint(1, 5, size=n_sim)
X_sample = d1 + d2  ### 两次骰子总和

### ---------------------- 3. 绘图，复刻教材图 ----------------------
fig, ax = plt.subplots(figsize=(8, 6))

### ① 灰色填充直方图：样本相对频率（density=True 转为概率尺度）
bins = np.arange(1.5, 9.5, 1)  ### 每个整数居中分箱，对齐 x=2~8
ax.hist(
    X_sample,
    bins=bins,
    density=True,
    color="gray",
    alpha=0.6,
    label="样本相对频率（阴影直方图）"
)

### ② 虚线空心柱状：理论 PMF，叠加在直方图上
ax.bar(
    x_vals,
    theory_p,
    width=0.8,
    fill=False,        ### 不填充，只画边框
    linestyle="--",   ### 虚线，和教材虚线完全对应
    edgecolor="black",
    linewidth=1.5,
    label="理论 PMF（虚线）"
)

### ---------------------- 4. 坐标轴、美化，和教材对齐 ----------------------
ax.set_xlabel("x（两次骰子总和）")
ax.set_ylabel("$h(x), f(x)$")
ax.set_title("两个四面骰子的总和（样本频率 + 理论 PMF）")
ax.set_xticks(x_vals)
ax.set_ylim(0, 0.26)  ### Y 轴上限 0.25，留出一点边距
ax.legend()
plt.grid(axis="y", alpha=0.2)
plt.tight_layout()
plt.show()

### ---------------------- 5. 打印表格数据（和书中表 2.1-1 对应） ----------------------
print("x | 观察数量 | 相对频率 | 理论概率")
print("--|----------|----------|--------")
counts, _ = np.histogram(X_sample, bins=bins)
for i, x in enumerate(x_vals):
    cnt = counts[i]
    freq = cnt / n_sim
    prob = theory_p[i]
    print(f"{x} | {cnt:6d} | {freq:.3f} | {prob:.4f}")
```

#### 代码和教材图一一对应说明

1. **灰色填充直方图**：`ax.hist(density=True)` 自动把计数转为**相对频率**，就是书中阴影部分；
2. **黑色虚线空心柱子**：`bar(fill=False, linestyle="--")`，完全复刻图里虚线轮廓的理论 PMF；
3. 分箱 `bins=np.arange(1.5,9.5,1)`：让每个柱子中心刚好落在 2,3,4…8，和原图 X 轴对齐；
4. 底部会打印表格，输出「x、观测数量、相对频率、理论概率」，和书中表 2.1-1 格式一致；
5. `np.random.seed(42)` 固定随机结果，每次运行的样本计数不会乱变。

#### 运行效果说明

- 灰色柱子高度 = 1000 次模拟里每个总和出现的频率；
- 虚线空心柱子高度 = 理论精确概率；
- 样本越多（把`n_sim=1000`改成 10000），灰色柱子会越来越贴近虚线理论柱，体现大数定律。

### 例 2.2-1

```python
import sympy as sp
sp.init_printing()

x, y = sp.symbols("x y", integer=True)
# 定义函数
f = (4 - x) / 6
g = (4 - sp.sqrt(y)) / 6

print("f(x) =", f)
print("g(y) =", g)
print("-"*30)

# 测试 f(x)
print("f(1) =", f.subs(x, 1))   # 3/6 = 1/2
print("f(3) =", f.subs(x, 3))   # 1/6
print("f(4) =", f.subs(x, 4))   # 0，x=4 才是零点
print("-"*30)

# 测试 g(y)
print("g(1)  =", g.subs(y, 1))  # (4-1)/6 = 3/6 = 1/2
print("g(9)  =", g.subs(y, 9))  # (4-3)/6 = 1/6
print("g(16) =", g.subs(y, 16)) # (4-4)/6 = 0 真正零点
print("g(5)  =", g.subs(y, 5))  # (4 - sqrt(5))/6 ≈ 0.2287，不是 0
# 转小数查看近似值
print("g(5) 数值近似 =", float(g.subs(y, 5)))
```

公式：

\(\begin{cases}
f(x)=\dfrac{4-x}{6},\quad x=1,2,3\\[6pt]
g(y)=\dfrac{4-\sqrt{y}}{6},\quad y=1,4,9
\end{cases}\)

```python
import sympy as sp
sp.init_printing()

# 定义符号
x = sp.Symbol("x", integer=True)
y = sp.Symbol("y", integer=True)

# 离散分布律 f(x)，仅 x=1,2,3 有效
f = sp.Piecewise(
    # sp.Piecewise( (表达式 1, 条件 1), (表达式 2, 条件 2), ..., (默认表达式，True) )
    ((4 - x) / 6, sp.Or(sp.Eq(x,1), sp.Eq(x,2), sp.Eq(x,3))),
    # Or = 逻辑或，对应数学里的「或者」，只要任意一个条件成立，整体条件就为真。
    # 配套：
    # sp.Or(a,b,c)：a 或 b 或 c，任一成立就 True
    # sp.And(a,b,c)：a 且 b 且 c，全部成立才 True
    (0, True)
    # 每一组参数都是 (表达式，判断条件)
    # 所以 (0, True) 的含义：
    # 第 1 项 0：满足条件时，函数返回的值
    # 第 2 项 True：永真逻辑条件，永远成立
)

# 离散分布律 g(y)，仅 y=1,4,9 有效
g = sp.Piecewise(
    ((4 - sp.sqrt(y)) / 6, sp.Or(sp.Eq(y,1), sp.Eq(y,4), sp.Eq(y,9))),
    (0, True)
)

# 打印表达式
print("===== f(x) 分布律 =====")
sp.pprint(f)
print("\n===== g(y) 分布律 =====")
sp.pprint(g)

# 测试 f(x) 所有合法点 + 非法点
print("\nf(x) 取值测试：")
for xi in [1, 2, 3, 4, 5]:
    print(f"f({xi}) = {f.subs(x, xi)}")

# 测试 g(y) 所有合法点 + 非法点
print("\ng(y) 取值测试：")
for yi in [1, 4, 9, 5, 16]:
    print(f"g({yi}) = {g.subs(y, yi)}")
```

#### 全部流程

1.已知三组对应关系：
\(x=1\Rightarrow g=3,\ x=2\Rightarrow g=2,\ x=3\Rightarrow g=1\)
假设分子是一次函数 \(g(x)=ax+b\)，用两点构造方程组求解系数 \(a,b\)，再验证第三点，最后构造

2.解方程组，推导方程

```python
import sympy as sp
# 定义符号
a, b, x = sp.symbols('a b x')

# 列方程组：代入 (1,3)、(2,2)
eq1 = sp.Eq(a*1 + b, 3)
eq2 = sp.Eq(a*2 + b, 2)
system = [eq1, eq2]
# system 英文直译：方程组、系统
# system of equations 是数学标准术语，意思就是「方程组」。
# 你这里 [eq1, eq2] 是把两个等式打包在一起，构成一套联立方程，起名 system 是贴合数学概念的规范命名。

# 解二元一次方程组
sol = sp.solve(system, (a, b))
# sp.solve(equations, variables, ...)
# 后面的变量必须传，不保证里面有其它参数，如{a+b=k；2a+b=2k}
# 你这里只有 a,b 两个未知数，所以必须传 (a, b) 或 [a, b] 告诉 solve 你要解的是这两个变量。


print("系数解：", sol)  # 输出 {a: -1, b: 4}
```

3.核验第三个点

```python
g_x = sol[a] * x + sol[b]
print("g(x) =", g_x)  # g(x) = 4 - x
g_3 = g_x.subs(x, 3)
print("x=3 时 g(3) =", g_3)  # 输出 1，和实际计数匹配
```

\(g(x)\) 的含义：取到回报 x 有多少个骰子样本点，它不是概率。

4.生成 pmf 并校验概率总样本数 6，因此概率质量函数：
\(f(x)=\frac{g(x)}{6}=\frac{4-x}{6}\)

```python
f_x = g_x / 6
print("pmf f(x) =", f_x)  # 2/3 - x/6 等价于 (4-x)/6

# 逐个输出取值概率
for xi in [1,2,3]:
    print(f"f({xi}) = {f_x.subs(x, xi)}")
# f(1)=1/2=3/6, f(2)=1/3=2/6, f(3)=1/6

# 验证归一性：概率和=1
total = sum(f_x.subs(x,xi) for xi in [1,2,3])
print("概率总和 =", total)  # 输出 1，满足 pmf 要求
```

5.关键结论
整个推导完全依靠符号方程组求解，不是瞪眼猜；
Sympy 严格解出线性系数 \(a=-1,b=4\)，自动得到统一表达式 \(4-x\)；
数学等价形式：
\(f(x)=\frac{4-x}{6} = \frac{2}{3}-\frac{x}{6},\quad x=1,2,3\)
校验结果和手动计数完全一致，概率总和为 1，符合离散 pmf 定义。

6.计算期望
期望公式（离散随机变量）\(E(X) = \sum\_{x} x\cdot f(x)\) 本题仅 \(x=1,2,3\) 有概率，因此：
\(E(X) = 1\cdot f(1) + 2\cdot f(2) + 3\cdot f(3)\)

```python
import sympy as sp

# 1. 定义符号
a, b, x = sp.symbols('a b x')

# 2. 构造方程组求线性系数
eq1 = sp.Eq(a * 1 + b, 3)
eq2 = sp.Eq(a * 2 + b, 2)
sol = sp.solve([eq1, eq2], (a, b))
# sp.solve(equations, variables, ...)
# 第 1 个参数：equations 待求解方程 / 方程组
# 两种形式：
# 单个方程：直接传 eq1
# 方程组：列表包裹多个 Eq 对象 [eq1, eq2]
# 你的例子：
# [eq1, eq2] 就是方程组参数。
# 补充：不用 sp.Eq 也可以，直接写表达式（默认表达式 = 0）
# sp.solve([a+b-3, 2*a+b-2], (a,b)) 和上面等价。
#
# 第 2 个参数：variables 需要解的未知变量
# 支持元组 / 列表，按你想求解的未知数顺序传入：
# 二元方程组未知量 a,b → 传 (a, b) 或 [a, b]
# 返回字典 {a: -1, b: 4}，键是变量，值是解。

print("解得系数：", sol)

# 3. 构造 g(x) 与 pmf f(x)
g_x = sol[a] * x + sol[b]
f_x = g_x / 6
print("概率质量函数 f(x) =", f_x)

# 4. 计算数学期望 E[X] = sum( x * f(x), x=1,2,3 )
E_X = 1 * f_x.subs(x, 1) + 2 * f_x.subs(x, 2) + 3 * f_x.subs(x, 3)
print("期望 E[X] =", E_X)
# print("期望小数近似值 =", float(E_X))

# 分步打印每一项
print("\n分项计算：")
for xi in [1,2,3]:
    prob = f_x.subs(x, xi)
    term = xi * prob
    print(f"x={xi}:{xi} * {prob} = {term}")
```

7.随机变量的变换

```python
import sympy as sp

# 1. 基础符号与原随机变量 X 的 pmf
x = sp.Symbol('x', integer=True, positive=True)
y = sp.Symbol('y', integer=True, positive=True)
# 原 pmf
f_x = (4 - x) / 6
print("原变量 X 的 pmf f(x) =", f_x)

# 2. 列出 X 所有取值，对应 Y=X²的取值映射
x_vals = [1, 2, 3]
# x：原随机变量 X
# vals = values 的缩写，意思是「取值、数值」
# 命名原因这个列表存放随机变量 X 所有能取到的数值：1、2、3，完整语义是 x_values，简写 x_vals 是 Python / 数据分析通用写法。
# 等价完整写法：x_values = [1,2,3]，简写更清爽。
y_mapping = {xi: xi**2 for xi in x_vals}
# 概率论里，随机变量 X 的第 i 个观测 / 取值标准记号就是：
# 代码里 xi 是 x_i 的简写
# 你这里的 xi 就是 x_1, x_2, x_3 这些取值，对应到 X 的观测值。
print("\nX → Y=X² 映射关系：", y_mapping)

# 3. 推导 Y 的 pmf g(y)
# Y 取值 y=1,4,9，由 y=x²反解 x=√y，代入 f(x) 得到 g(y)
g_y = (4 - sp.sqrt(y)) / 6
# 第一步：用 y 反解出 x 由 \(y=x^2\)，且 x 只取正数 \(1,2,3\)，开根号只保留正根：
# \(x = \sqrt{y}\)
# 对应取值映射：
# \(x=1 \Rightarrow y=1,\ x=2 \Rightarrow y=4,\ x=3 \Rightarrow y=9\) 第二步：替换 —— 把 \(f(x)\) 里所有 x 换成 \(\sqrt{y}\) 因为 Y 的概率 = 对应 X 的概率，即 \(g(y)=f(x)\)，将 \(x=\sqrt{y}\) 代入原 pmf：
# \(g(y) = f\big(\sqrt{y}\big) = \frac{4-\sqrt{y}}{6}\)
# 这就是 g_y = (4 - sp.sqrt(y)) / 6 的来源。
print("\nY=X²的 pmf g(y) =", g_y)

# 逐个打印 g(y) 验证
for yi in [1, 4, 9]:
    prob = g_y.subs(y, yi)
    # expr.subs(old, new)
    # # 或者批量替换（字典）
    # expr.subs({old1: new1, old2: new2, ...})
    # # 或者批量替换（元组列表）
    # expr.subs([(old1, new1), (old2, new2), ...])
    print(f"g({yi}) = {prob}")

# 4. 计算 E[X]、E[Y]=E[X²]
E_X = sum(xi * f_x.subs(x, xi) for xi in x_vals)
E_X2 = sum((xi**2) * f_x.subs(x, xi) for xi in x_vals)
# 用 Y 的 pmf 再算一遍 E[Y]，交叉验证
y_vals = [1,4,9]
E_Y_from_g = sum(yi * g_y.subs(y, yi) for yi in y_vals)

print("\n==== 期望计算 ====")
print(f"E[X] = {E_X}  ≈ {float(E_X):.4f}")
print(f"E[X²]（通过 X 求和） = {E_X2}")
print(f"E[Y]（通过 Y 的 pmf g(y) 求和） = {E_Y_from_g}")

# 5. 计算游戏净收益期望
# 场景 1：年轻人向玩家收 2 美元入场费
profit_2 = 2 - E_X
# 场景 2：收 4 美元入场费
profit_4 = 4 - E_X

print("\n==== 庄家每场平均盈利 ====")
print(f"收 2 美元入场费，单场期望盈利：2 - E[X] = {profit_2} ≈ {float(profit_2):.4f}")
print(f"收 4 美元入场费，单场期望盈利：4 - E[X] = {profit_4} ≈ {float(profit_4):.4f}")

# 拓展：Z=X³的期望（文中提到的 X³游戏）
E_X3 = sum((xi**3) * f_x.subs(x, xi) for xi in x_vals)
profit_10 = 10 - E_X3
print("\n==== Z=X³ 游戏（入场费 10 美元） ====")
print(f"E[X³] = {E_X3}")
print(f"收 10 美元入场费，庄家单场期望盈利：10 - E[X³] = {profit_10} ≈ {float(profit_10):.4f}")
```

### 例 2.2-2

#### 一、数学手写原式（课本标准写法）

已知：
$S_X=\{-1,0,1\}$，$f(x)=\dfrac13,\ x\in S_X$，求 $E(X^2)$

$$
\begin{aligned}
E(X^2) &= \sum_{x\in S_X} x^2 f(x) \\
&= (-1)^2\cdot \frac13 + 0^2\cdot \frac13 + 1^2\cdot \frac13 \\
&= 1\cdot \frac13 + 0 + 1\cdot \frac13 \\
&= \frac23
\end{aligned}
$$

#### 二、SymPy 代码两种写法（FiniteRV 最简单，匹配本题有限取值）

##### 写法 1：FiniteRV（推荐，3 个离散点）

```python
from sympy import S
from sympy.stats import FiniteRV, E
# S(1)/3 创建精确有理分数
# X = FiniteRV("X", {-1: S(1)/3, 0: S(1)/3, 1: S(1)/3}) # 或
X = FiniteRV("X", {-1: S("1/3"), 0: S("1/3"), 1: S("1/3")})

res = E(X ** 2)
print(res)  # 输出 2/3
```

```python
S("1/3")
```

##### 写法 2：DiscreteRV + Piecewise + Eq（严谨分段写法）

```python
from sympy import Symbol, Piecewise, Eq, Lambda, S
from sympy.stats import DiscreteRV, E

x = Symbol("x", integer=True)
#### 分段 PMF：x=-1/0/1 时 1/3，其余 0
pmf = Piecewise(
    (S(1)/3, Eq(x, -1)),
    (S(1)/3, Eq(x, 0)),
    (S(1)/3, Eq(x, 1)),
    (0, True)
)
#### 封装成离散随机变量
X = DiscreteRV(x, Lambda(x, pmf))
#### 计算 E(X²)
print(E(X**2))
```

#### 三、关键说明

1. 核心逻辑：只要拿到随机变量对象 `X`，任意函数 $g(X)$ 直接套进 `E(g(X))`，不用手动写求和；
2. 本题只有 3 个有限取值，优先 `FiniteRV`，代码最短；
3. `E(X**2)` 就是数学里的 $E(X^2)$，完美对应例题求变换后期望。
