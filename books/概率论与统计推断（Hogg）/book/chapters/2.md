# 离散分布

## 离散型随机变量

### 条形图与概率直方图

#### 一、图结构说明

左子图：**普通条形图**（定性分类：黑、白、灰三种球，横轴是无顺序文字标签）
右子图：**离散概率直方图**（定量数值：骰子点数1~6，横轴是有序实数）

#### 二、视觉上一眼能看出的区别

1. **横轴与柱子间距**
   - 条形图：横轴是分类文字，柱子之间留有空白，代表类别互相独立、无数值连续关系；
   - 概率直方图：横轴是数字，柱子宽度固定为1，左右紧紧贴在一起，代表实数轴连续排布。
2. **纵轴含义**
   - 条形图纵轴：样本频数（6、3、1），全部加起来是10，不需要等于1；
   - 概率直方图纵轴：单点概率 $f(x)=P(X=x)=\frac16$，6根柱子高度相加严格等于1。
3. **数学意义**
   直方图每根矩形面积 = 宽度(1) × 高度(概率) = 该取值的概率；条形图没有“面积代表概率”这个规则。

#### 三、可直接运行的完整Python代码（修复字体警告）

```python
import matplotlib.pyplot as plt
import numpy as np

plt.rcParams["axes.unicode_minus"] = False
# 缩小画布尺寸
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(8, 4))

# ========== 左图：条形图 ==========
color_labels = ["黑球", "白球", "灰球"]
sample_count = [6, 3, 1]
bars1 = ax1.bar(color_labels, sample_count, width=0.6, color=["#222222", "#dddddd", "#888888"])
ax1.set_title("条形图（定性分类：摸球颜色）", fontsize=10)
ax1.set_ylabel("样本频数（统计个数）", fontsize=9)
ax1.grid(axis="y", alpha=0.3)
for idx, num in enumerate(sample_count):
    ax1.text(idx, num + 0.1, str(num), ha="center")

# ========== 右图：概率直方图（修复版） ==========
x_dice = [1, 2, 3, 4, 5, 6]
prob = [1/6, 1/6, 1/6, 1/6, 1/6, 1/6]
width = 0.8  # 把宽度调小一点，避免贴边看不见
bars2 = ax2.bar(x_dice, prob, width=width, color="#4488dd", edgecolor="black")
ax2.set_title("概率直方图（离散变量：掷骰子）", fontsize=10)
ax2.set_xlabel("随机变量取值 X（点数）", fontsize=9)
ax2.set_ylabel("单点概率 f(x) = P(X=x)", fontsize=9)
ax2.set_xticks(x_dice)
ax2.set_ylim(0, 0.22)
ax2.grid(axis="y", alpha=0.3)
# 标注概率数字
for x, p in zip(x_dice, prob):
    ax2.text(x, p + 0.005, f"{p:.3f}", ha="center", fontsize=8)

plt.suptitle("条形图 与 概率直方图 直观对比", fontsize=12)
plt.tight_layout()
plt.show()
```

#### 四、文字总结核心差异

1. **条形图**：用于**分类定性数据**，横轴无数值大小，柱子分隔，纵轴是频数/频率；
2. **概率直方图**：用于**离散定量随机变量**，横轴是实数、有序连续，柱子无缝贴合，纵轴是理论概率，全部高度和为1，用来可视化分布律 $f(x)$。
