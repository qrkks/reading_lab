# 离散分布

## 离散型随机变量

### 条形图与概率直方图

原文：通常使用描绘 $X$ 的 pmf 图可以帮助我们更好地理解特定的概率分布. 注意，当 $f(x)>0$ 时，pmf 的图是简单的点集 $\{[x,f(x)]:x\in S\}$，其中 $S$ 是 $X$ 的空间. 可以使用两种类型的图更好地视觉评估 pmf：条形图和概率直方图. 随机变量 $X$ 的概率质量函数 $f(x)$ 的条形图是在 $S$ 中的每个 $x$ 处绘制一个具有从 $(x, 0)$ 到 $[x,f(x)]$ 的垂直线段的图. 如果 $x$ 只能假设整数值，则概率质量函数 $f(x)$ 的概率直方图是一个以空间 $S$ 中的每个 $x$ 为中心、高度为 $f(x)$、宽度为 $1$ 的矩形. 因此，每个矩形的面积等于相应的概率 $f(x)$，并且概率直方图的总面积是 $1$.

#### 一、图结构说明

左子图：**普通条形图**（定性分类：黑、白、灰三种球，横轴是无顺序文字标签）
右子图：**离散概率直方图**（定量数值：骰子点数1~6，横轴是有序实数）

#### 二、视觉上一眼能看出的区别

1. **横轴与柱子间距**
   - 条形图：横轴是分类文字，柱子之间留有空白，代表类别互相独立、无数值连续关系；
   - 概率直方图：横轴是数字，柱子宽度固定为1，左右紧紧贴在一起，代表实数轴连续排布。
2. **纵轴含义**
   - 条形图纵轴：样本频数（6、3、1），全部加起来是10，不需要等于1；
   - 概率直方图纵轴：单点概率 $f(x)=P(X=x)=\frac16$，6根柱子高度相加严格等于1。
3. **数学意义**
   直方图每根矩形面积 = 宽度(1) × 高度(概率) = 该取值的概率；条形图没有“面积代表概率”这个规则。

#### 三、可直接运行的完整Python代码（修复字体警告）

```python
import matplotlib.pyplot as plt
import numpy as np
plt.rcParams["axes.unicode_minus"] = False
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(8, 4))
# plt.subplots(nrows, ncols)
# 1, 2 = 生成 1 行、2 列 一共两个子图窗口：
# 
# 返回两个对象：fig + 子图数组
# axes：所有小绘图区的数组。
# 
# 1 行 2 列时 axes 是一维列表 [ax1, ax2]，所以直接解包写成 (ax1, ax2)；
# 后面绘图，分别用 ax1.plot()、ax2.stem() 在左右两张图分开画，互不干扰。
# 
# figsize=(8, 4) 整张画布宽 8 英寸，高 4 英寸；

# ========== 左图：教材标准 pmf 垂直线段条形图（stem杆图） ==========
x_dice = [1,2,3,4,5,6]
prob = [1/6, 1/6, 1/6, 1/6, 1/6, 1/6]
ax1.stem(x_dice, prob, linefmt="black", markerfmt="o", basefmt=" ")
ax1.set_title("PMF条形图（垂直线段，教材标准）", fontsize=10)
ax1.set_xlabel("X取值", fontsize=9)
ax1.set_ylabel("f(x)=P(X=x)", fontsize=9)
ax1.set_xticks(x_dice)
ax1.set_ylim(0, 0.22)
ax1.grid(axis="y", alpha=0.3)

# ========== 右图：合规概率直方图 width=1 ==========
bars2 = ax2.bar(x_dice, prob, width=1, color="#4488dd", edgecolor="black")
ax2.set_title("概率直方图（宽=1，中心在整数x）", fontsize=10)
ax2.set_xlabel("随机变量取值 X（点数）", fontsize=9)
ax2.set_ylabel("单点概率 f(x) = P(X=x)", fontsize=9)
ax2.set_xticks(x_dice)
ax2.set_ylim(0, 0.22)
ax2.grid(axis="y", alpha=0.3)
for x, p in zip(x_dice, prob):
    ax2.text(x, p + 0.005, f"{p:.3f}", ha="center", fontsize=8)

plt.suptitle("教材标准 PMF条形图 vs 概率直方图")
plt.tight_layout()
plt.show()
```

#### 四、文字总结核心差异

1. **条形图**：用于**分类定性数据**，横轴无数值大小，柱子分隔，纵轴是频数/频率；
2. **概率直方图**：用于**离散定量随机变量**，横轴是实数、有序连续，柱子无缝贴合，纵轴是理论概率，全部高度和为1，用来可视化分布律 $f(x)$。

### Python 绘制 PMF（概率质量函数）工具汇总

PMF 针对**离散型随机变量**（二项、泊松、几何、离散自定义分布），核心绘图库：`matplotlib` + `scipy.stats`、`seaborn`、`plotly`（交互式）。

#### 一、基础组合：matplotlib + scipy.stats（最常用）

`scipy.stats` 内置所有标准离散分布，自带 `.pmf(x)` 计算概率，matplotlib 画竖线/柱状图。

##### 示例1：二项分布 PMF

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import binom

### 分布参数 n次试验，p成功概率
n, p = 10, 0.5
x = np.arange(0, n+1)  ### 离散取值 0~10
pmf_vals = binom.pmf(x, n, p)

### 绘制PMF：竖线+散点（标准离散分布画法）
plt.stem(x, pmf_vals, basefmt=" ")  ### stem图是PMF标准绘图
plt.title("Binomial PMF n=10, p=0.5")
plt.xlabel("k")
plt.ylabel("P(X=k)")
plt.grid(alpha=0.3)
plt.show()
```

- `plt.stem()`：离散PMF推荐，竖线代表单点概率；
- 也可用 `plt.bar(x, pmf_vals)` 柱状图展示。

##### 示例2：自定义离散分布 PMF

```python
import numpy as np
import matplotlib.pyplot as plt

### 自定义取值与对应概率
x = np.array([1,2,3,4,5])
prob = np.array([0.1, 0.2, 0.4, 0.2, 0.1])

plt.stem(x, prob, basefmt="gray")
plt.xlabel("X")
plt.ylabel("PMF P(X=x)")
plt.show()
```

#### 二、seaborn 美化版本

适合快速美化、搭配统计绘图风格：

```python
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy.stats import poisson

sns.set_style("whitegrid")
lam = 3
x = np.arange(0, 10)
pmf = poisson.pmf(x, lam)

plt.bar(x, pmf, width=0.6, color="#4285F4")
plt.title(f"Poisson PMF λ={lam}")
plt.show()
```

#### 三、Plotly：交互式 PMF（网页/Notebook）

鼠标悬浮看精确概率，适合报告、交互分析：

```python
import numpy as np
import plotly.express as px
from scipy.stats import geom

p = 0.3
x = np.arange(1, 15)
pmf = geom.pmf(x, p)

fig = px.bar(x=x, y=pmf, title=f"Geometric PMF p={p}", labels={"x":"k", "y":"P(X=k)"})
fig.show(renderer="browser") # 在浏览器中渲染
```

#### 四、专用高阶工具

1. **sympy.stats**
   符号计算分布，可符号输出PMF再绘图，适合数学推导：

```python
from sympy import symbols, init_printing
from sympy.stats import Binomial, density
init_printing(use_latex="mathjax")

k = symbols("k", integer=True, nonnegative=True)
# 抛10次硬币 p=0.5
X = Binomial("X", n=10, p=0.5)
pmf = density(X)(k)
pmf
```

2. **statsmodels**
   侧重统计建模，支持分布诊断绘图，少量使用。

3. **matplotlib.pyplot.vlines**
   自定义竖线，精细控制样式：

```python
plt.vlines(x, ymin=0, ymax=pmf_vals, lw=3, color="blue")
plt.plot(x, pmf_vals, "o", color="red")
```

#### 关键区分：PMF vs PDF

- PMF：离散变量 → `stem/bar`，scipy `.pmf()`
- PDF：连续变量 → 曲线，scipy `.pdf()`

#### 最简通用封装函数（直接复制）

```python
def plot_pmf(x, pmf_values, title="PMF Plot"):
    import matplotlib.pyplot as plt
    plt.stem(x, pmf_values, linefmt="#222222", markerfmt="o", basefmt="lightgray")
    plt.xlabel("X values")
    plt.ylabel("P(X=x) PMF")
    plt.title(title)
    plt.grid(alpha=0.2)
    plt.show()
```
