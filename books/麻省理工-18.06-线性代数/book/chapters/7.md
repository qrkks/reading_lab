## R = reduced row echolen form

```python
import sympy as sp

U = sp.Matrix([
    [1, 2, 2, 2],
    [0, 0, 2, 4],
    [0, 0, 0, 0],
])

R = U.rref()
sp.pprint(R)
# [1 2 0 -2]
# [0 0 1 2 ]
# [0 0 0 0 ]
```

### 行最简式分块形式

$$
\boldsymbol{s}_1 = \begin{bmatrix} -F_{:,1} \\ 1 \\ 0 \end{bmatrix},\quad
\boldsymbol{s}_2 = \begin{bmatrix} -F_{:,2} \\ 0 \\ 1 \end{bmatrix}
$$

代入 $F=\begin{bmatrix}2 & -2 \\0 & 2\end{bmatrix}$：

$$
\boldsymbol{s}_1 = \begin{bmatrix} -2 \\ 0 \\ 1 \\ 0 \end{bmatrix},\quad
\boldsymbol{s}_2 = \begin{bmatrix} 2 \\ -2 \\ 0 \\ 1 \end{bmatrix}
$$

## A 的转置

```python
import sympy as sp

A = sp.Matrix([
    [1, 2, 3],
    [2, 4, 6],
    [2, 6, 8],
    [2, 8, 10],
])
print("A: ")
sp.pprint(A)

print("\nU: ")
sp.pprint(U := A.echelon_form())
print("\nR:")
sp.pprint(R := A.rref())
print("\nZ:")
sp.pprint(Z := A.nullspace())
```
