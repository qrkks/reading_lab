### 知识点详解：LU 分解（A=LU）

#### 1. 定义

将方阵 $\boldsymbol{A}$ 拆分为两个三角矩阵的乘积：
$$\boldsymbol{A} = \boldsymbol{L}\boldsymbol{U}$$

- $\boldsymbol{L}$：**下三角矩阵**，对角线元素恒为 1（单位下三角）
- $\boldsymbol{U}$：**上三角矩阵**

#### 2. 核心原理

本质是**不带行交换的高斯消元**：

1. 只用倍加初等行变换消去下方元素，不做换行；
2. 消元的乘数存入下三角矩阵 $\boldsymbol{L}$；
3. 消元后的阶梯矩阵即为 $\boldsymbol{U}$。

#### 3. 用途

- 快速求解线性方程组 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{b}$：拆为 $\boldsymbol{L}\boldsymbol{y}=\boldsymbol{b}$（前代）+ $\boldsymbol{U}\boldsymbol{x}=\boldsymbol{y}$（回代）
- 矩阵行列式、求逆、数值模拟、有限元、机器学习特征分解底层工具

#### 4. 补充说明

若消元过程必须换行，则拓展为 **PA=LU 分解**（P 为置换矩阵）。
“重制版”一般代表该章节/课件对 LU 分解的推导、例题、代码实现做了修订优化。

### 简易计算示例

设

$$
\boldsymbol{A}=
\begin{bmatrix}
2 & 1 \\
6 & 7
\end{bmatrix}
$$

消元：第 2 行减 3 倍第 1 行，乘数 $l_{21}=3$

$$
\boldsymbol{U}=
\begin{bmatrix}
2 & 1 \\
0 & 4
\end{bmatrix},\quad
\boldsymbol{L}=
\begin{bmatrix}
1 & 0 \\
3 & 1
\end{bmatrix}
$$

验证：

$$
\boldsymbol{L}\boldsymbol{U}=
\begin{bmatrix}
1 & 0 \\
3 & 1
\end{bmatrix}
\begin{bmatrix}
2 & 1 \\
0 & 4
\end{bmatrix}
=
\begin{bmatrix}
2 & 1 \\
6 & 7
\end{bmatrix}
=\boldsymbol{A}
$$

## SymPy 完全支持 LU 分解 ✅

SymPy 内置 `LUdecomposition()` 方法，支持**数值矩阵、符号矩阵**，默认输出带置换矩阵的 $PA=LU$ 分解；也可实现无换行纯 $A=LU$（不需要行交换时）。

### 一、核心语法说明

```python
L, U, P = A.LUdecomposition()
```

返回三元组：

1. `L`：单位下三角矩阵（对角线全 1）
2. `U`：上三角矩阵
3. `P`：置换矩阵（行交换记录，无交换时为单位阵）
   满足恒等式：$\boldsymbol{P}\boldsymbol{A} = \boldsymbol{L}\boldsymbol{U}$

---

### 二、基础数值示例（无行交换，纯 A=LU）

#### 代码

```python
from sympy import Matrix, pprint, eye

# 1. 定义测试方阵
A = Matrix([
    [2, 1],
    [6, 7],
])

# 2. LU 分解，返回 L(单位下三角), U(上三角), perm(行交换索引列表)
L, U, perm = A.LUdecomposition()

# 3. 根据 perm 列表生成置换矩阵 P（n 阶单位矩阵交换对应行）
n = A.shape[0]
P = eye(n)
# perm 存储需要交换的行下标对，遍历交换
for i, j in perm:
    P.row_swap(i, j)

# 打印各矩阵信息
print("===== L 单位下三角矩阵 =====")
pprint(L)
print("\n===== U 上三角矩阵 =====")
pprint(U)
print("\n===== perm 行交换索引列表（空=无换行） =====")
print(perm)
print("\n===== P 置换矩阵 =====")
pprint(P)

# 矩阵等式验证（全部为 Matrix 对象，无类型错误）
PA = P * A
LU = L * U
print("\n===== 等式验证 =====")
print(f"P*A = \n{PA}")
print(f"L*U = \n{LU}")
print(f"P*A == L*U 结果：{PA == LU}")
print(f"L*U == A 结果：{LU == A}")
```

#### 输出结果

```
L（单位下三角）：
⎡1  0⎤
⎣3  1⎦

U（上三角）：
⎡2  1⎤
⎣0  4⎦

置换矩阵P（无交换=单位阵）：
⎡1  0⎤
⎣0  1⎦

验证 P*A == L*U： True
验证 L*U == A： True
```

### 三、需要行交换的矩阵（PA=LU）

当主元为 0 必须换行时，$A \neq LU$，只能满足 $PA=LU$：

```python
from sympy import Matrix, pprint

# 首行首列主元为 0，必须换行
A = Matrix([[0, 2], [3, 1]])
L, U, P = A.LUdecomposition()

pprint(P)
pprint(L)
pprint(U)
print("P*A =", P * A)
print("L*U =", L * U)
print("P*A == L*U：", P * A == L * U)
```

### 四、符号矩阵 LU 分解（代数符号推导）

SymPy 独有优势：可以对带变量的符号矩阵做符号 LU 分解：

```python
from sympy import Matrix, symbols, pprint

a, b, c, d = symbols("a b c d")

A = Matrix([[a, b], [c, d]])
L, U, P = A.LUdecomposition()
pprint(L)
pprint(U)
```

### 五、配套工具：LU 求解线性方程组 `LUsolve()`

分解后直接解 $\boldsymbol{A}\boldsymbol{x}=\boldsymbol{b}$，底层复用 LU 分解：

```python
A = Matrix([[2, 1], [6, 7]])
b = Matrix([5, 27])
x = A.LUsolve(b)
pprint(x)  # 输出 [2, 1]
```

### 六、关键注意点

1. **区分 `A=LU` 和 `PA=LU`**
   - 若 `P=eye(n)`（单位阵），则 $\boldsymbol{A}=\boldsymbol{L}\boldsymbol{U}$；
   - 若存在行交换，只能保证 $\boldsymbol{P}\boldsymbol{A}=\boldsymbol{L}\boldsymbol{U}$，此时直接 $L*U \neq A$。
2. **仅方阵支持 LU 分解**，非方阵报错。
3. 对比 NumPy/SciPy：SymPy 支持**符号代数推导**，适合手写推导、理论证明；数值计算优先 SciPy。
4. 若只要无置换的 A=LU（课堂手工消元题型），先判断矩阵顺序主子式非零，此时 P 自动为单位阵。

## 黑板内容讲解：矩阵转置与逆的核心公式

### 1. 两条基础前置等式

1. $AA^{-1}=I$
   - $A$ 是可逆方阵，$A^{-1}$ 是 $A$ 的逆矩阵，$I$ 是单位矩阵。
   - 定义：一个矩阵乘它的逆 = 单位矩阵。

2. $\boldsymbol{(A^{-1})^T A^T = I}$
   这里用到**矩阵乘积的转置法则**：$(XY)^T=Y^T X^T$
   推导：
   $$
   (AA^{-1})^T = I^T \implies (A^{-1})^T A^T = I
   $$

### 2. 核心结论：$\boldsymbol{(A^T)^{-1} = (A^{-1})^T}$

根据**逆矩阵的定义**：
如果 $MN=I$，则 $M=N^{-1}$、$N=M^{-1}$。

观察式子 $(A^{-1})^T \cdot A^T = I$：

- 左边是「矩阵 $(A^{-1})^T$」乘以「矩阵 $A^T$」等于单位阵 $I$
- 按照逆的定义，$(A^{-1})^T$ 就是 $A^T$ 的逆矩阵，记作 $\boldsymbol{(A^T)^{-1}}$

黑板手写标注：

> This is $(A^T)^{-1}$，INVERSE OF $A^T$
> 翻译：$(A^{-1})^T$ 这个东西，就是 $A^T$ 的逆矩阵。

### 3. 公式名称与含义

等式 $\boldsymbol{(A^T)^{-1} = (A^{-1})^T}$ 叫**逆的转置 = 转置的逆**：
对可逆方阵 $A$，**先求逆再转置**，等价于**先转置再求逆**。

#### 完整逻辑梳理

1. $AA^{-1}=I$（逆矩阵定义）
2. 两边同时转置：$(AA^{-1})^T = I^T$
3. 转置分配律：$(A^{-1})^T A^T = I$
4. 由逆的定义，$(A^{-1})^T$ 是 $A^T$ 的逆，即 $(A^T)^{-1}=(A^{-1})^T$。

## 黑板内容完整解读（LU 分解）

### 1. 核心公式：$\boldsymbol{A=LU}$

这是**矩阵 LU 分解**（高斯消元分解）：

- $A$：原方阵
- $L$：**下三角单位矩阵**（对角线全为 1，下方存消元乘数）
- $U$：**上三角矩阵**（高斯消元后得到的阶梯矩阵）

### 2. 手写英文逐句翻译（修正潦草手写）

原句：

> IF no row exchanges multipliers go directly into L

翻译：
**如果消元过程中不需要交换任意两行（无行互换操作），那么所有消元乘数可以直接放进下三角矩阵 L 里。**

#### 关键词拆解

1. **no row exchanges**：消元全程不做换行、不交换两行
2. **multipliers**：消元乘数（就是之前$E_{21}$里的$-4$这类系数）
3. **go directly into L**：直接存入$L$的下三角位置

### 3. 背后原理（结合之前$E_{21}$初等矩阵）

高斯消元是用初等矩阵左乘$A$得到上三角$U$：
$$E_k \dots E_2 E_1 A = U$$
变形得：
$$A = \underbrace{E_1^{-1} E_2^{-1} \dots E_k^{-1}}_{L} U$$

- 每个消元初等矩阵 $E_{ij}=\begin{pmatrix}1&0\\-m&1\end{pmatrix}$，它的逆是 $E_{ij}^{-1}=\begin{pmatrix}1&0\\m&1\end{pmatrix}$
- $m$ 就是**消元乘数**（比如之前消去 8 用的$4$）
- 只要全程**不交换行**，所有逆初等矩阵相乘后，所有乘数$m$会全部落在$L$的下三角空位，不用额外调整，直接写进$L$。

### 4. 举个例子（衔接你刚才的$\begin{pmatrix}2&1\\8&7\end{pmatrix}$）

消元操作 $R_2 = R_2 - 4R_1$，乘数$m=4$，无换行：
$$L=\begin{pmatrix}1&0\\4&1\end{pmatrix},\quad U=\begin{pmatrix}2&1\\0&3\end{pmatrix}$$
验算：

$$
LU=\begin{pmatrix}1&0\\4&1\end{pmatrix}\begin{pmatrix}2&1\\0&3\end{pmatrix}=\begin{pmatrix}2&1\\8&7\end{pmatrix}=A
$$

乘数$4$直接放进$L_{21}$，完美对应黑板那句话。

### 5. 补充前提（反向理解）

如果消元中途**交换了两行（row exchange）**，单纯$A=LU$就不成立，要升级成带置换矩阵的分解：$PA=LU$。黑板这句话就是强调「无换行」这个能直接构造$L$的前提条件。

## Permutations

### permutation（置换/排列）在线性代数（LU 分解）里的完整解释

#### 一、单词本义

permutation = **置换、排列**
动词：permute 重新排序、交换次序

#### 二、线性代数语境：Permutation Matrix 置换矩阵 $P$

##### 1. 定义

置换矩阵是**把单位矩阵 $I$ 的行（或列）打乱重排**得到的方阵，满足 2 个特点：

- 每行、每列**恰好只有 1 个 1，其余全是 0**
- 一定可逆，且逆等于自身转置：$P^{-1}=P^T$

##### 2. 作用

- **左乘 $PA$**：对矩阵 $A$ **交换行**（行置换）
- **右乘 $AP$**：对矩阵 $A$ **交换列**（列置换）

##### 3. 举个 2 阶例子

单位阵 $I=\begin{bmatrix}1&0\\0&1\end{bmatrix}$
交换第 1、2 行得到置换矩阵：
$$P=\begin{bmatrix}0&1\\1&0\end{bmatrix}$$
左乘验证行交换：
$$PA=\begin{bmatrix}0&1\\1&0\end{bmatrix}\begin{bmatrix}a&b\\c&d\end{bmatrix}=\begin{bmatrix}c&d\\a&b\end{bmatrix}$$
$A$ 的两行直接互换。

#### 三、结合你黑板上的 $A=LU$ 知识点（核心）

之前黑板写了一句：**IF no row exchanges multipliers go directly into L**

- `row exchanges` = 行交换，本质就是**乘置换矩阵 $P$**

1. 消元全程**不用交换行**：
   直接 $A=LU$，不用置换矩阵
2. 消元中途**必须交换行**（主元位置出现 0，无法消元）：
   标准分解变成 $\boldsymbol{PA=LU}$
   - $P$：置换矩阵，记录你所有行交换操作；
   - 先把 $A$ 的行排好序（$PA$），再做普通 LU 分解。

#### 四、直观理解“置换”

Permutation 就是**对一组有序对象重新调换顺序**：

- 数字：$[1,2,3] \to [2,1,3]$ 是一次置换
- 矩阵行：行 1、行 2 互换，就是对矩阵行做一次置换，对应左乘置换矩阵$P$

#### 五、补充小性质

1. 任意置换矩阵相乘，结果仍是置换矩阵；
2. $P^2=I$：交换同一组行两次，矩阵变回原样；
3. 行列式 $\det(P)=\pm1$，交换奇数次行为$-1$，偶数次为$1$。

##### 场景举例

3 阶矩阵消元时第一行主元为 0，必须把第三行换到第一行：
对应的置换矩阵 $P$ 就是把单位阵第 1、3 行互换，$PA$ 会把原矩阵第 3 行挪到最上方，之后再正常做 LU 分解。
