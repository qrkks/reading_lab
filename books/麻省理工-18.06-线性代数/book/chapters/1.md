# The Geometry of Linear Equations

方程组的几何解释

## 行图像、列图像、矩阵形式

这是线性代数里**线性方程组 \(Ax=b\) 的三种经典解读视角**，也是 Gilbert Strang《线性代数导论》课程开篇的核心框架，从左到右分别是：行图像、列图像、矩阵形式。

---

### 1. Row picture（行图像）

**几何视角：方程的交点**
把方程组的每一行看作一个几何图形：

- 二维里，每一行对应平面上的**一条直线**；
- 三维里，每一行对应空间里的**一个平面**；
- 高维里对应超平面。

方程组的解，就是所有这些直线/平面的**公共交点**。
举个例子：

$$
\begin{cases}
2x+y=3 \\
x-y=0
\end{cases}
$$

行图像就是在 xy 平面画两条直线，它们的交点 \((1,1)\) 就是方程组的解。

### 2. Column picture（列图像）

**线性代数核心视角：列向量的线性组合**
把整个方程组改写为「矩阵的列向量做加权相加」：

$$
x\cdot\begin{bmatrix}2\\1\end{bmatrix} + y\cdot\begin{bmatrix}1\\-1\end{bmatrix} = \begin{bmatrix}3\\0\end{bmatrix}
$$

也就是：**找到一组系数 \(x,y\)，让矩阵的列向量线性组合后，刚好等于右侧的向量 \(b\)**。

这是线性代数最本质的思维方式——解的存在性、列空间、秩这些核心概念，都是从列图像延伸出来的，也是**Strang 最强调的理解角度**。

### 3. Matrix form（矩阵形式）

**代数简写形式**
把系数、未知数、右侧常数分别打包，写成紧凑的矩阵乘法格式：

$$
Ax = b
$$

对应上面的例子就是：

$$
\begin{bmatrix}2&1\\1&-1\end{bmatrix}
\begin{bmatrix}x\\y\end{bmatrix}
=
\begin{bmatrix}3\\0\end{bmatrix}
$$

它是最简洁的书写方式，方便后续做逆矩阵、矩阵分解、数值计算等操作。

---

### 一句话总结

三者描述的是同一个东西：

- 行图像 = 「几条线/面交在哪里」的几何直观
- 列图像 = 「列向量能不能拼出目标向量」的线性代数本质
- 矩阵形式 = 「浓缩成符号」的代数表达

## SymPy 解二元一次线性方程组

方程组：

$$
\begin{cases}
2x - y = 0 \\
-x + 2y = 3
\end{cases}
$$

### 完整可运行 SymPy 代码

```python
import sympy as sp

## 1. 定义符号变量
x, y = sp.symbols('x y')

## 2. 构建两个方程（等式右边移到左边等于 0）
eq1 = sp.Eq(2*x - y, 0)
eq2 = sp.Eq(-x + 2*y, 3)

## 3. 求解方程组
sol = sp.solve((eq1, eq2), (x, y))
print("方程组解：", sol)
```

### 运行输出结果

```
方程组解： {x: 1, y: 2}
```

### 补充两种写法

#### 写法 1：直接传表达式（不用 Eq）

```python
sol = sp.solve([2*x - y, -x + 2*y - 3], (x, y))
```

#### 写法 2：矩阵线性求解（对应 Strang 线代行/列图像）

```python
import sympy as sp
A = sp.Matrix([[2, -1], [-1, 2]])
b = sp.Matrix([0, 3])
ans = A.LUsolve(b)
print(ans)
## 输出 Matrix([[1], [2]])
```

### 验证解

2. $-1 + 2\times 2 = 3$ 满足第二式

完全成立。

## 解线性方程方程组

### 一、先还原黑板上的矩阵方程

$$
\underbrace{
\begin{bmatrix}
2 & -1 \\
-1 & 2
\end{bmatrix}}_{A}
\underbrace{
\begin{bmatrix}
x \\
y
\end{bmatrix}}_{\boldsymbol x}
=
\underbrace{
\begin{bmatrix}
0 \\
3
\end{bmatrix}}_{\boldsymbol b}
$$

这是标准线性方程组 $A\boldsymbol x=\boldsymbol b$，正好对应刚才讲的**行图像、列图像、矩阵形式**三种解读。

#### 1. Row picture（行图像：两条直线求交点）

拆开成普通二元一次方程组：

$$
\begin{cases}
2x - y = 0 \quad (第一行)\\
-x + 2y = 3 \quad (第二行)
\end{cases}
$$

- 第一条直线：$y=2x$，过原点、斜率 2；
- 第二条直线：$2y=x+3 \Rightarrow y=\frac12 x+\frac32$，斜率$\frac12$；
  方程组的解就是两条直线的**交点坐标**。

#### 2. Column picture（列图像：列向量线性组合）

把矩阵拆成两列向量，写成向量加权相加：

$$
x \cdot \begin{bmatrix}2\\-1\end{bmatrix}
+
y \cdot \begin{bmatrix}-1\\2\end{bmatrix}
=
\begin{bmatrix}0\\3\end{bmatrix}
$$

含义：找系数 $x,y$，让第一个列向量放大$x$倍、第二个列向量放大$y$倍，两个向量相加刚好等于右侧$\begin{bmatrix}0\\3\end{bmatrix}$。

#### 3. Matrix form（矩阵乘法简写）

就是黑板上这种紧凑写法 $A\boldsymbol x=\boldsymbol b$，是线性代数通用标准格式。

### 二、用 LU 分解 + LUsolve 完整解这道题

##### 步骤 1：对 $A=\begin{bmatrix}2&-1\\-1&2\end{bmatrix}$ 做 LU 分解

高斯消元消去第二行首元素：
消元乘数 $l_{21} = \dfrac{1}{2}$，第二行 = 第二行 $+\dfrac12 \times$ 第一行

$$
L=\begin{bmatrix}1 & 0 \\ \frac12 & 1\end{bmatrix},\quad
U=\begin{bmatrix}2 & -1 \\ 0 & \frac32\end{bmatrix}
$$

验算 $LU=A$：

$$
LU=
\begin{bmatrix}1 & 0 \\ \frac12 & 1\end{bmatrix}
\begin{bmatrix}2 & -1 \\ 0 & \frac32\end{bmatrix}
=
\begin{bmatrix}2 & -1 \\ -1 & 2\end{bmatrix}=A
$$

##### 步骤 2：前向代换 $L\boldsymbol y=\boldsymbol b$，求 $\boldsymbol y$

$$
\begin{bmatrix}1 & 0 \\ \frac12 & 1\end{bmatrix}
\begin{bmatrix}y_1\\y_2\end{bmatrix}
=
\begin{bmatrix}0\\3\end{bmatrix}
$$

1. $y_1=0$
2. $\dfrac12 y_1 + y_2 = 3 \Rightarrow y_2=3$
   得 $\boldsymbol y=\begin{bmatrix}0\\3\end{bmatrix}$

##### 步骤 3：回代 $U\boldsymbol x=\boldsymbol y$，求最终解 $\boldsymbol x$

$$
\begin{bmatrix}2 & -1 \\ 0 & \frac32\end{bmatrix}
\begin{bmatrix}x\\y\end{bmatrix}
=
\begin{bmatrix}0\\3\end{bmatrix}
$$

1. 下式：$\dfrac32 y = 3 \Rightarrow$ $y=2$
2. 上式：$2x - y = 0 \Rightarrow 2x=2 \Rightarrow x=1$

### 三、最终解

$$
\boldsymbol x=
\begin{bmatrix}x\\y\end{bmatrix}
=
\begin{bmatrix}1\\2\end{bmatrix}
$$

##### 验证

代入原方程：

- $2\times 1 - 2 = 0$ ✔️
- $-1\times 1 + 2\times 2 = -1+4=3$ ✔️

```python
import sympy as sp

# 构造矩阵 A, 向量 b
A = sp.Matrix([
    [2, -1, 0],
    [-1, 2, -1],
    [0, -3, 4]
])
b = sp.Matrix([1, 1, -3])

# 方法 1：直接解方程 A*x = b
x = A.solve(b)
print("解向量 x =")
sp.pprint(x)

# 方法 2：求逆矩阵 x = A⁻¹b
A_inv = A.inv()
# inv = inverse，英文含义：逆、倒数、逆矩阵
# A.inv() = compute the inverse matrix of A，也就是求矩阵 A 的逆矩阵 \(A^{-1}\)。
x2 = A_inv * b
print("\n用逆矩阵求解 x = A⁻¹b：")
sp.pprint(x2)
```

## SymPy / NumPy 两种生成 9 阶随机矩阵方案

### 方案 1：NumPy（数值随机矩阵，最常用）

适合浮点/整数随机矩阵，速度快：

```python
import numpy as np

## 1. 9×9 0~1 之间浮点数随机矩阵
rand9_float = np.random.rand(9, 9)
print("9 阶浮点随机矩阵：\n", rand9_float)

## 2. 9×9 整数随机矩阵，范围 [-10, 10]
rand9_int = np.random.randint(low=-10, high=11, size=(9, 9))
print("9 阶整数随机矩阵：\n", rand9_int)

## 转成 SymPy Matrix
import sympy as sp
A_sp = sp.Matrix(rand9_int)
sp.pprint(A_sp)
```

### 方案 2：纯 SymPy 随机矩阵（符号计算环境）

SymPy 内置 `sp.randMatrix()`，专门生成随机整数矩阵：

#### 基础用法：9 阶方阵

```python
import sympy as sp

## n=9，范围默认 0~9
A = sp.randMatrix(9)
sp.pprint(A)
print(A)
print(A.det())
```

#### 自定义取值范围、尺寸

```python
## 9 行 9 列，元素取值 -5 ~ 5
A9 = sp.randMatrix(
    rows=9,
    cols=9,
    min=-5,
    max=5
)
sp.pprint(A9)
```

#### 关键参数说明

- `rows, cols`：行数、列数，同为 9 就是 9 阶方阵
- `min, max`：随机整数上下界（闭区间）
- `symmetric=True`：生成**对称随机矩阵**（主对角线对称）

```python
## 9 阶对称随机矩阵
A_sym = sp.randMatrix(9,9,-3,3,symmetric=True)
sp.pprint(A_sym)
```

### 方案 3：生成随机可逆 9 阶矩阵（常用解方程）

很多时候需要行列式≠0 的可逆矩阵，避免 `.inv()` 报错：

```python
import sympy as sp

def rand_invertible(n, min_val=-5, max_val=5):
    while True:
        M = sp.randMatrix(n,n,min_val,max_val)
        if M.det() != 0:
            return M

## 获取 9 阶可逆随机矩阵
A_inv9 = rand_invertible(9)
print("行列式 det(A) =", A_inv9.det())
sp.pprint(A_inv9)
```

### 小总结

1. 数值运算用 `np.random`；
2. 符号线性代数（解方程、求逆、行列式）直接用 `sp.randMatrix(9)`；
3. 需要可逆矩阵写循环过滤行列式为 0 的奇异矩阵。
