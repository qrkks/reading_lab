# 方程组的几何解释

## 行图像、列图像、矩阵形式

这是线性代数里**线性方程组 \(Ax=b\) 的三种经典解读视角**，也是Gilbert Strang《线性代数导论》课程开篇的核心框架，从左到右分别是：行图像、列图像、矩阵形式。

---

### 1. Row picture（行图像）

**几何视角：方程的交点**
把方程组的每一行看作一个几何图形：

- 二维里，每一行对应平面上的**一条直线**；
- 三维里，每一行对应空间里的**一个平面**；
- 高维里对应超平面。

方程组的解，就是所有这些直线/平面的**公共交点**。
举个例子：

$$
\begin{cases}
2x+y=3 \\
x-y=0
\end{cases}
$$

行图像就是在xy平面画两条直线，它们的交点 \((1,1)\) 就是方程组的解。

### 2. Column picture（列图像）

**线性代数核心视角：列向量的线性组合**
把整个方程组改写为「矩阵的列向量做加权相加」：

$$
x\cdot\begin{bmatrix}2\\1\end{bmatrix} + y\cdot\begin{bmatrix}1\\-1\end{bmatrix} = \begin{bmatrix}3\\0\end{bmatrix}
$$

也就是：**找到一组系数 \(x,y\)，让矩阵的列向量线性组合后，刚好等于右侧的向量 \(b\)**。

这是线性代数最本质的思维方式——解的存在性、列空间、秩这些核心概念，都是从列图像延伸出来的，也是**Strang最强调的理解角度**。

### 3. Matrix form（矩阵形式）

**代数简写形式**
把系数、未知数、右侧常数分别打包，写成紧凑的矩阵乘法格式：

$$
Ax = b
$$

对应上面的例子就是：

$$
\begin{bmatrix}2&1\\1&-1\end{bmatrix}
\begin{bmatrix}x\\y\end{bmatrix}
=
\begin{bmatrix}3\\0\end{bmatrix}
$$

它是最简洁的书写方式，方便后续做逆矩阵、矩阵分解、数值计算等操作。

---

### 一句话总结

三者描述的是同一个东西：

- 行图像 = 「几条线/面交在哪里」的几何直观
- 列图像 = 「列向量能不能拼出目标向量」的线性代数本质
- 矩阵形式 = 「浓缩成符号」的代数表达

# SymPy 解二元一次线性方程组

方程组：

$$
\begin{cases}
2x - y = 0 \\
-x + 2y = 3
\end{cases}
$$

## 完整可运行 SymPy 代码

```python
import sympy as sp

# 1. 定义符号变量
x, y = sp.symbols('x y')

# 2. 构建两个方程（等式右边移到左边等于0）
eq1 = sp.Eq(2*x - y, 0)
eq2 = sp.Eq(-x + 2*y, 3)

# 3. 求解方程组
sol = sp.solve((eq1, eq2), (x, y))
print("方程组解：", sol)
```

## 运行输出结果

```
方程组解： {x: 1, y: 2}
```

## 补充两种写法

### 写法1：直接传表达式（不用Eq）

```python
sol = sp.solve([2*x - y, -x + 2*y - 3], (x, y))
```

### 写法2：矩阵线性求解（对应Strang线代行/列图像）

```python
import sympy as sp
A = sp.Matrix([[2, -1], [-1, 2]])
b = sp.Matrix([0, 3])
ans = A.LUsolve(b)
print(ans)
# 输出 Matrix([[1], [2]])
```
