# Elimination with Matrices 利用矩阵进行消元

## 噢，sympy 里 LUsolve 就是指定高斯消元法计算？

### 结论：是的，`LUsolve` 底层就是**高斯消元衍生的 LU 分解解法**，专门用高斯消元做矩阵分解来解方程 $A\boldsymbol{x}=\boldsymbol{b}$

#### 1. 底层逻辑对应 MIT 18.06 第 2 节课内容

1. 先对方阵 $A$ 做**高斯消元**，得到 $A=LU$
   - $L$：下三角（记录所有消元倍数）
   - $U$：高斯消元后的上三角矩阵
2. 拆分方程 $A\boldsymbol{x}=\boldsymbol{b}$ → $LU\boldsymbol{x}=\boldsymbol{b}$
   ① 前向回代解 $L\boldsymbol{y}=\boldsymbol{b}$
   ② 后向回代解 $U\boldsymbol{x}=\boldsymbol{y}$
   整个流程完全是这节课讲的**矩阵消元思路**。

#### 2. 和普通 `.solve(b)` 的区别

- `A.solve(b)`：SymPy 自动选算法（克莱姆、高斯消元、LU 等），不固定；
- `A.LUsolve(b)`：**强制只用 LU 分解（高斯消元路线）**，不会切换其他解法；
- 等价写法：`A.solve(b, method='LU')` 内部直接调用 `LUsolve`。

#### 3. 使用限制（贴合你之前学的知识点）

1. **只能用于可逆方阵**
   若 $\det(A)=0$、主元出现 0，LU 分解失败，直接报错；
2. 只处理方阵线性方程组 $A_{n\times n}\boldsymbol{x}_{n\times1}=\boldsymbol{b}_{n\times1}$，长方形矩阵不能用；
3. 优势：同一个 A 多次解不同 b 时，只分解一次 LU，重复计算更快。

#### 4. 完整代码示例

```python
import sympy as sp

A = sp.Matrix([
    [2, -1, 0],
    [-1, 2, -1],
    [0, -3, 4],
])
b = sp.Matrix([0, -1, 4])

### 强制 LU 分解（高斯消元路线）求解
x = A.LUsolve(b)
sp.pprint(x)

### 等价写法
x2 = A.solve(b, method="LU")
print(x == x2)  ### True
```

#### 5. 一句话区分概念

- 高斯消元：原始行变换操作，得到上三角 $U$；
- LU 分解：把高斯消元的全过程打包成 $A=LU$；
- `LUsolve`：基于 LU 分解（高斯消元）的专用解方程函数。

## SymPy 完整实现高斯消元（两种方式：自动化简 + 分步手动模拟消元过程）

### 一、核心工具

1. `Matrix.rref()`：一键算出**行最简阶梯形（简化高斯消元最终结果）**
2. `Matrix.echelon_form()`：算出**行阶梯形（高斯消元中间三角形态）**
3. 手动行变换：`row_op()` 可以**一步步模拟消元操作**，完美复现手写高斯消元全过程

### 1. 一键自动高斯消元（快速出结果）

[echelon_form、rref、增广矩阵](../appendices/Echelon_form%20&%20RREF.md)

```python
import sympy as sp

# 定义矩阵
A = sp.Matrix([
    [2, 4, 1],
    [1, 2, 3],
    [3, 1, 2],
])

print("原始矩阵 A：")
sp.pprint(A)

# 1. 高斯消元：行阶梯形（上三角）
echelon = A.echelon_form()
print("\n高斯消元后 行阶梯形：")
sp.pprint(echelon)

# 2. 高斯 - 若尔当消元：行最简形 RREF（考试最终标准形式）
rref_mat, pivot_cols = A.rref()
print("\n行最简阶梯形 RREF：")
sp.pprint(rref_mat)
print("主元 pivot 所在列索引：", pivot_cols)
```

输出会直接标出所有 `pivot`（主元），对应你学的 `1st pivot` 概念。

### 2. 分步手动模拟高斯消元（重点：复刻手写每一步）

`row_op(i, j, k)`：第 i 行 = 第 i 行 + k × 第 j 行（消元核心操作）
`row_swap(i,j)`：交换两行（找主元）
`row_mult(i, k)`：第 i 行整体乘系数（把主元化为 1）

```python
import sympy as sp

# 增广矩阵示例方程组：
# 2x + 4y = 6
# x -  y = 0
M = sp.Matrix([
    [2, 4, 6],
    [1, -1, 0],
])
print("初始增广矩阵：")
sp.pprint(M)

# 步骤 1：交换 0 行、1 行，把 1 放到第一行第一个位置（找 1st pivot）
M1 = M.copy()  # 复制独立副本
M1.row_swap(0, 1)  # 原地修改，不要赋值 M1 = ...
print("\n步骤 1：交换两行，把主元 1 换到第一行首列：")
sp.pprint(M1)

# 步骤 2：第 1 行 = 第 1 行 - 2×第 0 行，消去第二行第一个元素
M2 = M1.copy()
M2.row_add(1, 0, -2)  # 等价 lambda，更简洁：R1 = R1 - 2*R0
print("\n步骤 2：消去第二行 x 项：")
sp.pprint(M2)

# 步骤 3：第二行整体除以 -6，把 pivot 变成 1
M3 = M2.copy()
# 替换这一行
M3.row_mult(1, sp.Rational(-1, 6))
print("\n步骤 3：主元归一化为 1：")
sp.pprint(M3)

# 步骤 4：回代，消去第一行 y 项，得到最简形
M4 = M3.copy()
M4.row_add(0, 1, 4)  # R0 = R0 + 4*R1
print("\n步骤 4：回代消元，得到 RREF：")
sp.pprint(M4)
```

运行后每一步矩阵变化都会打印，**完全和你黑板上手写消元步骤一致**，可以用来验算作业、理解每一步行变换逻辑。

### 3. 提取主元 pivot（匹配你之前学的 1st pivot）

```python
A = sp.Matrix([
    [2, 5, 1],
    [0, -3, 4],

```

### 4. 求解线性方程组（高斯消元最终用途）

```python
x, y, z = sp.symbols("x y z")
eq1 = sp.Eq(2 * x + y, 5)
eq2 = sp.Eq(x - 3 * y, 1)

sol = sp.solve((eq1, eq2), (x, y))
print("方程组解：", sol)

# 底层等价：把方程组转为增广矩阵做高斯消元
aug = sp.Matrix([
    [2, 1, 5],
    [1, -3, 1],
])
rref, _ = aug.rref()
sp.pprint(rref)
```

### 关键总结

1. **自动消元**：`echelon_form()`（高斯消元三角阵）/ `rref()`（高斯若尔当最简形），一键出答案；
2. **模拟全过程**：`row_swap / row_op / row_mult` 分步操作，复现手写每一步消元、找主元、归一化；
3. 配套功能：自动识别所有 `pivot` 主元位置，完美对应你课堂上 `1st pivot` 的知识点。

## 例题

$$
\begin{cases}
x + 2y + z = 2 \\
3x + 8y + z = 12 \\
4y + z = 2
\end{cases}
$$

```python
import sympy as sp

A = sp.Matrix([
    [1, 2, 1],
    [3, 8, 1],
    [0, 4, 1],
])
print("原始矩阵：")
sp.pprint(A)

print("\n高斯消元 Echelon Form 结果：")
sp.pprint(A.echelon_form())  # 它执行了缩放，实际最后一行应该是 0 0 5
# 当矩阵已经完全化为行阶梯形后，任意单独缩放任意一行，都只改变该行系数；其中最后一行因为下方无元素，在消元全过程中，只要不再做反向消元，缩放它完全不会干扰其他所有行。
# 行阶梯形只约束两点，不限制主元数值大小：
# 全零行统一放在矩阵最底部；
# 每一行主元（首个非零数）的列编号，严格大于上一行主元的列编号。
# 你这三种最后一行：
# \([0,0,5],\ [0,0,10],\ [0,0,20]\)
# 主元都在第 3 列，位置完全不变，仅数值缩放，完全满足 REF 定义，全部合规。

print("\n高斯消元 RREF 结果：")
rref, pivots = A.rref()
print(A.rref())
sp.pprint("\nrref =")
sp.pprint(rref)
sp.pprint("\npivots =")
sp.pprint(pivots)

print("\nA.det() =", A.det())  # 会有无数个主元乘积，但是只有一个行列式
# 只有全程只用行倍加、不缩放任何一行、不交换行得到的上三角矩阵，主元相乘才等于原矩阵行列式；
# 只要你手动缩放任意一行，这个阶梯矩阵的行列式就会跟着变，此时它的主元乘积不再代表原矩阵的行列式。
# 原始矩阵的行列式永远只有一个确定值，不会因为阶梯形写法不同而改变。
```

## 主元是可以随便缩放的？

不是“随便”缩放，而是**取决于你采用的行变换规则和目标矩阵形式**，核心结论可以总结为：

### 1. 行阶梯形（REF）：主元可以缩放，结果不唯一

行阶梯形（Row Echelon Form）的核心要求只有两个：

- 全零行在最下方；
- 每行的主元（首个非零元）所在列，位置比上一行主元更靠右。

它**没有规定主元必须等于某个固定值，也没有要求主元必须为 1**。

根据初等行变换的规则，**任意一行乘以一个非零常数，依然是合法的初等行变换，得到的矩阵仍然是行阶梯形**。对应到你的例子：

- 只用「行倍加」消元（不做行缩放），得到主元 `1, 2, 5`（LU 分解的 U 矩阵），是行阶梯形；
- SymPy `echelon_form()` 用 Bareiss 整数算法，得到主元 `1, 2, 10`，也是行阶梯形；
- 你把最后一行除以 10，主元变成 `1`，依然是行阶梯形。

这些都是正确的，只是执行的行变换策略不同，**行阶梯形本身就不唯一**。

### 2. 行最简形（RREF）：主元不能缩放，结果唯一

行最简形（Reduced Row Echelon Form）在 REF 的基础上增加了两条强制规则：

- 所有主元必须为 1；
- 主元所在列的**上下所有元素都必须是 0**。

这种形式是**唯一的**，主元不可以随意缩放，必须归一化为 1。这也是 `A.rref()` 的输出结果，无论用什么算法，最终得到的 RREF 矩阵一定完全相同。

### 3. 补充：不同缩放策略的用途

主元缩放不是“乱缩放”，不同场景有不同的约定：

- **LU 分解（Doolittle 形式）**：约定只用行倍加、不用行缩放，保证 L 是单位下三角（对角线全 1），U 的主元就是自然消元结果，此时 `det(A) = U 主元的乘积`。
- **Bareiss 算法**：全程保持整数运算，不引入分数，主元依次等于矩阵的各阶顺序主子式，适合符号计算中避免浮点/分数误差。
- **数值计算（高斯消元）**：常做“主元归一化”或选主元缩放，目的是提升数值稳定性，减少计算误差。

### 一句话总结

主元的数值本身没有“标准答案”：只要主元位置不变、非零，通过非零常数缩放行得到的都是合法的行阶梯形；只有行最简形（RREF）要求主元必须是 1，且结果唯一。

## 矩阵乘法

```python
import sympy as sp

# 3 阶方阵 3×3
A = sp.randMatrix(3, seed=42)
print("矩阵 A：")
sp.pprint(A)

# 3 行 1 列 标准列向量
x = sp.randMatrix(3, 1, seed=99)
print("\n列向量 x：")
sp.pprint(x)

# 行向量左乘矩阵 xᵀA (1×3)
print("\nx.T * A 结果（行向量）：")
res_left = x.T * A
sp.pprint(res_left)

# 矩阵右乘列向量 Ax (3×1)
print("\nA * x 结果（列向量）：")
res_right = A * x
sp.pprint(res_right)
```

### 一、核心区别的本质：维度相容与向量形式

矩阵乘法遵循严格的**维度相容原则**：若矩阵 $A$ 形状为 $m \times n$，矩阵 $B$ 形状为 $n \times p$，则乘积 $AB$ 形状为 $m \times p$，要求**左边矩阵的列数 = 右边矩阵的行数**。

向量是特殊的矩阵：

- 列向量：形状为 $n \times 1$（$n$ 行 1 列），是默认的向量形式
- 行向量：形状为 $1 \times n$（1 行 $n$ 列）

因此两种乘法的差异完全由维度规则和向量的行列形式决定，具体对比如下：

| 对比维度 | 矩阵 × 向量（左乘列向量）$A\boldsymbol{x}$                        | 向量 × 矩阵（右乘行向量）$\boldsymbol{y}A$                      |
| -------- | ----------------------------------------------------------------- | --------------------------------------------------------------- |
| 维度要求 | $A$ 为 $m \times n$，$\boldsymbol{x}$ 为 $n \times 1$ 列向量      | $\boldsymbol{y}$ 为 $1 \times m$ 行向量，$A$ 为 $m \times n$    |
| 结果形状 | $m \times 1$ 的列向量                                             | $1 \times n$ 的行向量                                           |
| 代数本质 | 以 $\boldsymbol{x}$ 的元素为系数，对 $A$ 的**列向量**做线性组合   | 以 $\boldsymbol{y}$ 的元素为系数，对 $A$ 的**行向量**做线性组合 |
| 几何意义 | 对列向量 $\boldsymbol{x}$ 施加 $A$ 定义的线性变换（旋转、缩放等） | 行向量视为线性泛函，右乘矩阵等价于对泛函做拉回变换              |
| 交换性   | 与向量右乘矩阵不相等，维度不匹配时直接非法                        | 与矩阵左乘向量不相等，维度不匹配时直接非法                      |

### 二、SymPy 代码演示

以下代码完整复现两种乘法的计算过程、形状差异，以及维度不匹配时的报错。

```python
import sympy as sp

# 1. 定义符号变量
a, b, c, d = sp.symbols("a b c d")
x1, x2 = sp.symbols("x1 x2")
y1, y2 = sp.symbols("y1 y2")

# 2. 定义 2×2 矩阵 A
A = sp.Matrix([[a, b], [c, d]])
print("矩阵 A：")
sp.pprint(A)
print(f"形状：{A.shape}\n")

# 3. 演示：矩阵 × 列向量（左乘）
x_col = sp.Matrix([[x1], [x2]])  # 2×1 列向量
Ax = A * x_col
print("=== 矩阵 A × 列向量 x ===")
print("列向量 x：")
sp.pprint(x_col)
print(f"形状：{x_col.shape}")
print("乘积 A*x：")
sp.pprint(Ax)
print(f"结果形状：{Ax.shape}\n")

# 4. 演示：行向量 × 矩阵（右乘）
y_row = sp.Matrix([[y1, y2]])  # 1×2 行向量
yA = y_row * A
print("=== 行向量 y × 矩阵 A ===")
print("行向量 y：")
sp.pprint(y_row)
print(f"形状：{y_row.shape}")
print("乘积 y*A：")
sp.pprint(yA)
print(f"结果形状：{yA.shape}\n")

# 5. 演示：维度不匹配的非法乘法
print("=== 维度不匹配的报错示例 ===")
try:
    x_col * A  # 列向量左乘矩阵：2×1 乘 2×2，列数≠行数
except Exception as e:
    print(f"列向量左乘矩阵报错：{type(e).__name__}: {e}")

try:
    A * y_row  # 矩阵右乘行向量：2×2 乘 1×2，列数≠行数
except Exception as e:
    print(f"矩阵右乘行向量报错：{type(e).__name__}: {e}\n")

# 6. 补充验证：转置性质 (A x)^T = x^T A^T
print("=== 转置性质验证 ===")
Ax_T = Ax.T  # 左乘结果的转置
xT_AT = x_col.T * A.T  # 行向量乘矩阵的转置
print("(A x)^T 与 x^T A^T 是否相等：", sp.simplify(Ax_T - xT_AT) == sp.zeros(1, 2))

# 对称矩阵特殊情况：(A x)^T = x^T A
A_sym = sp.Matrix([[a, b], [b, d]])
Ax_sym = A_sym * x_col
xT_A = x_col.T * A_sym
print(
    "对称矩阵下 (A x)^T 与 x^T A 是否相等：",
    sp.simplify(Ax_sym.T - xT_A) == sp.zeros(1, 2),
)
```

### 三、输出结果与解释

1. **矩阵 × 列向量结果**

$$
A\boldsymbol{x} = \begin{bmatrix} a & b \\ c & d \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} = \begin{bmatrix} a x_1 + b x_2 \\ c x_1 + d x_2 \end{bmatrix}
$$

结果为 2×1 列向量，每个元素是矩阵对应行与向量的内积，本质是矩阵两列的线性组合：$x_1\begin{bmatrix}a\\c\end{bmatrix} + x_2\begin{bmatrix}b\\d\end{bmatrix}$。

2. **行向量 × 矩阵结果**

$$
\boldsymbol{y}A = \begin{bmatrix} y_1 & y_2 \end{bmatrix} \begin{bmatrix} a & b \\ c & d \end{bmatrix} = \begin{bmatrix} a y_1 + c y_2 & b y_1 + d y_2 \end{bmatrix}
$$

结果为 1×2 行向量，每个元素是向量与矩阵对应列的内积，本质是矩阵两行的线性组合：$y_1\begin{bmatrix}a & b\end{bmatrix} + y_2\begin{bmatrix}c & d\end{bmatrix}$。

3. **维度不匹配报错**
   列向量左乘矩阵、矩阵右乘行向量都会触发 `ShapeError`，因为不符合“左列=右行”的乘法规则。

4. **转置性质**
   两种乘法通过转置关联：$(A\boldsymbol{x})^T = \boldsymbol{x}^T A^T$；若 $A$ 是对称矩阵（$A^T=A$），则 $(A\boldsymbol{x})^T = \boldsymbol{x}^T A$。

### 四、关键结论

- 日常语境中“矩阵乘向量”默认指**矩阵左乘列向量**，这是线性变换的标准形式；
- “向量乘矩阵”必须使用行向量，本质是左乘列向量的转置对偶形式；
- 二者绝不能随意交换顺序，绝大多数情况下维度不匹配，运算本身不成立。

## 例题

```python
import sympy as sp

A = sp.Matrix([
    [1, 2, 1],
    [3, 8, 1],
    [0, 4, 1],
])
I = sp.eye(3)  # 单位矩阵 Identity Matrix
C = sp.Matrix([
    [1, 2, 1],
    [0, 2, -2],
    [0, 4, 1],
])
result = A * I
sp.pprint(result)
```

先把黑板上两个矩阵写清楚：
设左矩阵（原始）

$$
A=
\begin{bmatrix}
1 & 2 & 1 \\
3 & 8 & 1 \\
0 & 4 & 1
\end{bmatrix},\quad
B=
\begin{bmatrix}
1 & 2 & 1 \\
0 & 2 & -2 \\
0 & 4 & 1
\end{bmatrix}
$$

题目：求矩阵 $E$，使得 $EA = B$（左乘变换，行初等变换），或 $AE=B$（右乘，列变换）。
黑板是**行化简**（第二行做行变换），标准是左乘初等矩阵 $E$：
$$EA = B \implies E = BA^{-1}$$

#### 第一步：求 $A$ 的逆矩阵 $A^{-1}$

$$
A^{-1}=
\begin{bmatrix}
\dfrac{2}{5} & \dfrac{1}{5} & -\dfrac{3}{5}\\[4pt]
-\dfrac{3}{10} & \dfrac{1}{10} & \dfrac{1}{5}\\[4pt]
\dfrac{6}{5} & -\dfrac{2}{5} & \dfrac{1}{5}
\end{bmatrix}
$$

#### 第二步：计算 $E=BA^{-1}$

$$
E=BA^{-1}=
\begin{bmatrix}
1 & 2 & 1 \\
0 & 2 & -2 \\
0 & 4 & 1
\end{bmatrix}
A^{-1}=
\begin{bmatrix}
1 & 0 & 0 \\
-3 & 1 & 0 \\
0 & 0 & 1
\end{bmatrix}
$$

#### 验证 $EA=B$

$$
\begin{bmatrix}
1 & 0 & 0 \\
-3 & 1 & 0 \\
0 & 0 & 1
\end{bmatrix}
\begin{bmatrix}
1 & 2 & 1 \\
3 & 8 & 1 \\
0 & 4 & 1
\end{bmatrix}
=
\begin{bmatrix}
1 & 2 & 1 \\
0 & 2 & -2 \\
0 & 4 & 1
\end{bmatrix}=B
$$

### 解释这个矩阵 $E$ 的含义

$E$ 是**初等行变换矩阵**：左乘 $E$ 等价于对 $A$ 做行操作：
$$R_2 \leftarrow R_2 - 3R_1$$

1. 第 1 行不变：$[1,0,0]$
2. 第 2 行 = 原第 2 行 $-3\times$ 第 1 行：$[-3,1,0]$
3. 第 3 行不变：$[0,0,1]$

完全对应黑板上的变化：

- 第 1、3 行完全一样；
- 第二行：$[3,8,1]-3[1,2,1]=[0,2,-2]$，和右边矩阵第二行完全匹配。

### 总结

若要求**左乘一个矩阵得到右边结果**，这个变换矩阵是：

$$
\boldsymbol{E}=
\boxed{
\begin{bmatrix}
1 & 0 & 0 \\
-3 & 1 & 0 \\
0 & 0 & 1
\end{bmatrix}
}
$$

补充：如果你需要「右乘矩阵 $F$ 满足 $AF=B$」，解法是 $F=A^{-1}B$，和本题行变换场景无关，黑板明显是行化简，只用左乘 $E$。

### SymPy 完整求解过程（对应题目：找左乘矩阵 $E$ 使得 $EA=B$）

#### 1. 核心公式

已知 $EA = B$，两边右乘 $A^{-1}$：
$$E = B \cdot A^{-1}$$

#### 2. 完整代码（可直接复制运行）

```python
import sympy as sp

# 1. 定义黑板上的两个矩阵
A = sp.Matrix([
    [1, 2, 1],
    [3, 8, 1],
    [0, 4, 1],
])

B = sp.Matrix([
    [1, 2, 1],
    [0, 2, -2],
    [0, 4, 1],
])

# 2. 求 A 的逆矩阵
A_inv = A.inv()

# 3. 计算变换矩阵 E
E = B * A_inv

# 4. 打印结果
print("初等变换矩阵 E：")
sp.pprint(E)

# 5. 验证结果
print("\n验证 E·A：")
sp.pprint(E * A)
print("E·A 与 B 是否相等：", E * A == B)
```

#### 3. 输出结果解析

##### 变换矩阵 $E$

$$
E=
\begin{bmatrix}
1 & 0 & 0 \\
-3 & 1 & 0 \\
0 & 0 & 1
\end{bmatrix}
$$

- 含义：**初等行变换矩阵**，对应行操作 $R_2 \leftarrow R_2 - 3R_1$
- 第 1 行、第 3 行保持不变；第 2 行 = 原第 2 行 − 3×第 1 行，正好把 $A$ 的第二行 $[3,8,1]$ 变成 $[0,2,-2]$

##### 验证输出

`E·A` 完全等于目标矩阵 $B$，输出布尔值 `True`，证明结果正确。

#### 补充：如果需求是「右乘矩阵 $F$ 满足 $AF=B$」

公式改为 $F = A^{-1}B$，代码只需改一行：

```python
F = A_inv * B
sp.pprint(F)
```

黑板明显是**行变换**，因此用左乘 $E=BA^{-1}$ 是贴合题意的解法。

### 核心结论：对，矩阵乘积里**每一个元素**，都是「左矩阵的一行向量」和「右矩阵的一列向量」做点积

#### 1. 严格规则

设 $A$ 是 $m \times k$ 矩阵，$B$ 是 $k \times n$ 矩阵，乘积 $C=AB$ 是 $m\times n$：

$$
C_{ij} = \text{row}_i(A) \;\cdot\; \text{col}_j(B)
$$

- $\text{row}_i(A)$：$A$ 的第 $i$ 行，长度为 $k$ 的行向量
- $\text{col}_j(B)$：$B$ 的第 $j$ 列，长度为 $k$ 的列向量
- 二者维度必须匹配（长度都是 $k$）才能做点积，结果是一个标量，放在 $C$ 的第 $i$ 行第 $j$ 格。

#### 2. 举你刚才的初等矩阵例子直观验证

$$
E=\begin{bmatrix}1&0&0\\-3&1&0\\0&0&1\end{bmatrix},\quad
A=\begin{bmatrix}1&2&1\\3&8&1\\0&4&1\end{bmatrix}
$$

计算 $C=EA$ 的第二行第一列 $C_{21}$：
取 $E$ 第 2 行 $[-3,1,0]$，$A$ 第 1 列 $[1,3,0]$
点积：

$$
(-3)\times 1 + 1\times 3 + 0\times 0 = -3+3=0
$$

正好对应结果矩阵第二行第一个数字 $0$；
第二行第二列：$E$第 2 行 $[-3,1,0] \cdot A$第 2 列 $[2,8,4]$

$$
-3\times 2 + 1\times8 +0\times4 = -6+8=2
$$

也就是结果第二行第二个数字 $2$，完全吻合黑板上的 $[0,2,-2]$。

#### 3. 两个容易混淆的边界情况

##### （1）行向量 × 列向量（$1\times k$ 乘 $k\times1$）

本质就是纯向量点积，输出 1×1 标量矩阵，是矩阵乘法最小单元。

$$
\begin{bmatrix}a_1&a_2&a_3\end{bmatrix}
\begin{bmatrix}b_1\\b_2\\b_3\end{bmatrix}
= a_1b_1+a_2b_2+a_3b_3
$$

##### （2）列向量 × 行向量（$k\times1$ 乘 $1\times k$）

这是**外积 outer product**，不是点积，输出 $k\times k$ 矩阵，每个元素只是单纯相乘，没有求和。

$$
\begin{bmatrix}a_1\\a_2\end{bmatrix}\begin{bmatrix}b_1&b_2\end{bmatrix}
=\begin{bmatrix}a_1b_1 & a_1b_2\\a_2b_1 & a_2b_2\end{bmatrix}
$$

#### 4. 补充区分：矩阵乘法 ≠ 整体点积

只能说：**矩阵乘积的每个元素由行、列向量点积算出**，不能说“矩阵之间在做点积”。

- 点积：输入两个同长向量，输出标量；
- 矩阵乘积：输入两个矩阵，输出新矩阵，内部拆解为多组行/列点积。

#### 5. 代码层面对应（SymPy/Numpy）

`M.dot(N)` / `M @ N` 就是标准矩阵乘法，底层自动循环所有行、列做向量点积；
而 `M * N` 是逐元素哈达玛积，完全不涉及点积求和。

## 使用 rref 解线性方程组

```python
import sympy as sp

x, y = sp.symbols("x y")
# 构造增广矩阵 [A|b]
aug = sp.Matrix([
    [1, 1, 3],
    [2, -1, 0],
])
# 高斯消元得到行最简形
rref_mat, pivot = aug.rref()
print("行最简阶梯矩阵：\n")
sp.pprint(rref_mat)
print("主元列：", pivot)
```

### 算出的增广矩阵的最后一列就是解？

分情况，**只有满秩、唯一解的时候，rref 最后一列才是解；有无穷解/无解不能直接拿最后一列当答案**。

#### 1. 唯一解（系数矩阵满秩，主元覆盖所有变量）

方程组

$$
\begin{cases}
x+y=3\\
2x-y=0
\end{cases}
$$

增广矩阵：

$$
\left[\begin{array}{cc|c}
1 & 1 & 3\\
2 & -1 & 0
\end{array}\right]
$$

消元 rref 结果：

$$
\left[\begin{array}{cc|c}
1 & 0 & 1\\
0 & 1 & 2
\end{array}\right]
$$

左边单位矩阵，**最后一列 [1,2] 就是 $x=1,y=2$**，这时候可以直接取最后一列。

#### 2. 无解（出现矛盾行 $0\ 0\cdots\mid c,\ c\neq0$）

$$
\begin{cases}
x+y=2\\
2x+2y=5
\end{cases}
$$

增广矩阵 rref：

$$
\left[\begin{array}{cc|c}
1 & 1 & 2\\
0 & 0 & 1
\end{array}\right]
$$

第二行 $0x+0y=1$，矛盾，**不存在解，最后一列没有意义**。

#### 3. 无穷多解（秩 < 变量数，有自由未知量）

$$
\begin{cases}
x+y=2\\
2x+2y=4
\end{cases}
$$

rref：

$$
\left[\begin{array}{cc|c}
1 & 1 & 2\\
0 & 0 & 0
\end{array}\right]
$$

只有一个主元，$y$ 是自由变量，方程 $x = 2 - y$。
这时**不能直接把最后一列 [2,0] 当成 x、y 的解**，0 完全无效，需要设参数写通解。

#### 4. 通用判断规则

设增广矩阵 $[A\mid b]$，rref 后分块 $[R\mid d]$：

1. 若左边 $R$ 是单位矩阵 → $d$ 就是唯一解；
2. 若存在行 $[0,0,\dots,0\mid k],k\neq0$ → 无解；
3. 若左边有全零行、主元数量少于变量个数 → 无穷多解，要结合自由变量构造通解，不能直接取 $d$。

#### 补充示例代码验证

```python
import sympy as sp

x, y = sp.symbols("x y")
aug = sp.Matrix([
    [1, 1, 3],
    [2, -1, 0],
])
r, _ = aug.rref()
sp.pprint(r)
# 最后一列 r[:, -1] 就是解向量
print("解向量：", r[:, -1])
```

### 我怎么知道有没有解，怎么知道最后一列能不能用？

#### 一套完整判断流程（SymPy rref 增广矩阵专用）

设增广矩阵消元后：$\boldsymbol{M} = [R \mid d]$
左边块$R$是系数行最简，最后一列$d$是常数项。

##### 第一步：先找矛盾行 → 无解

遍历每一行，满足下面条件就是**无解**：

> 该行左边所有系数全是 0，但最后一列数字≠0
> 形式：$[0,0,\dots,0 \mid c],\ c\neq 0$

例子：

$$
\begin{bmatrix}
1 & 1 & 2 \\
0 & 0 & 1
\end{bmatrix}
$$

第二行 $0x+0y=1$，矛盾，直接判定无解，**最后一列完全没用**。

###### 代码快速检查矛盾行

```python
import sympy as sp

aug = sp.Matrix([
    [1, 1, 2],
    [2, 2, 5],
])
r, pivot = aug.rref()
rows = r.shape[0]
cols = r.shape[1]
has_no_sol = False

for i in range(rows):
    row = r[i, :]
    # 前 n 列系数全 0
    all_zero_coeff = all(entry == 0 for entry in row[:-1])
    const = row[-1]
    if all_zero_coeff and const != 0:
        has_no_sol = True
        break

print("是否无解：", has_no_sol)
```

##### 第二步：没有矛盾行，分两种情况

###### 情况 A：唯一解 → 最后一列直接用

判断标准：
主元数量 = 变量个数（系数矩阵列数）
等价：左边$R$左上角是单位矩阵。

示例 2 变量：

$$
\begin{bmatrix}
1 & 0 & 1 \\
0 & 1 & 2
\end{bmatrix}
$$

主元 2 个 = 变量 2 个，$x=1,y=2$，最后一列就是解向量。

###### 情况 B：无穷多解 → 不能直接拿最后一列当答案

判断标准：
主元数量 ＜ 变量个数
存在自由未知量，需要设参数写通解。

示例：

$$
\begin{bmatrix}
1 & 1 & 2 \\
0 & 0 & 0
\end{bmatrix}
$$

主元只有 1 个 < 变量 2 个。
只能读出 $x=2-y$，$y$ 自由变量，不能直接取$[2,0]$当 x、y。

#### 极简判断口诀（一眼看懂）

1. 有行：$0\ 0\cdots\mid c\neq0$ → **无解，最后一列作废**
2. 无矛盾行，主元数 = 变量数 → **唯一解，最后一列就是解**
3. 无矛盾行，主元数 ＜ 变量数 → **无穷多解，不能直接用最后一列**

#### 快速取主元数量代码

```python
r, pivot = aug.rref()
num_pivot = len(pivot)  # 主元个数
n_var = aug.shape[1] - 1  # 变量数量 = 总列数 -1
```

对比 `num_pivot` 和 `n_var` 即可区分唯一/无穷解。

#### 完整示例三合一

```python
import sympy as sp


def judge_solution(aug_mat):
    r, pivot = aug_mat.rref()
    rows, cols = r.shape
    n_var = cols - 1
    # 判断无解
    for i in range(rows):
        coeff_part = r[i, :-1]
        const = r[i, -1]
        if all(c == 0 for c in coeff_part) and const != 0:
            return "无解，最后一列无效", r
    num_p = len(pivot)
    if num_p == n_var:
        sol_vec = r[:, -1]
        return f"唯一解，解向量={sol_vec}", r
    else:
        return "无穷多解，不能直接使用最后一列，需设自由变量", r


# 测试 1：唯一解
aug1 = sp.Matrix([
    [1, 1, 3],
    [2, -1, 0],
])
print(judge_solution(aug1))

# 测试 2：无解
aug2 = sp.Matrix([
    [1, 1, 2],
    [2, 2, 5],
])
print(judge_solution(aug2))

# 测试 3：无穷多解
aug3 = sp.Matrix([
    [1, 1, 2],
    [2, 2, 4],
])
print(judge_solution(aug3))

print(
    "\naug1.rref:",
    aug1.rref(),
    "\naug2.rref:",
    aug2.rref(),
    "\naug3.rref:",
    aug3.rref(),
)
```

### 是不是一般用 A.solve(b)，不太直接用 rref 来求解？

#### 结论先说：日常做题/写代码分场景，各有分工

##### 1. `A.solve(b)`：**工程、计算、只求结果首选**，平时绝大多数情况用它

###### 适用场景

方程 $A\boldsymbol x = \boldsymbol b$，系数方阵可逆（唯一解）

```python
import sympy as sp

A = sp.Matrix([
    [1, 1],
    [2, -1],
])
b = sp.Matrix([3, 0])
x = A.solve(b)
print(x)
# Matrix([[1], [2]])
```

优点：

1. 一行出解，不用手动构造增广、看 rref、判断主元；
2. 自动内部高斯消元封装好，不用自己处理逻辑；
3. 输出纯解向量，直接取用。

###### 局限（很重要）

1. **只适用于方阵且可逆（唯一解）**
   - 长方形矩阵（方程数≠未知数）、奇异矩阵（无穷解/无解）会直接报错；
2. 看不到消元过程，不能用来手算教学、分析秩、自由变量。

##### 2. `rref()` 增广矩阵消元：**教学、分析有无穷/无解、看消元步骤专用**

###### 什么时候必须用 rref？

1. 方程个数 ≠ 未知数个数（长方形方程组）；
2. 奇异矩阵，有无穷多解，需要写出通解、找自由变量；
3. 判断方程组是否无解；
4. 学习线性代数，想看每一步行变换、主元位置、秩。

示例（无穷多解，A.solve(b) 会报错，只能 rref）

$$
\begin{cases}
x+y=2\\
2x+2y=4
\end{cases}
$$

```python
aug = sp.Matrix([
    [1, 1, 2],
    [2, 2, 4],
])
R, piv = aug.rref()
sp.pprint(R)
```

##### 3. 补充两个中间工具

1. `sp.solve(方程组列表, 变量)`
   兼顾符号方程，线性/非线性都能解，自动处理唯一/无穷/无解，适合带符号参数的方程组；
2. `solve_linear_system(增广矩阵, x,y,z...)`
   基于 rref 封装，输入增广矩阵直接返回解字典，介于`A.solve(b)`和裸 rref 之间。

##### 4. 使用场景对比总结

| 需求                                                | 推荐写法               |
| --------------------------------------------------- | ---------------------- |
| 方阵可逆，只求答案，不想管消元过程                  | `A.solve(b)`（最省事） |
| 长方形方程组、奇异矩阵、判断无解/无穷解、看消元步骤 | 构造增广矩阵 `.rref()` |
| 带符号参数、混合线性/非线性方程                     | `sp.solve()`           |

##### 一句话总结

单纯算数值唯一解：优先 `A.solve(b)`；
要分析方程组结构、看消元过程、处理奇异/长方形方程组：必须用增广矩阵 + rref。

### solve_linear_system 完整解释

#### 1. 函数作用

`sp.solve_linear_system(aug_matrix, var1, var2, var3...)`
专门**基于高斯行消元（rref）** 求解线性方程组，输入**增广矩阵**，直接返回「变量：解」的字典。
等价流程：
增广矩阵 → rref 消元 → 自动解析行最简矩阵 → 整理解

#### 2. 和其他方法区别

##### ① 和 A.solve(b) 对比

- `A.solve(b)`：只**接受方阵** A、右端向量 b，**只能处理可逆方阵唯一解**，奇异/长方形矩阵直接报错；
- `solve_linear_system`：输入增广矩阵，支持长方形、奇异、无解、无穷多解，功能更强。

##### ② 和 手动 rref() 对比

手动 rref 需要自己写代码判断主元、自由变量、矛盾行；
`solve_linear_system` 把这套判断逻辑封装好了，直接出结果。

##### ③ 和 sp.solve() 对比

- `sp.solve()`：既能解线性，也能解非线性方程；
- `solve_linear_system`：**只解线性方程组**，底层纯行消元，线性代数专用工具。

#### 3. 语法示例

方程组：

$$
\begin{cases}
x+y=3\\
2x-y=0
\end{cases}
$$

```python
import sympy as sp

x, y = sp.symbols("x y")
# 构造增广矩阵 [A|b]
aug = sp.Matrix([
    [1, 1, 3],
    [2, -1, 0],
])
# 传入增广矩阵 + 所有未知数
res = sp.solve_linear_system(aug, x, y)
# solve_linear_system 强制要求手动把变量符号依次传入，不能只丢矩阵，底层断言卡死了，没法省略。
# 只想用矩阵求解，不用管符号 → A.solve(b)
# system of linear equations = 线性方程组
# 日常简写直接叫 linear system，数学圈内默认就是指一联立的方程组，没有别的含义。
print(res)  # {x: 1, y: 2}
```

#### 4. 三种情况返回结果

##### （1）唯一解：返回完整字典

```text
{x:1, y:2}
```

##### （2）无穷多解：用自由变量表示表达式

方程组 $x+y=2, 2x+2y=4$

```python
aug = sp.Matrix([
    [1, 1, 2],
    [2, 2, 4],
])
print(sp.solve_linear_system(aug, x, y))
# {x: 2 - y}
```

意思：y 是自由变量，x 由 y 表示。

##### （3）无解：返回 None

方程组 $x+y=2, 2x+2y=5$

```python
aug = sp.Matrix([
    [1, 1, 2],
    [2, 2, 5],
])
print(sp.solve_linear_system(aug, x, y))
# None
print(aug.rref())
# 核心判断口诀：增广矩阵 rref 里出现「全 0 系数 + 非零常数」行 = 无解。
```

看到返回 `None` 直接判定无解。

适合你正在学高斯消元、做线性代数习题：

1. 手写增广矩阵习惯，不想拆分 A 和 b；
2. 需要处理方程数≠未知数、奇异矩阵；
3. 想省去手动分析 rref 最简矩阵的步骤；

`solve_linear_system` = **增广矩阵专用线性方程组求解器**，底层就是 rref 通用消元法，自动判断无解/唯一/无穷解。

### RREF 主元列的清零：主元负责上面所有主元列的清零；然后前面的主元再负责它的上面的主元列所有的清零

对，你总结得完全到位，理顺完整流程：
整个 RREF 分两大阶段：

#### 阶段 1：从上往下消（造阶梯 REF）

只拿当前行的主元，**消掉它下面所有行**同列的数字，只保证下一行主元靠右。
这一步只管下方，不管上方，此时主元上面还留着数字。

#### 阶段 2：从下往上反向消（化成最简 RREF，就是你说的逻辑）

从**最底下一行的主元**开始，逐层往上处理：

1. 最底层主元：把它所在列，**所有上面每一行**的该位置全部消为 0；
2. 往上走一行的主元：再把它所在列，**它上方所有行**同列消为 0；
3. 一路循环到第一行，每一列只留主元位置是 1，本列其余全 0。

##### 举三阶直观例子

阶梯形（REF）：

$$
\begin{bmatrix}
1 & 2 & 3 & 4 \\
0 & 1 & 5 & 6 \\
0 & 0 & 1 & 7
\end{bmatrix}
$$

1. 先处理最下方主元 (3,3)=1：
   $r_1=r_1-3r_3,\; r_2=r_2-5r_3$

$$
\begin{bmatrix}
1 & 2 & 0 & -17 \\
0 & 1 & 0 & -29 \\
0 & 0 & 1 & 7
\end{bmatrix}
$$

第三列全部清零（只剩主元 1）2. 再处理中间主元 (2,2)=1：
$r_1=r_1-2r_2$

$$
\begin{bmatrix}
1 & 0 & 0 & 41 \\
0 & 1 & 0 & -29 \\
0 & 0 & 1 & 7
\end{bmatrix}
$$

第二列上方清零，完成 RREF。

##### 核心口诀

下主元清上方整列，上主元再清自己上方整列，由下到上逐层清扫。

### `subtract 3 * row1 from row2`

#### 直译与矩阵行变换定义

`subtract 3 * row1 from row2`
标准行操作记号：$R_2 = R_2 - 3R_1$

1. **row1 = 第 1 行，row2 = 第 2 行**
2. `3 * row1`：把第 1 行每个元素整体乘以 3
3. `subtract A from B` = 用 B 减去 A → $B - A$
   所以是：**用第 2 行，减去三倍的第 1 行，结果覆盖原来的第 2 行**

#### 举个实例

设矩阵两行：

$$
R_1 = \begin{bmatrix}1 & 2\end{bmatrix},\quad R_2=\begin{bmatrix}4 & 5\end{bmatrix}
$$

1. 计算 $3R_1 = [3,\ 6]$
2. $R_2 - 3R_1 = [4-3,\ 5-6] = [1,\ -1]$
3. 新的第 2 行变为 $[1,\ -1]$，第 1 行保持不变

#### 易混淆区分（关键）

- subtract $3R_1$ from $R_2$ → $R_2 = R_2 - 3R_1$
- subtract $R_2$ from $3R_1$ → $R_1 = 3R_1 - R_2$
  介词 `from` 后面的那一行是**被减数**。

### 这一部分是讲的行变换吗？

#### 是的，完整拆解这块黑板内容

##### 1. 核心主题：初等行变换（`subtract 3*row1 from row2`）

黑板标题：`Matrices: Subtract 3*row1 from row2`
操作定义：$R_2 = R_2 - 3R_1$，属于**第三种初等行变换**：某一行加上另一行的常数倍。

##### 2. 左边矩阵：初等行变换矩阵（单位矩阵改造而来）

$$ E=
\begin{bmatrix}
1 & 0 & 0 \\
-3 & 1 & 0 \\
0 & 0 & 1
\end{bmatrix}
$$

单位矩阵 $I$ 把第二行第一列改成 $-3$，这个矩阵左乘任意矩阵，等价于对该矩阵执行行变换：$R_2 \leftarrow R_2 - 3R_1$。

##### 3. 中间原始矩阵 $A$

$$ A=
\begin{bmatrix}
1 & 2 & 1 \\
3 & 8 & 1 \\
0 & 4 & 1
\end{bmatrix}
$$

##### 4. 乘法 $EA$ = 对 $A$ 做行变换，得到右侧结果矩阵

逐行计算验证：

- 第 1 行不变：$[1,2,1]$
- 第 2 行 = 原第 2 行 $-3\times$ 第 1 行：
  $[3,8,1] - 3[1,2,1] = [3-3,\ 8-6,\ 1-3] = [0,2,-2]$
- 第 3 行不变：$[0,4,1]$

输出结果和黑板完全一致：

$$
\begin{bmatrix}
1 & 2 & 1 \\
0 & 2 & -2 \\
0 & 4 & 1
\end{bmatrix}
$$

##### 5. 关键知识点（Strang 经典内容）

1. **初等矩阵左乘 = 行变换**
   单位矩阵修改一行得到初等矩阵，放在左边相乘，等价于对右侧矩阵做对应行操作；
2. 你刚才问的英文指令 `subtract 3*row1 from row2`，直接对应这个 $R_2=R_2-3R_1$ 操作；
3. 这套逻辑刚好衔接你之前的认知：**左乘算子操纵矩阵的行**，完美呼应你卷积窗口/权重的理解框架。

总结：这一段完整讲解「初等行变换 ↔ 初等矩阵左乘」的等价关系，是线性代数消元法（高斯消元）的核心基础。

`subtract 2 * row2 from row3`

这里 $X=2\cdot row2,\ Y=row3$
行变换记号：
$$R_3 = R_3 - 2R_2$$

#### 2. 构造对应的初等消元矩阵（Sympy 逻辑）

设矩阵是 $n$ 阶方阵，下标从 0 开始：

- row3 → Python 下标 `2`
- row2 → Python 下标 `1`
  单位矩阵 $I$，修改位置 $E[2, 1] = -2$

3 阶消元矩阵示例：

$$
E=
\begin{bmatrix}
1 & 0 & 0 \\

$$

#### 3. Sympy 代码实现

##### 方式 1：手动构造消元矩阵

import sympy as sp

E = sp.eye(3)

# 见后面详解；这种方式与 strang 讲的更一致

##### 方式 2：用内置 `elementary_row_op` 生成

```python
import sympy as sp

# 生成初等消元矩阵 E
E = sp.eye(3).elementary_row_op(
    op="n->n+km",
    # op 就是单词 operation 的缩写，中文意思：运算、操作。
    # "n->n+km"：倍加行操作（某一行加上另一行的倍数，高斯消元最常用）
    # "n->kn"：行缩放操作（一行整体乘以常数）
    # "n<->m"：行交换操作（两行互换位置）
    row=2,  # 目标行：第 3 行（下标 2）
    row2=1,  # 基准行：第 2 行（下标 1）
    k=-2,
)
sp.pprint(E)
```

#### 4. 运算示例演示

取原矩阵：

$$ A=
\begin{bmatrix}
1 & 2 & 1 \\
0 & 2 & -2 \\
0 & 4 & 1
\end{bmatrix}
$$

执行 $R_3 = R_3 - 2R_2$：

$$
\begin{align*}
\text{新}R_3 &= [0,4,1] - 2\cdot[0,2,-2] \\
&= [0,4,1] - [0,4,-4] \\
&= [0,0,5]
\end{align*}
$$

变换后矩阵：

$$
EA=
\begin{bmatrix}
1 & 2 & 1 \\
0 & 2 & -2 \\
0 & 0 & 5
\end{bmatrix}
$$

### `E[2, 1] = -2 # R3 = R3 - 2*R2`

#### 1. 先明确下标规则（Python / SymPy 统一：从 0 开始计数）

矩阵 `E` 是 3 阶单位矩阵：
行号：0 → R1，1 → R2，2 → R3
列号：0 → C1，1 → C2，2 → C3

`E[行下标, 列下标] = 值`
`E[2, 1] = -2`

- 第一个数字 `2` = **第 3 行（R3，目标修改行）**
- 第二个数字 `1` = **第 2 列（对应原矩阵的 R2 基准行）**
- 赋值 `-2` = 变换系数

#### 2. 单位矩阵原始样子

$$
E_{初始}=
\begin{bmatrix}
1 & 0 & 0 \\
0 & 1 & 0 \\
0 & 0 & 1
\end{bmatrix}
$$

执行 `E[2,1] = -2` 后，只修改第三行第二列的元素：

$$
E_{修改后}=
\begin{bmatrix}
1 & 0 & 0 \\
0 & 1 & 0 \\
0 & \boldsymbol{-2} & 1
\end{bmatrix}
$$

#### 3. 为什么这个矩阵左乘 A 等价 $R_3 = R_3 - 2R_2$？

设原矩阵 $A$ 三行：$R_1,\ R_2,\ R_3$
矩阵乘法 $E \cdot A$ 展开第三行结果：

$$
\text{新}R_3 = 0\cdot R_1 + (-2)\cdot R_2 + 1\cdot R_3 = R_3 - 2R_2
$$

##### 通用万能规律（所有消元矩阵都适用）

执行行变换：$\boldsymbol{R_{目标} = R_{目标} + k \cdot R_{基准}}$
消元矩阵构造规则：

1. 先生成同阶单位矩阵 `E = sp.eye(n)`
2. `E[目标行下标, 基准行下标] = k`

##### 套你两个例题验证

1. 黑板例题：$R_2 = R_2 - 3R_1$
   目标行 R2=下标 1，基准行 R1=下标 0，k=-3
   代码：`E[1, 0] = -3`
2. 当前例题：$R_3 = R_3 - 2R_2$
   目标行 R3=下标 2，基准行 R2=下标 1，k=-2
   代码：`E[2, 1] = -2`

#### 4. 一句话总结注释含义

```python
E[2, 1] = -2  # R3 = R3 - 2*R2
```

在消元矩阵第三行、第二列填入系数 -2，这个矩阵左乘任意矩阵，就会把该矩阵的第三行替换成「原第三行 减 2 倍第二行」。

## 例题 $E_{32}(E_{21}A)=u$

### 1. 公式含义拆解（Strang 课堂写法）

$$E_{32}(E_{21}A)=U$$

- $E_{21}$：初等消元矩阵，执行 $R_2 = R_2 - 3R_1$（你黑板第一个变换）
- $E_{32}$：初等消元矩阵，执行 $R_3 = R_3 - 2R_2$（你后面的变换）
- 矩阵乘法结合律：$E_{32}(E_{21}A) = (E_{32}E_{21})A$，先对$A$做$E_{21}$行变换，再做$E_{32}$行变换，最终得到上三角矩阵$U$

下标规则（数学 1 开始 → SymPy 下标 0 开始）：

- $E_{21}$：数学行 2、列 1 → 代码`E21[1, 0] = -3`
- $E_{32}$：数学行 3、列 2 → 代码`E32[2, 1] = -2`

### 2. SymPy 完整可运行代码

```python
import sympy as sp

sp.init_printing()

# 原矩阵 A（黑板例题矩阵）
A = sp.Matrix([
    [1, 2, 1],
    [3, 8, 1],
    [0, 4, 1],
])

# 构造消元矩阵 E21：R2 = R2 - 3R1
E21 = sp.eye(3)
E21[1, 0] = -3

# 构造消元矩阵 E32：R3 = R3 - 2R2
E32 = sp.eye(3)
E32[2, 1] = -2

# 公式 E32(E21 * A) = U
step1 = E21 * A  # 先执行 E21 变换
U = E32 * step1  # 再执行 E32 变换

print("E21 矩阵：")
sp.pprint(E21)
print("\nE32 矩阵：")
sp.pprint(E32)
print("\nE21 * A:")
sp.pprint(step1)
print("\nU = E32*(E21*A):")
sp.pprint(U)
```

### 3. 输出结果对应黑板

1. $E_{21}A$ 就是第一张黑板结果：

$$
\begin{bmatrix}
1 & 2 & 1 \\
0 & 2 & -2 \\
0 & 4 & 1
\end{bmatrix}
$$

2. $U=E_{32}(E_{21}A)$ 最终上三角矩阵：

$$
U=
\begin{bmatrix}
1 & 2 & 1 \\
0 & 2 & -2 \\
0 & 0 & 5
\end{bmatrix}
$$

### 4. 等价写法（合并两个消元矩阵）

利用结合律 $E_{32}E_{21}A=U$，效果完全相同：

```python
E_total = E32 * E21
U2 = E_total * A  # 结合律
# associative law（名词短语，课堂常用）
# 简称：associativity（名词，抽象概念）
sp.pprint(E_total)
sp.pprint(U2)  # 和上面 U 一模一样
```

$E_{\text{total}}=E_{32}E_{21}$ 是两个初等矩阵乘积，是 LU 分解里$L^{-1}$。

### 5. 对应 Strang 核心逻辑

1. 每次行变换 = 左乘一个$E$；
2. 多个行变换按**从右到左**顺序相乘，先做的变换放靠近$A$的一侧；
3. 全部消元矩阵左乘原矩阵，得到上三角$U$。

## 初等矩阵

### 三类初等矩阵（Elementary Matrix）

单位矩阵 $I$ 经过**一次初等行变换**得到的矩阵，统称初等矩阵；左乘矩阵等价做对应行变换，右乘等价做列变换。

#### 1. 交换两行：置换初等矩阵 $E_{ij}$

操作：交换单位矩阵第 $i$、第 $j$ 行。
例（3 阶，交换行 1、行 2）：

$$
E_{12}=
\begin{bmatrix}
0 & 1 & 0\\
1 & 0 & 0\\
0 & 0 & 1
\end{bmatrix}
$$

性质：$E_{ij}^{-1}=E_{ij}$，自逆。

#### 2. 某一行乘非零常数 $k$：倍乘初等矩阵 $E_i(k)$

操作：单位矩阵第 $i$ 行全部乘以 $k\neq0$。
例（3 阶，第 2 行×5）：

$$
E_2(5)=
\begin{bmatrix}
1 & 0 & 0\\
0 & 5 & 0\\
0 & 0 & 1
\end{bmatrix}
$$

逆矩阵：$E_i(k)^{-1}=E_i(\frac1k)$。

#### 3. 一行的 $k$ 倍加到另一行：消元初等矩阵 $E_{ij}(k)$（Strang 最常用）

操作：单位矩阵第 $j$ 行的 $k$ 倍加到第 $i$ 行；对应变换 $R_i \leftarrow R_i + kR_j$。

- 你课上的 $E_{21}(-3)$：$R_2 = R_2 -3R_1$

$$
E_{21}(-3)=
\begin{bmatrix}
1 & 0 & 0\\
-3 & 1 & 0\\
0 & 0 & 1
\end{bmatrix}
$$

- $E_{32}(-2)$：$R_3 = R_3 -2R_2$

$$
E_{32}(-2)=
\begin{bmatrix}
1 & 0 & 0\\
0 & 1 & 0\\
0 & -2 & 1
\end{bmatrix}
$$

逆矩阵：$E_{ij}(k)^{-1}=E_{ij}(-k)$，直接把系数取反。

### 核心规律（Strang 重点）

1. 左乘初等矩阵 = 对原矩阵做**行变换**；
2. 右乘初等矩阵 = 对原矩阵做**列变换**；
3. 任意可逆矩阵，都可以拆成若干初等矩阵的乘积。

## 逆矩阵

### 逆矩阵相当于标量的倒数？

#### 核心类比结论

**可以这么类比，但有严格边界，不能完全等同**

1. 标量：$a \cdot a^{-1} = 1$（倒数，乘积=乘法单位元 1）
2. 方阵：$A \cdot A^{-1} = I$（逆矩阵，乘积=矩阵乘法单位元$I$）

从「乘法抵消、还原成单位元」这个逻辑上，逆矩阵就是矩阵版本的倒数。

##### 一、完美对应的地方（类比成立）

###### 1. 单位元对应

- 数字乘法单位：$1$，任何数乘 1 不变 $a\cdot1=a$
- 矩阵乘法单位：单位矩阵 $I$，任何方阵乘$I$不变 $AI=IA=A$

###### 2. 逆的定义完全对齐

标量（$a\neq0$）：
$$a \cdot \frac{1}{a} = 1$$
方阵（$A$可逆，行列式$\det(A)\neq0$）：
$$A A^{-1} = A^{-1} A = I$$

###### 3. 乘除运算等价

标量除法等价乘倒数：$x \div a = x \cdot \frac{1}{a}$
矩阵“除法”等价乘逆矩阵：解 $AX=B$ 得 $X=A^{-1}B$，等价于 $X=B \div A$

##### 二、关键区别（不能直接照搬标量倒数规则，极易踩坑）

###### 1. 交换律失效（最大区别）

标量：$a \cdot \frac1a = \frac1a \cdot a$，左右乘无区别
矩阵：
$A A^{-1}=I$ 且 $A^{-1}A=I$（逆自身可交换）
但 $AB \neq BA$，所以**不能随便调换逆的左右位置**
例：$XA=B \implies X=BA^{-1}$，不能写成$A^{-1}B$

###### 2. 不是所有“矩阵”都有逆

标量：只有 $a=0$ 没有倒数
矩阵限制更多：

1. 必须是**方阵**（行列数相等），长方形矩阵一定无逆；
2. 方阵行列式$\det(A)=0$（奇异矩阵），无逆。

###### 3. 倒数公式不能直接套用

标量倒数：$\displaystyle \frac{1}{ab}=\frac1b \cdot \frac1a$
矩阵逆：$(AB)^{-1}=B^{-1}A^{-1}$，顺序要颠倒，和标量完全不一样。

###### 4. 不存在“除以矩阵”的原生运算

数字有 `/` 除法运算符；矩阵没有除法，只能靠左/右乘逆实现等效除法。

##### 三、极简记忆（Strang 课堂类比）

- 功能上：$\boldsymbol{A^{-1} \iff \dfrac1a}$，用来抵消乘法、还原单位元；
- 运算上：矩阵乘法不交换，逆的顺序必须颠倒，不能完全照搬数字倒数规则。

##### 举个 SymPy 例子直观对照

```python
import sympy as sp

# 数字
a = sp.Symbol("a")
print(a * (1 / a))  # 输出 1

# 矩阵
A = sp.Matrix([[1, 2], [3, 4]])
A_inv = A.inv()
sp.pprint(A * A_inv)  # 输出单位矩阵 I
```
