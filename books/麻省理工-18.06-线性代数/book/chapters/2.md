# Elimination with Matrices

利用矩阵进行消元

## 噢，sympy里LUsolve就是指定高斯消元法计算？

### 结论：是的，`LUsolve` 底层就是**高斯消元衍生的LU分解解法**，专门用高斯消元做矩阵分解来解方程 $A\boldsymbol{x}=\boldsymbol{b}$

#### 1. 底层逻辑对应MIT 18.06第2节课内容

1. 先对方阵 $A$ 做**高斯消元**，得到 $A=LU$
   - $L$：下三角（记录所有消元倍数）
   - $U$：高斯消元后的上三角矩阵
2. 拆分方程 $A\boldsymbol{x}=\boldsymbol{b}$ → $LU\boldsymbol{x}=\boldsymbol{b}$
   ① 前向回代解 $L\boldsymbol{y}=\boldsymbol{b}$
   ② 后向回代解 $U\boldsymbol{x}=\boldsymbol{y}$
   整个流程完全是这节课讲的**矩阵消元思路**。

#### 2. 和普通 `.solve(b)` 的区别

- `A.solve(b)`：SymPy自动选算法（克莱姆、高斯消元、LU等），不固定；
- `A.LUsolve(b)`：**强制只用LU分解（高斯消元路线）**，不会切换其他解法；
- 等价写法：`A.solve(b, method='LU')` 内部直接调用 `LUsolve`。

#### 3. 使用限制（贴合你之前学的知识点）

1. **只能用于可逆方阵**
   若 $\det(A)=0$、主元出现0，LU分解失败，直接报错；
2. 只处理方阵线性方程组 $A_{n\times n}\boldsymbol{x}_{n\times1}=\boldsymbol{b}_{n\times1}$，长方形矩阵不能用；
3. 优势：同一个A多次解不同b时，只分解一次LU，重复计算更快。

#### 4. 完整代码示例

```python
import sympy as sp
A = sp.Matrix([
    [2, -1, 0],
    [-1, 2, -1],
    [0, -3, 4]
])
b = sp.Matrix([0, -1, 4])

### 强制LU分解（高斯消元路线）求解
x = A.LUsolve(b)
sp.pprint(x)

### 等价写法
x2 = A.solve(b, method="LU")
print(x == x2) ### True
```

#### 5. 一句话区分概念

- 高斯消元：原始行变换操作，得到上三角 $U$；
- LU分解：把高斯消元的全过程打包成 $A=LU$；
- `LUsolve`：基于LU分解（高斯消元）的专用解方程函数。
