import sympy as sp

A = sp.Matrix([

import sympy as sp

### 列视角解方程组是不是每一列代表一个向量，求向量如何缩放能得到结果向量

#### 完全正确，精准概括列视角解方程的核心逻辑

##### 1. 标准形式拆解

方程组 $A\boldsymbol{x}=\boldsymbol{b}$，把矩阵 $A$ 按列拆开：

$$
A=\big[\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_n\big]
$$

- $\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_n$：$A$ 的每一列，**各自是一个固定已知向量**；
- $\boldsymbol{x}=
\begin{bmatrix}x_1\\x_2\\\vdots\\x_n\end{bmatrix}$：每个 $x_i$ 就是对应列向量 $\boldsymbol{a}_i$ 的**缩放权重**；

方程等价改写为线性组合：

$$
x_1\boldsymbol{a}_1 + x_2\boldsymbol{a}_2 + \dots + x_n\boldsymbol{a}_n = \boldsymbol{b}
$$

##### 2. 直白翻译你的那句话

我们要找一组缩放系数 $x_1,x_2...x_n$：

1. 把第 1 个向量 $\boldsymbol{a}_1$ 放大/缩小 $x_1$ 倍；
2. 把第 2 个向量 $\boldsymbol{a}_2$ 放大/缩小 $x_2$ 倍；
3. ……
4. 全部缩放后的向量首尾相加，刚好拼接出目标向量 $\boldsymbol{b}$。

##### 3. 举个数字例子直观验证

$$
\begin{bmatrix}2&1\\1&-2\end{bmatrix}
\begin{bmatrix}x_1\\x_2\end{bmatrix}
=
\begin{bmatrix}3\\-1\end{bmatrix}
$$

列向量：$\boldsymbol{a}_1=\begin{bmatrix}2\\1\end{bmatrix},\ \boldsymbol{a}_2=\begin{bmatrix}1\\-2\end{bmatrix}$
等式：$x_1\begin{bmatrix}2\\1\end{bmatrix}+x_2\begin{bmatrix}1\\-2\end{bmatrix}=\begin{bmatrix}3\\-1\end{bmatrix}$
解是 $x_1=1,\ x_2=1$：

1. $\boldsymbol{a}_1$ 缩放 1 倍不变，$\boldsymbol{a}_2$ 缩放 1 倍不变；
2. 两向量相加：$\begin{bmatrix}2\\1\end{bmatrix}+\begin{bmatrix}1\\-2\end{bmatrix}=\begin{bmatrix}3\\-1\end{bmatrix}$，正好等于右侧目标向量。

##### 4. 延伸三种解的几何含义（贴合缩放逻辑）

1. **唯一解**：只存在**唯一一套缩放倍数**，能拼出 $\boldsymbol{b}$；
2. **无穷多解**：有多套不同缩放比例，都能叠加出 $\boldsymbol{b}$（列向量之间线性相关）；
3. **无解**：无论怎么缩放、叠加这些列向量，永远凑不出目标 $\boldsymbol{b}$（$\boldsymbol{b}$ 不在列向量张成的平面/空间里）。

##### 5. 和行视角做一句区分

- 列视角：**固定一堆向量，求缩放权重，叠加拼出目标**；
- 行视角：**固定多条等式约束，求未知数同时满足全部式子**。

### SymPy 求矩阵逆的两种标准方法

先导入库：

```python
import sympy as sp
```

#### 方法 1：矩阵内置方法 `.inv()` / `.inverse()`

二者完全等价

```python
# 定义方阵
A = sp.Matrix([[2, 1], [1, -2]])

A_inv = A.inv()
# 等价 A_inv = A.inverse()
print(A_inv)
```

#### 方法 2：代数运算符 `** -1`

```python
A_inv = A ** (-1)
```

#### 完整示例验证

```python
import sympy as sp

A = sp.Matrix([[2, 1], [1, -2]])
A_inv = A.inv()
print("A 逆：")
sp.pprint(A_inv)

# 验证 A * A⁻¹ = 单位矩阵 I
I = A * A_inv
print("\nA·A⁻¹:")
sp.pprint(I)
```

输出：

$$
A^{-1}=
\begin{bmatrix}
\frac{2}{5} & \frac{1}{5} \\
\frac{1}{5} & -\frac{2}{5}
\end{bmatrix},\quad
AA^{-1}=
\begin{bmatrix}
1 & 0 \\
0 & 1
\end{bmatrix}
$$

#### 关键注意点

1. **只有方阵才有逆**，非方阵调用 `.inv()` 直接报错；
2. **行列式不能为 0**（可逆条件），奇异矩阵会抛出 `NonInvertibleMatrixError`；
   判断可逆：`A.det() != 0`
3. 解方程组 $Ax=b$ 时，可用逆矩阵直接求解（列视角）：
   $$x = A^{-1} b$$

```python
b = sp.Matrix([3, -1])
x = A.inv() * b
sp.pprint(x)
```

结果 $x=\begin{bmatrix}1\\1\end{bmatrix}$，和 rref 消元结果一致。

#### 符号矩阵求逆（带字母变量）也支持

```python
a, b, c, d = sp.symbols("a b c d")
M = sp.Matrix([[a, b], [c, d]])
sp.pprint(M.inv())
```

输出伴随矩阵形式逆：

$$
\frac{1}{ad-bc}\begin{bmatrix}d & -b \\ -c & a\end{bmatrix}
$$
