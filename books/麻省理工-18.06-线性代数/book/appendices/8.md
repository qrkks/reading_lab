## sympy 能解非齐次方程吗？

当然能。SymPy 可以解非齐次线性方程组

$$A\mathbf{x}=\mathbf{b},\qquad \mathbf{b}\ne\mathbf 0$$

例如：

```python
from sympy import Matrix
import sympy as sp

A = Matrix([
    [1, 2],
    [3, 4],
])
b = Matrix([5, 6])

x = A.solve_least_squares(b)  # 更常用的精确解见下面
sp.pprint(x)
```

> [x = A.solve_least_squares(b)](../appendices/8.md#x-asolve_least_squaresb)

如果是方阵且可逆，更直接：

```python
x = A.inv() * b
sp.pprint(x)
```

但一般不建议显式求逆，推荐：

```python
x = A.solve(b)
sp.pprint(x)
```

结果为：

```text
⎡-4 ⎤
⎢   ⎥
⎣9/2⎦

```

对于可能欠定、超定或含自由变量的系统，最适合用：

```python
from sympy import linsolve

sp.pprint(linsolve((A, b)))
```

例如一个有无穷多解的非齐次系统：

```python
A = Matrix([
    [1, 2, 3],
    [2, 4, 6],
])
b = Matrix([1, 2])

sp.pprint(linsolve((A, b)))
```

会返回类似：

```text
{(-2*tau0 - 3*tau1 + 1, tau0, tau1)}
```

也就是

$$ \mathbf{x}=
\begin{bmatrix}1\\0\\0\end{bmatrix}
+\tau_0
\begin{bmatrix}-2\\1\\0\end{bmatrix}
+\tau_1
\begin{bmatrix}-3\\0\\1\end{bmatrix}.
$$

这个结果正好体现了非齐次方程解集的结构：

$$\text{非齐次方程的通解}=\text{一个特解}+\text{对应齐次方程的零空间}.$$

如果系统无解，`linsolve((A, b))` 会返回 `EmptySet`。因此你现在学 Strang 的 $Ax=b$，用 `linsolve((A, b))` 最能观察“无解、唯一解、无穷多解”这三个核心情形。

### x = A.solve_least_squares(b)

#### `x = A.solve_least_squares(b)` 完整解析

##### 1. 核心含义

这是**线性最小二乘求解**的矩阵 API 调用，目标：给定矩阵 $A\in\mathbb{R}^{m\times n}$、向量 $b\in\mathbb{R}^m$，求向量 $x$ 使得残差平方和最小：

$$
\min_x \|Ax - b\|_2^2
$$

- $m$：方程数量（观测数），$n$：未知量个数
- 典型场景：$m>n$（超定方程组，无解，只能找最优近似解）；也兼容欠定/方阵

##### 2. 和普通精确解方程 `solve` 的区别（注释里“更常用的精确解”）

###### （1）普通精确解 `A.solve(b)`

适用前提：
$A$ 是**方阵且满秩**（$m=n,\det(A)\neq0$），方程组 $Ax=b$ 存在唯一精确解。
底层算法：LU 分解、高斯消元，直接得到 $x=A^{-1}b$。
限制：

- 非方阵、奇异矩阵、超定方程组直接报错/数值崩溃；
- 仅适用于有唯一精确解的场景。

###### （2）最小二乘 `solve_least_squares(b)`

无严格方阵限制，通用：

1. **超定 $m>n$**：实验拟合、回归（最常用），无精确解，找误差最小近似；
2. **方阵满秩 $m=n$**：等价于精确解，数值稳定性通常比直接求逆更好；
3. **欠定 $m<n$**：无穷多解，返回范数最小的那个解。

底层主流算法：**QR 分解**（数值稳定，优先）、SVD 奇异值分解（处理奇异/病态矩阵鲁棒），不会直接计算矩阵逆，避免数值误差爆炸。

##### 3. 数学等价写法（手动推导最小二乘）

正规方程（仅作数学理解，代码不推荐直接用，易病态）：

$$
A^T A x = A^T b \quad\Rightarrow\quad x=(A^T A)^{-1}A^T b
$$

- `solve_least_squares` 内部不会用上面的求逆形式，而是 QR/SVD，数值精度远高于手动算 $A^TA$ 求逆；
- 注释“更常用的精确解”指方阵满秩场景直接用 `.solve(b)`，运算更快；超定拟合场景必须用最小二乘。

##### 4. 适用库举例（语法来源）

这类面向矩阵的 `.solve_least_squares` 方法常见：

1. Python `scipy.linalg.lstsq` / `numpy.linalg.lstsq`（函数式）；
2. Eigen C++ 矩阵库：`A.leastSquaresSolve(b)`；
3. Matlab：`x = A\b`（自动区分最小二乘/精确解）；
4. SymPy/线性代数数值框架封装为 `.solve_least_squares` 成员方法。

##### 5. 举例场景

###### 场景 1：线性回归（超定，必须最小二乘）

100 个样本（$m=100$），2 个特征（$n=2$），$A_{100\times2}$，标签向量 $b_{100\times1}$

```python
# 拟合直线 y = x1*w1 + x2*w2
w = A.solve_least_squares(b)
```

无法精确满足 100 个方程，只能最小化预测误差平方。

###### 场景 2：方阵满秩（两种方法都可用）

$A_{3\times3}$ 可逆，$b_{3\times1}$

```python
x1 = A.solve(b)  # 精确解，更快
x2 = A.solve_least_squares(b)  # 结果一样，多一步分解，略慢
```

此时注释里“更常用的精确解”就是指 `.solve()`。

##### 6. 关键总结

1. `solve_least_squares`：通用拟合工具，处理**无解、多解、唯一解**全部情况，核心是最小化平方误差；
2. `.solve(b)`：仅方阵可逆时用，求严格精确解，计算开销更小；
3. 注释 `# 更常用的精确解见下面` 提示读者：如果你的方程组是方阵且可逆，优先用普通`solve`，最小二乘留给拟合/超定问题。

## 如果要方程组有解，b 需要符合什么条件？

对于方程组

$$A\mathbf{x}=\mathbf{b},$$

要有解，核心条件只有一个：

$$\boxed{\mathbf b\in C(A)}$$

也就是：$\mathbf b$ 必须属于 $A$ 的列空间。

因为如果把 $A$ 按列写成

$$A=[\mathbf a_1\ \mathbf a_2\ \cdots\ \mathbf a_n],$$

那么

$$A\mathbf x=x_1\mathbf a_1+x_2\mathbf a_2+\cdots+x_n\mathbf a_n.$$

所以求解 $A\mathbf x=\mathbf b$，其实是在问：

> 能不能用 $A$ 的各列进行线性组合，得到 $\mathbf b$？

如果能，方程组有解；如果不能，就无解。

从消元角度看，等价条件是增广矩阵 $[A\mid\mathbf b]$ 消元后，不能出现：

$$[0\quad0\quad\cdots\quad0\mid c],\qquad c\ne0,$$

因为这一行代表

$$0=c,$$

产生矛盾。

也可以用秩来表达：

$$\boxed{\operatorname{rank}(A)=\operatorname{rank}([A\mid\mathbf b])}.$$

三个说法完全等价：

- 几何上：$\mathbf b\in C(A)$；
- 消元上：不会出现 $0=c$；
- 秩上：$\operatorname{rank}(A)=\operatorname{rank}([A\mid\mathbf b])$。

还有一个很漂亮、之后学到左零空间会更清楚的条件：

$$\boxed{\mathbf y^T\mathbf b=0\quad\text{对所有满足}\quad A^T\mathbf y=0\text{ 的 }\mathbf y}.$$

意思是：$\mathbf b$ 必须垂直于 $A^T$ 的零空间。因为

$$C(A)=N(A^T)^\perp.$$

你现在刚学到零空间的话，先牢牢抓住一句即可：

> $A\mathbf x=\mathbf b$ 有解，当且仅当 $\mathbf b$ 是 $A$ 的列向量的线性组合。

## 我们看 AX 等于 B 有没有解，就是它的可解性吗？就是讨论它的可解性吗？

### 一、核心结论

判断线性方程组 \(A\boldsymbol{x}=\boldsymbol{b}\) **有没有解**，本质就是讨论方程组的**可解性**，二者完全是一回事。
可解性只分两种情况：有解、无解；有解又细分唯一解、无穷多解。

#### 1. 判定定理（秩判别法，最通用）

设：

- \(A\) 是 \(m\times n\) 系数矩阵
- \(\overline{A}=(A\mid \boldsymbol{b})\) 是增广矩阵

1. **无解**：\(r(A) < r(\overline{A})\)
2. **有解**：\(r(A) = r(\overline{A})\)
   - 唯一解：\(r(A)=r(\overline{A})=n\)（未知数个数）
   - 无穷多解：\(r(A)=r(\overline{A})<n\)

#### 2. 几何直观理解

\(A\boldsymbol{x}\) 等价于**矩阵列向量的线性组合**：
\[
A\boldsymbol{x}=x_1\boldsymbol{\alpha}_1+x_2\boldsymbol{\alpha}_2+\dots+x_n\boldsymbol{\alpha}_n=\boldsymbol{b}
\]
可解性翻译：向量 \(\boldsymbol{b}\) **能否被 \(A\) 的列向量线性表示**。

- 能表示 → 有解；
- 不能表示 → 无解。

#### 3. 方阵特殊情形（\(A\) 是 \(n\) 阶方阵）

此时不用看增广矩阵也能快速判断：

- \(|A|\neq0\)，\(r(A)=n\)：必有唯一解；
- \(|A|=0\)，\(r(A)<n\)：再对比 \(r(A)\) 和 \(r(\overline{A})\)，可能无解或无穷多解。

### 二、举个简单例子

\[
\begin{cases}
x_1+x_2=1\\
2x_1+2x_2=3
\end{cases}
\]
系数矩阵 \(A=\begin{pmatrix}1&1\\2&2\end{pmatrix},\ r(A)=1\)
增广矩阵 \(\overline{A}=\begin{pmatrix}1&1&1\\2&2&3\end{pmatrix},\ r(\overline{A})=2\)
\(r(A)<r(\overline{A})\) → 无解，即可解性为“不可解”。

### 三、补充：齐次方程组 \(A\boldsymbol{x}=\boldsymbol{0}\)

齐次方程永远满足 \(r(A)=r(\overline{A})\)（右边零向量一定能被列向量组合表示），**一定有解**，不存在无解情况：

- \(r(A)=n\)：只有零解；
- \(r(A)<n\)：有无穷多非零解。

简单总结：讨论 \(A\boldsymbol{x}=\boldsymbol{b}\) 是否存在向量 \(\boldsymbol{x}\) 满足等式，就是在讨论该方程组的可解性，秩相等是有解的充要条件。

## $Ax=b \text{ solvable when } b \text{ is in } C(A)$

### 图片文字提取与完整解读

#### 原文

$$Ax=b \text{ solvable when } b \text{ is in } C(A)$$

##### 翻译

线性方程组 \(A\boldsymbol{x}=\boldsymbol{b}\) 有解（可解）的充要条件是：向量 \(\boldsymbol{b}\) 属于矩阵 \(A\) 的列空间 \(C(A)\)。

#### 1. 符号释义

- \(C(A)\)：Column Space，**列空间**
  定义：矩阵 \(A\) 所有列向量的全部线性组合构成的向量空间
  \[
  C(A)=\operatorname{span}\{\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_n\},\quad A=[\boldsymbol{a}_1\ \boldsymbol{a}_2\ \dots\ \boldsymbol{a}_n]
  \]
- solvable：可解的，指方程组存在至少一个解 \(\boldsymbol{x}\)

#### 2. 几何本质

方程 \(A\boldsymbol{x}=x_1\boldsymbol{a}_1+x_2\boldsymbol{a}_2+\dots+x_n\boldsymbol{a}_n=\boldsymbol{b}\)
等价于：**能否找到一组系数 \(x_1,\dots,x_n\)，用 \(A\) 的列向量线性拼出向量 \(\boldsymbol{b}\)**。

- 若 \(\boldsymbol{b}\) 在列空间 \(C(A)\) 内 → 能拼出 → 方程组有解
- 若 \(\boldsymbol{b}\) 不在列空间 \(C(A)\) 内 → 无法拼出 → 方程组无解

#### 3. 和秩判别法的等价关系

这条空间表述和秩判定定理完全等价：
\[
\boldsymbol{b}\in C(A) \iff r(A)=r(\overline{A})
\]
\(\overline{A}=(A\mid \boldsymbol{b})\) 为增广矩阵。

#### 4. 细分解的情况

1. \(\boldsymbol{b}\in C(A)\) 且 \(\dim C(A)=n\)（\(A\) 列满秩）：**唯一解**
2. \(\boldsymbol{b}\in C(A)\) 且 \(\dim C(A)<n\)（\(A\) 列不满秩）：**无穷多解**
3. \(\boldsymbol{b}\notin C(A)\)：**无解**

#### 5. 补充术语区分

- \(C(A)\)：列空间（Column Space），对应右侧向量 \(\boldsymbol{b}\) 的存在范围，用于判定 \(A\boldsymbol{x}=\boldsymbol{b}\) 可解性
- \(N(A)\)：零空间（Null Space），齐次方程 \(A\boldsymbol{x}=\boldsymbol{0}\) 的全部解构成的空间，用于刻画解的自由度

💡 总结：这句话是线性方程组可解性最核心的**空间语言表述**，从向量组合的几何角度解释了“什么时候方程有解”。

## If a combination of rows of A gives zero row then Same combination of b must give zero (number)

### 完整翻译、数学推导与课堂逻辑解读

#### 一、句子完整翻译

**If a combination of rows of A gives zero row then Same combination of b must give zero (number)**
翻译：
如果对矩阵 \(A\) 的各行取某一组线性组合能得到零行，那么对右端向量 \(\boldsymbol{b}\) 的分量取完全相同的线性组合，结果必须等于数字 0。

#### 二、严格数学表达

设矩阵 \(A\) 是 \(m\times n\) 矩阵，行向量 \(\boldsymbol{r}_1,\boldsymbol{r}_2,\dots,\boldsymbol{r}_m\)；向量 \(\boldsymbol{b}=\begin{bmatrix}b_1\\b_2\\\vdots\\b_m\end{bmatrix}\)。
已知存在不全为 0 的系数 \(c_1,c_2,\dots,c_m\)，满足：
\[
c_1\boldsymbol{r}_1 + c_2\boldsymbol{r}_2 + \dots + c_m\boldsymbol{r}_m = \boldsymbol{0}_{1\times n}
\]
若方程组 \(A\boldsymbol{x}=\boldsymbol{b}\) **有解**，则一定成立：
\[
c_1 b_1 + c_2 b_2 + \dots + c_m b_m = 0
\]

#### 三、核心原理推导

原方程组写成方程组形式：
\[
\begin{cases}
\boldsymbol{r}_1 \boldsymbol{x} = b_1 \\
\boldsymbol{r}_2 \boldsymbol{x} = b_2 \\
\quad\vdots \\
\boldsymbol{r}_m \boldsymbol{x} = b_m
\end{cases}
\]
等式两边同时乘系数 \(c_1,c_2,\dots,c_m\) 再相加：
\[
c_1\boldsymbol{r}_1\boldsymbol{x}+c_2\boldsymbol{r}_2\boldsymbol{x}+\dots+c_m\boldsymbol{r}_m\boldsymbol{x}=c_1b_1+c_2b_2+\dots+c_mb_m
\]
左边提取 \(\boldsymbol{x}\)：
\[
\big(c_1\boldsymbol{r}_1 + c_2\boldsymbol{r}_2+\dots+c_m\boldsymbol{r}*m\big)\boldsymbol{x} = \sum*{i=1}^m c_i b_i
\]
条件给出左边括号内为零行 \(\boldsymbol{0}\)，因此左边整体为 \(\boldsymbol{0}\boldsymbol{x}=0\)，于是必须：
\[
\sum_{i=1}^m c_i b_i = 0
\]

#### 四、无解判定规则（最重要用途）

- 若能找到系数 \(c_1,\dots,c_m\) 使 \(A\) 的行组合为零行，但 \(\displaystyle\sum_{i=1}^m c_i b_i \neq 0\)
  → 出现矛盾等式 \(0=\text{非零数}\) → \(A\boldsymbol{x}=\boldsymbol{b}\) **无解**
- 若 \(\displaystyle\sum_{i=1}^m c_i b_i = 0\)
  → 无矛盾，方程组存在解（无穷多解）

#### 五、例题直观演示

\[
A=\begin{bmatrix}1&2\\2&4\end{bmatrix},\quad \boldsymbol{b}=\begin{bmatrix}1\\3\end{bmatrix}
\]
取系数 \(c_1=-2,\ c_2=1\)：
行组合：\(-2[1\ \ 2]+1[2\ \ 4]=[0\ \ 0]\)（得到零行）
对 \(\boldsymbol{b}\) 做相同组合：\(-2\times 1 + 1\times 3 = 1 \neq 0\)
矛盾，因此 \(A\boldsymbol{x}=\boldsymbol{b}\) 无解。

若换 \(\boldsymbol{b}=\begin{bmatrix}1\\2\end{bmatrix}\)：
\(-2\times 1 + 1\times 2 = 0\)，无矛盾，方程组有无穷多解。

#### 六、和之前黑板知识点串联

1. \(A\boldsymbol{x}=\boldsymbol{b}\) 有解 \(\iff \boldsymbol{b}\in C(A)\)
2. 等价行判定：所有能把 \(A\) 行组合出零行的系数，作用在 \(\boldsymbol{b}\) 上结果必须为 0；不满足则无解
3. 行满秩时不存在非平凡行组合得到零行，因此对任意 \(\boldsymbol{b}\) 都不会出现矛盾，恒有解

## To find complete solution to Ax=b

### 求解 \(A\boldsymbol{x}=\boldsymbol{b}\) 的**通解（complete solution）**完整步骤 + LaTeX 写法

#### 一、核心原理：通解结构

非齐次线性方程组的完整解 = **一个特解** + **齐次方程组 \(A\boldsymbol{x}=\boldsymbol{0}\) 的全部通解（零空间解）**
\[
\boldsymbol{x}_{\text{complete}} = \boldsymbol{x}_p + \boldsymbol{x}_n
\]

- \(\boldsymbol{x}_p\)：particular solution，特解（随便找一个满足 \(A\boldsymbol{x}_p=\boldsymbol{b}\) 的向量）
- \(\boldsymbol{x}_n\)：nullspace solution，零空间通解（齐次方程全部解的线性组合）

#### 二、标准分步解题流程

##### 步骤 1：写出增广矩阵 \(\overline{A}=[A\mid \boldsymbol{b}]\)，高斯行化简为行最简形 \(R\)

LaTeX 增广矩阵示例：

```latex
\[
\overline{A}=
\left[\begin{array}{ccc|c}
2 & 4 & 1 & 5 \\
1 & 2 & 3 & 4 \\
3 & 6 & 2 & 9
\end{array}\right]
\]
```

##### 步骤 2：判断可解性

1. 若 \(r(A)\neq r(\overline{A})\)：方程组**无解**，不存在完整解；
2. 若 \(r(A)=r(\overline{A})\)：有解，继续求特解 + 零空间解。

##### 步骤 3：求特解 \(\boldsymbol{x}_p\)

1. 标记主元列（pivot columns）、自由变量列（free columns）；
2. 令所有**自由变量 = 0**，回代最简方程解出主元变量，得到一组特解 \(\boldsymbol{x}_p\)。

##### 步骤 4：求零空间通解 \(\boldsymbol{x}_n\)（解齐次方程 \(A\boldsymbol{x}=\boldsymbol{0}\)）

1. 依次令单个自由变量 = 1，其余自由变量 = 0；
2. 每组取值对应一个**基础解系向量** \(\boldsymbol{v}_1,\boldsymbol{v}_2,\dots,\boldsymbol{v}_{n-r}\)；
3. 零空间全部解：
   \[
   \boldsymbol{x}_n = c_1\boldsymbol{v}_1 + c_2\boldsymbol{v}*2 + \dots + c*{n-r}\boldsymbol{v}_{n-r},\quad c_1,c_2,\dots\in\mathbb R
   \]
   \(n\) = 未知数个数，\(r=r(A)\) 为矩阵秩。

##### 步骤 5：拼接完整通解

\[
\boldsymbol{x} = \boldsymbol{x}_p + c_1\boldsymbol{v}_1 + c_2\boldsymbol{v}*2 + \dots + c*{n-r}\boldsymbol{v}_{n-r}
\]

#### 三、例题演示（带完整 LaTeX）

方程组：
\[
\begin{cases}
x_1 + 2x_2 = 3\\
2x_1 + 4x_2 = 6
\end{cases}
\]

1. 增广矩阵化简：
   \[
   \left[\begin{array}{cc|c}
   1 & 2 & 3 \\
   2 & 4 & 6
   \end{array}\right]
   \xrightarrow{R_2-2R_1}
   \left[\begin{array}{cc|c}
   1 & 2 & 3 \\
   0 & 0 & 0
   \end{array}\right]
   \]
   秩 \(r(A)=r(\overline{A})=1\)，有解；
   主元：\(x_1\)，自由变量：\(x_2\)

2. 求特解 \(\boldsymbol{x}_p\)
   令自由变量 \(x_2=0\)，得 \(x_1=3\)
   \[
   \boldsymbol{x}_p = \begin{pmatrix}3\\0\end{pmatrix}
   \]

3. 求零空间解 \(A\boldsymbol{x}=\boldsymbol{0}\)
   方程：\(x_1+2x_2=0\)，令 \(x_2=1\Rightarrow x_1=-2\)
   基础解系 \(\boldsymbol{v}_1=\begin{pmatrix}-2\\1\end{pmatrix}\)
   零空间通解：\(\boldsymbol{x}_n = c\begin{pmatrix}-2\\1\end{pmatrix},\ c\in\mathbb R\)

4. 完整通解 complete solution

```latex
\[
\boldsymbol{x}_{\text{complete}}
=
\begin{pmatrix}3\\0\end{pmatrix}
+
c\begin{pmatrix}-2\\1\end{pmatrix},\quad c\in\mathbb R
\]
```

输出：
\[
\boldsymbol{x}_{\text{complete}}
=

\begin{pmatrix}3\\0\end{pmatrix} +
c\begin{pmatrix}-2\\1\end{pmatrix},\quad c\in\mathbb R
\]

#### 四、配套课堂英文句式 LaTeX 写法

```latex
\usepackage{amsmath,bm}
\begin{document}
\[
\text{To find the complete solution to } A\bm{x}=\bm{b},\quad
\bm{x}_{\text{complete}} = \bm{x}_p + \bm{x}_n
\]
\[
\bm{x}_p:\text{ particular solution},\quad
\bm{x}_n:\text{ all solutions of homogeneous equation } A\bm{x}=\bm{0}
\]
\end{document}
```

##### 翻译

To find complete solution to \(A\boldsymbol{x}=\boldsymbol{b}\)
→ 求解非齐次方程组 \(A\boldsymbol{x}=\boldsymbol{b}\) 的全部通解。

## 怎么用 sympy 求增广矩阵的解？

### SymPy 求解增广矩阵线性方程组完整教程

#### 一、核心原理

线性方程组 $Ax=b$，**增广矩阵** $[A|b]$，两种求解思路：

1. 直接构造增广矩阵，用 `rref()` 行最简形观察解
2. 分开系数矩阵 $A$、常数向量 $b$，用 `solve_linear_system` / `Matrix.solve()`

#### 二、完整代码示例

##### 1. 导入库、定义符号

```python
import sympy as sp

# 定义变量
x, y, z = sp.symbols("x y z")
```

##### 方法 1：构造增广矩阵，求行最简形 RREF（最直观看增广矩阵变换）

方程组：

$$
\begin{cases}
x + 2y - z = 1 \\
2x + y + z = 5 \\
x - y + 2z = 7
\end{cases}
$$

增广矩阵：

$$
\left[\begin{array}{ccc|c}
1 & 2 & -1 & 1 \\
2 & 1 & 1 & 5 \\
1 & -1 & 2 & 7
\end{array}\right]
$$

```python
# 构造增广矩阵 [A|b]
aug = sp.Matrix([[1, 2, -1, 1], [2, 1, 1, 5], [1, -1, 2, 7]])

# 求行最简形，返回 (rref 矩阵，主元列下标)
rref_mat, pivot = aug.rref()
print("增广矩阵行最简形：")
sp.pprint(rref_mat)
print("主元列：", pivot)
```

输出行最简后直接读出唯一解。

##### 方法 2：专用函数 solve_linear_system（直接传入增广矩阵求解）

```python
# 直接用增广矩阵解方程组
sol = sp.solve_linear_system(aug, x, y, z)
print("方程组解：", sol)
```

##### 方法 3：分开系数矩阵 A、常数项 b，标准矩阵求解

```python
A = sp.Matrix([[1, 2, -1], [2, 1, 1], [1, -1, 2]])
b = sp.Matrix([1, 5, 7])

# Ax = b 求解
x_sol = A.solve(b)
sp.pprint(x_sol)
```

#### 三、三种解的情况演示（唯一解/无解/无穷多解）

##### 情形 1：唯一解（上面例子）

##### 情形 2：无解（增广矩阵 rref 出现 `[0 0 | k], k≠0`）

方程组：

$$
\begin{cases}
x+y=2 \\
2x+2y=5
\end{cases}
$$

```python
aug2 = sp.Matrix([[1, 1, 2], [2, 2, 5]])
r2, p2 = aug2.rref()
sp.pprint(r2)
# 调用求解会报错，代表无解
# sp.solve_linear_system(aug2, x, y)
```

rref 会得到第二行 `[0 0 | 1]`，矛盾，无解。

##### 情形 3：无穷多解（自由变量，带参数通解）

方程组：

$$
\begin{cases}
x + y + z = 3 \\
2x + 2y + 2z = 6
\end{cases}
$$

```python
aug3 = sp.Matrix([[1, 1, 1, 3], [2, 2, 2, 6]])
r3, p3 = aug3.rref()
sp.pprint(r3)
sol3 = sp.solve_linear_system(aug3, x, y, z)
print(sol3)
```

会用自由变量表示通解。

#### 四、封装通用函数：输入增广矩阵自动输出结果

```python
def solve_augmented(aug_mat, vars_list):
    """
    aug_mat: sympy 增广矩阵
    vars_list: 变量列表 [x,y,z...]
    return: 解字典 + rref 矩阵
    """
    rref_mat, pivot = aug_mat.rref()
    try:
        sol = sp.solve_linear_system(aug_mat, *vars_list)
        return sol, rref_mat
    except Exception as e:
        return "无解", rref_mat


# 调用示例
aug = sp.Matrix([[1, 2, -1, 1], [2, 1, 1, 5], [1, -1, 2, 7]])
res, r = solve_augmented(aug, [x, y, z])
print("解：", res)
sp.pprint(r)
```

#### 关键知识点总结

1. `Matrix.rref()`：对**增广矩阵**做初等行变换，直观判断解的类型；
2. `solve_linear_system(增广矩阵, 变量)`：专门针对增广矩阵求解线性方程组；
3. `A.solve(b)`：分离系数矩阵与右端向量，等价增广矩阵解法；
4. 判断规则：
   - 系数矩阵秩 = 增广矩阵秩 = 变量数 → 唯一解
   - 系数矩阵秩 < 增广矩阵秩 → 无解
   - 系数矩阵秩 = 增广矩阵秩 < 变量数 → 无穷多通解

## rref 怎么看哪个是解？

### SymPy rref 行最简形怎么读解（一步一步看懂）

#### 前置概念

增广矩阵格式：$[A \mid b]$
每一行：系数 1 系数 2 … 系数 n | 常数项
rref 是**行最简阶梯形**，满足：

1. 主元（每行第一个非 0 数）= 1
2. 主元所在列其余全是 0
3. 主元从上到下往右偏移

变量：$x_1,x_2,...,x_n$，对应前 $n$ 列；最后一列是常数项。

#### 一、唯一解示例（3 变量）

方程组

$$
\begin{cases}
x + 2y - z = 1 \\
2x + y + z = 5 \\
x - y + 2z = 7
\end{cases}
$$

增广矩阵

$$
\left[\begin{array}{ccc|c}
1 & 2 & -1 & 1 \\
2 & 1 & 1 & 5 \\
1 & -1 & 2 & 7
\end{array}\right]
$$

rref 结果：

$$
\left[\begin{array}{ccc|c}
1 & 0 & 0 & 3 \\
0 & 1 & 0 & -1 \\
0 & 0 & 1 & 0
\end{array}\right]
$$

##### 读法规则

每一行对应一个等式：

1. 第 1 行：$1\cdot x + 0\cdot y + 0\cdot z = 3 \implies \boldsymbol{x=3}$
2. 第 2 行：$0\cdot x + 1\cdot y + 0\cdot z = -1 \implies \boldsymbol{y=-1}$
3. 第 3 行：$0\cdot x + 0\cdot y + 1\cdot z = 0 \implies \boldsymbol{z=0}$

主元列刚好全部覆盖变量列，无自由变量，唯一解。

#### 二、无解怎么看

例子：

$$
\begin{cases}
x+y=2 \\
2x+2y=5
\end{cases}
$$

增广矩阵 rref：

$$
\left[\begin{array}{cc|c}
1 & 1 & 2 \\
0 & 0 & 1
\end{array}\right]
$$

最后一行：$0x+0y=1$，矛盾等式，**无解**。
判定标准：存在一行 `[0 0 … 0 | k]`，$k\neq0$。

#### 三、无穷多解（有自由变量，重点）

方程组

$$
\begin{cases}
x + y + z = 3 \\
2x + 2y + 2z = 6
\end{cases}
$$

增广矩阵 rref：

$$
\left[\begin{array}{ccc|c}
1 & 1 & 1 & 3 \\
0 & 0 & 0 & 0
\end{array}\right]
$$

##### 步骤 1：区分主元变量 & 自由变量

`aug.rref()` 返回第二个返回值是 `pivot`，即主元列下标（从 0 开始）
本例 `pivot = (0,)`，只有第 0 列是主元列（对应$x$）

- 主元变量：$x$（主元列对应的变量）
- 自由变量：不在 pivot 里的列 → $y,z$，设为参数 $s,t$

##### 步骤 2：写出行等式

第一行：$x + y + z = 3$
把主元变量放左边，自由变量移到右边：
$$x = 3 - y - z$$
令自由变量 $y=s,\ z=t$（$s,t\in\mathbb{R}$）
通解：

$$
\begin{cases}
x = 3 - s - t \\
y = s \\
z = t
\end{cases}
$$

##### 复杂一点的无穷解例子

3 变量增广矩阵 rref：

$$
\left[\begin{array}{ccc|c}
1 & 0 & 2 & 5 \\
0 & 1 & -3 & 2 \\
0 & 0 & 0 & 0
\end{array}\right]
$$

pivot=(0,1)，主元变量 $x,y$；自由变量 $z=t$
等式：

$$
\begin{cases}
x + 2z = 5 \\
y - 3z = 2
\end{cases}
$$

移项：

$$
\begin{cases}
x = 5 - 2t \\
y = 2 + 3t \\
z = t
\end{cases}
$$

#### 四、通用读解步骤（万能流程）

1. 拆分矩阵：前 n 列=变量系数，最后一列=常数项
2. 拿到 `pivot` 主元列下标，区分主元变量/自由变量
3. 逐行写等式：
   - 主元系数=1，其余变量移到右侧
4. 分类判断：
   - 有 $[0\cdots0|k],k\neq0$ → 无解
   - 主元列数量 = 变量总数 → 唯一解，每一行直接读出变量值
   - 主元列数量 < 变量总数 → 无穷多解，自由变量设参数，写出通解

#### 配套代码演示

```python
import sympy as sp

x, y, z = sp.symbols("x y z")

# 无穷多解增广矩阵
aug = sp.Matrix([[1, 1, 1, 3], [2, 2, 2, 6]])
r, pivot = aug.rref()
print("rref:")
sp.pprint(r)
print("主元列下标 (从 0 开始)：", pivot)
# 直接求解得到参数化解
print(sp.solve_linear_system(aug, x, y, z))
```

输出会自动用符号表示自由变量，和手动读 rref 结果完全对应。

## $x_{nullspace}$ 是什么

$x_{\text{nullspace}}$ 通常表示向量 $x$ 中**属于矩阵 $A$ 的零空间的那部分**。它不是统一规定的标准符号，而是一种直观命名。

如果

$$
x_{\text{nullspace}} \in N(A),
$$

那么按照零空间的定义：

$$
Ax_{\text{nullspace}}=0.
$$

因此，方程 $Ax=b$ 的任意解常写成：

$$
x=x_{\text{particular}}+x_{\text{nullspace}},
$$

其中：

- $x_{\text{particular}}$ 是某个特解，满足 $Ax_{\text{particular}}=b$；
- $x_{\text{nullspace}}$ 是零空间中的任意向量，不会改变 $Ax$ 的结果。

因为：

$$
Ax=A(x_{\text{particular}}+x_{\text{nullspace}})
=Ax_{\text{particular}}+Ax_{\text{nullspace}}
=b+0=b.
$$

直观地说，$x_{\text{nullspace}}$ 是**输入 $x$ 中会被 $A$ 完全压扁、因而无法从输出 $b$ 中观察到的部分**。这也正是为什么只要 $A$ 有非零零空间，$Ax=b$ 的解就可能不唯一。

可以。$x_{\text{nullspace}}$ 背后其实连接了线性方程组、自由变量、解的不唯一性、行空间以及最小范数解，是零空间最核心的用途之一。

### 1. 它到底表示什么？

$x_{\text{nullspace}}$ 不是线性代数中统一规定的标准符号，通常是为了表达：

> 向量 $x$ 中位于矩阵 $A$ 的零空间里的部分。

矩阵 $A$ 的零空间定义为：

$$
N(A)={x\in\mathbb R^n:Ax=0}.
$$

因此，如果

$$
x_{\text{nullspace}}\in N(A),
$$

就一定有：

$$
Ax_{\text{nullspace}}=0.
$$

也就是说，$A$ 会把这个向量压缩成零向量。

---

### 2. 为什么解中会出现 $x_{\text{nullspace}}$？

考虑非齐次方程：

$$
Ax=b.
$$

假设已经找到了一个特解 $x_p$：

$$
Ax_p=b.
$$

现在取零空间里的任意向量 $x_n$：

$$
Ax_n=0.
$$

那么：

$$
A(x_p+x_n)=Ax_p+Ax_n=b+0=b.
$$

所以 $x_p+x_n$ 仍然是方程的解。

因此，$Ax=b$ 的完整解可以写成：

$$
x=x_p+x_n,\qquad x_n\in N(A).
$$

也经常写成：

$$
x=x_{\text{particular}}+x_{\text{nullspace}}.
$$

这里：

- $x_{\text{particular}}$：负责产生 $b$；
- $x_{\text{nullspace}}$：不会改变输出 $b$。

一句话概括：

> 特解决定你到达哪个 $b$，零空间决定你可以在解集中怎样自由移动。

---

### 3. 一个最简单的例子

考虑：

$$
A=\begin{bmatrix}1&1\end{bmatrix},\qquad b=3.
$$

方程 $Ax=b$ 就是：

$$
x_1+x_2=3.
$$

其中一个特解是：

$$
x_p=\begin{bmatrix}3\\0\end{bmatrix}.
$$

因为：

$$
Ax_p=\begin{bmatrix}1&1\end{bmatrix}\begin{bmatrix}3\\0\end{bmatrix}=3.
$$

再求零空间：

$$
Ax_n=0.
$$

也就是：

$$
x_1+x_2=0.
$$

因此：

$$
x_n=t\begin{bmatrix}1\\-1\end{bmatrix}.
$$

完整解为：

$$
x=\begin{bmatrix}3\\0\end{bmatrix}+t\begin{bmatrix}1\\-1\end{bmatrix}.
$$

展开就是：

$$
x=\begin{bmatrix}3+t\\-t\end{bmatrix}.
$$

无论 $t$ 是多少：

$$
(3+t)+(-t)=3.
$$

这里的

$$
x_{\text{nullspace}}=t\begin{bmatrix}1\\-1\end{bmatrix}
$$

表示沿着方向 $(1,-1)$ 移动。这个移动会让 $x_1$ 增加，同时让 $x_2$ 等量减少，因此二者的和始终不变。

---

### 4. 几何上是什么意思？

方程

$$
x_1+x_2=3
$$

在二维平面中是一条直线。

零空间

$$
x_1+x_2=0
$$

是另一条经过原点的直线。

这两条直线平行，因为所有解都可以由一个特解加上零空间中的向量得到：

$$
{\text{$Ax=b$ 的全部解}}=x_p+N(A).
$$

也就是说：

> 非齐次方程的解集，是零空间整体平移后的结果。

如果零空间是：

- 一条线，那么 $Ax=b$ 的解集也是一条平移后的线；
- 一个平面，那么 $Ax=b$ 的解集也是一个平移后的平面；
- 只有零向量，那么 $Ax=b$ 至多只有一个解。

---

### 5. 为什么它对应自由变量？

考虑矩阵：

$$
A=\begin{bmatrix}
1&2&3\\
2&4&6
\end{bmatrix},\qquad
b=\begin{bmatrix}6\\12\end{bmatrix}.
$$

方程组实际上只有一个独立方程：

$$
x_1+2x_2+3x_3=6.
$$

令自由变量：

$$
x_2=s,\qquad x_3=t.
$$

那么：

$$
x_1=6-2s-3t.
$$

所以：

$$
x=
\begin{bmatrix}
6-2s-3t\\
s\\
t
\end{bmatrix}.
$$

把它拆开：

$$
x=
\begin{bmatrix}
6\\
0\\
0
\end{bmatrix}
+s
\begin{bmatrix}
-2\\
1\\
0
\end{bmatrix}
+t
\begin{bmatrix}
-3\\
0\\
1
\end{bmatrix}.
$$

第一部分是一个特解：

$$
x_p=\begin{bmatrix}6\\0\\0\end{bmatrix}.
$$

后两部分属于零空间：

$$
x_n = s\begin{bmatrix}-2\\1\\0\end{bmatrix} + t\begin{bmatrix}-3\\0\\1\end{bmatrix}.
$$

因此：

$$
x=x_p+x_n.
$$

而零空间为：

$$
N(A)=\operatorname{span}\left\{
\begin{bmatrix}-2\\1\\0\end{bmatrix},
\begin{bmatrix}-3\\0\\1\end{bmatrix}
\right\}.
$$

这里有两个自由变量，所以零空间是二维的。

---

### 6. 它和秩有什么关系？

如果 $A$ 是一个 $m\times n$ 矩阵，那么输入向量 $x$ 属于 $\mathbb R^n$。

秩—零度定理告诉我们：

$$
\operatorname{rank}(A)+\operatorname{nullity}(A)=n.
$$

其中：

- $\operatorname{rank}(A)$：主元变量的个数；
- $\operatorname{nullity}(A)$：自由变量的个数，也是零空间的维数。

因此：

$$
\dim N(A)=n-\operatorname{rank}(A).
$$

例如上面的矩阵有三列，但秩为 $1$：

$$
\dim N(A)=3-1=2.
$$

所以它有两个独立的零空间方向，也就有两个自由参数 $s,t$。

---

### 7. 从“信息丢失”的角度理解

假设两个不同的输入满足：

$$
Ax_1=b,\qquad Ax_2=b.
$$

两式相减：

$$
A(x_1-x_2)=0.
$$

所以：

$$
x_1-x_2\in N(A).
$$

这说明：

> 所有产生同一个输出 $b$ 的输入，彼此之间都只相差一个零空间向量。

例如：

$$
A=\begin{bmatrix}1&1\end{bmatrix}.
$$

下面这些输入都会得到同一个输出 $3$：

$$
\begin{bmatrix}3\\0\end{bmatrix},
\quad
\begin{bmatrix}2\\1\end{bmatrix},
\quad
\begin{bmatrix}1\\2\end{bmatrix},
\quad
\begin{bmatrix}0\\3\end{bmatrix}.
$$

矩阵 $A$ 只知道两个分量的和，不知道这个和具体是怎样分配的。

因此，$A$ 丢失了方向

$$
\begin{bmatrix}1\\-1\end{bmatrix}
$$

上的信息。这个方向正是零空间方向。

---

### 8. “一个解中的零空间部分”还有更严格的含义

对于任意 $m\times n$ 矩阵 $A$，输入空间 $\mathbb R^n$ 可以正交分解为：

$$
\mathbb R^n=C(A^\mathsf T)\oplus N(A).
$$

其中：

- $C(A^\mathsf T)$ 是 $A$ 的行空间；
- $N(A)$ 是 $A$ 的零空间；
- 两个空间互相正交。

因此，任何向量 $x\in\mathbb R^n$ 都能唯一拆成：

$$
x=x_{\text{row}}+x_{\text{nullspace}},
$$

其中：

$$
x_{\text{row}}\in C(A^\mathsf T),
\qquad
x_{\text{nullspace}}\in N(A).
$$

于是：

$$
Ax=Ax_{\text{row}}+Ax_{\text{nullspace}}
=Ax_{\text{row}}.
$$

因为：

$$
Ax_{\text{nullspace}}=0.
$$

这意味着：

> $A$ 真正“看得见”的只有 $x$ 的行空间部分；零空间部分对输出完全不可见。

这比“零空间是 $Ax=0$ 的解集”更深一层：零空间实际上表示线性变换丢失的信息。

---

### 9. 为什么最小范数解没有零空间部分？

如果 $Ax=b$ 有无穷多个解，可以写成：

$$
x=x_{\text{row}}+x_{\text{nullspace}}.
$$

由于行空间和零空间正交，根据勾股定理：

$$
\lVert x\rVert^2
=\lVert x_{\text{row}}\rVert^2
+\lVert x_{\text{nullspace}}\rVert^2.
$$

只要添加非零的 $x_{\text{nullspace}}$，向量长度就会增加。

因此最短的那个解必然满足：

$$
x_{\text{nullspace}}=0.
$$

所以最小范数解完全位于行空间中：

$$
x_{\min}\in C(A^\mathsf T).
$$

这正是伪逆解

$$
x_{\min}=A^+b
$$

的重要性质。

以 $x_1+x_2=3$ 为例，所有解是：

$$
x=\begin{bmatrix}3/2\\3/2\end{bmatrix}
+t\begin{bmatrix}1\\-1\end{bmatrix}.
$$

其中最短的解是：

$$
x_{\min}=\begin{bmatrix}3/2\\3/2\end{bmatrix}.
$$

因为它没有任何零空间分量。沿零空间向任意方向移动，都会离原点更远。

---

### 10. 两种相似但略有区别的用法

看到 $x_{\text{nullspace}}$ 时，需要根据上下文判断它是哪一种含义。

#### 用法一：通解中的任意零空间向量

$$
x=x_p+x_{\text{nullspace}}.
$$

这里 $x_p$ 只是随便找的一个特解，$x_{\text{nullspace}}$ 表示齐次方程的通解。

这种拆分不一定是正交分解，选择不同的特解，$x_{\text{nullspace}}$ 的表达也会改变。

#### 用法二：特定向量的正交投影

$$
x=x_{\text{row}}+x_{\text{nullspace}}.
$$

这里的 $x_{\text{nullspace}}$ 是 $x$ 在 $N(A)$ 上的正交投影：

$$
x_{\text{nullspace}}=\operatorname{proj}_{N(A)}(x).
$$

这种拆分是唯一的。

所以两种写法都正确，但侧重点不同：

- $x_p+x_n$：强调线性方程组的全部解；
- $x_{\text{row}}+x_n$：强调矩阵保留和丢失了哪些信息。

---

### 11. 最核心的一张关系表

| 概念                                    | 含义                                  |
| --------------------------------------- | ------------------------------------- |
| $N(A)$                                  | 所有被 $A$ 映射为零的输入             |
| $x_{\text{nullspace}}$                  | 位于 $N(A)$ 中的向量或向量分量        |
| $Ax_{\text{nullspace}}=0$               | 零空间部分不影响输出                  |
| $x=x_p+x_{\text{nullspace}}$            | 非齐次方程的完整解                    |
| 自由变量                                | 决定零空间中的移动幅度                |
| $\dim N(A)$                             | 自由变量的数量                        |
| $x_1-x_2\in N(A)$                       | 同一个 $b$ 的任意两个解之差属于零空间 |
| $x=x_{\text{row}}+x_{\text{nullspace}}$ | 输入由可见部分和不可见部分组成        |
| 最小范数解                              | 零空间部分等于零的解                  |

最本质的理解就是：

> $x_{\text{nullspace}}$ 是 $x$ 中不能通过 $Ax$ 被观察到的部分。它可以改变输入，却不改变输出。

## 我是不是得理解上一节 Ax=0 的一些概念？

是的。要真正理解 $x_{\text{nullspace}}$，确实需要先掌握上一节 $Ax=0$，因为：

> $x_{\text{nullspace}}$ 本身就是 $Ax=0$ 的解。

不过不需要把上一节的所有计算技巧练得特别熟，抓住几个核心概念就足够继续。

### 需要掌握的四个概念

#### 1. 零空间是什么

$$
N(A)={x:Ax=0}.
$$

它是所有被矩阵 $A$ 压成零向量的输入组成的集合。

你至少要能理解：

- 为什么零向量一定属于 $N(A)$；
- 为什么有些矩阵还有非零解；
- 非零零空间意味着 $A$ 丢失了某些输入信息。

#### 2. 特殊解如何产生零空间

对 $A$ 进行行化简，通过自由变量求出 $Ax=0$ 的特殊解。

例如：

$$
x_1+2x_2+3x_3=0.
$$

令：

$$
x_2=s,\qquad x_3=t,
$$

便得到：

$$
x=s\begin{bmatrix}-2\1\0\end{bmatrix}
+t\begin{bmatrix}-3\0\1\end{bmatrix}.
$$

两个向量就是零空间的一组基。

这里最重要的是理解：

> 每个自由变量通常对应一个独立的零空间方向。

#### 3. 主元变量和自由变量

你需要知道：

- 主元列对应主元变量；
- 非主元列对应自由变量；
- 自由变量的数量就是零空间的维数。

也就是：

$$
\dim N(A)=n-\operatorname{rank}(A).
$$

这会直接决定 $Ax=b$ 的解是否可能不唯一。

#### 4. 为什么零空间中的向量不影响 $Ax=b$

只要 $x_n\in N(A)$：

$$
Ax_n=0.
$$

因此：

$$
A(x_p+x_n)=Ax_p+Ax_n=b.
$$

这一步就是从 $Ax=0$ 过渡到 $Ax=b$ 的桥梁。

> $x_{\text{nullspace}}$ 就是一个满足 $Ax=0$ 的向量。

$$

$$

$$
\begin{bmatrix}1\\-1\end{bmatrix}
$$

它的意思特别朴素：

> 第一个数加 $1$，第二个数减 $1$，总和不变。

$$
\begin{bmatrix}2\\1\end{bmatrix},

$$

$$
\begin{bmatrix}2\\1\end{bmatrix}
+
\begin{bmatrix}1\\-1\end{bmatrix}

\begin{bmatrix}3\\0\end{bmatrix},
$$

结果仍然满足：

$$
3+0=3.
$$

> $x_{\text{nullspace}}$ 是一种“虽然改变了 $x$，但没有改变 $Ax$”的移动。
