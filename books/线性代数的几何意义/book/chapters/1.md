# 什么是线性代数

## 代数的意义

```python
import sympy as sp

# 空格自动分割字符串，返回符号元组，可直接解包
a, b, A = sp.symbols("a b A")

expr = a**2 + (b * sp.cos(A)) ** 2

sp.pprint(expr)
```

### SymPy 完整实现：向量加法 + 向量完全平方（点积）几何对应

#### 核心知识点铺垫

1. 向量平方 $\boldsymbol{a}^2 = \boldsymbol{a}\cdot\boldsymbol{a} = |\boldsymbol{a}|^2$（自身点积=模长平方）
2. $(\boldsymbol{a}+\boldsymbol{b})^2 = (\boldsymbol{a}+\boldsymbol{b})\cdot(\boldsymbol{a}+\boldsymbol{b}) = \boldsymbol{a}\cdot\boldsymbol{a} + 2\boldsymbol{a}\cdot\boldsymbol{b} + \boldsymbol{b}\cdot\boldsymbol{b}$
3. 几何形式：$|\boldsymbol{c}|^2 = |\boldsymbol{a}|^2 + |\boldsymbol{b}|^2 + 2|\boldsymbol{a}||\boldsymbol{b}|\cos\theta$（$\theta$ 是两向量夹角）

#### 方案 1：符号向量（2 维，对应图中$x_1,x_2$平面）

```python
import sympy as sp

# 1. 定义符号：向量分量、夹角、模长
a1, a2, b1, b2 = sp.symbols("a1 a2 b1 b2", real=True)
theta = sp.Symbol("theta", real=True)
abs_a, abs_b = sp.symbols("|a| |b|", positive=True)

# 2. 构造向量 a, b
a = sp.Matrix([a1, a2])
b = sp.Matrix([b1, b2])
c = a + b  # c = a + b 对应图 OB = OA + AB

print("向量 a =", a)
print("向量 b =", b)
print("和向量 c = a + b =", c)

# 3. 向量完全平方：代数展开 (a+b)·(a+b)
dot_cc = c.dot(c)  # (a+b)² = |c|²
dot_aa = a.dot(a)  # a² = |a|²
dot_bb = b.dot(b)  # b² = |b|²
dot_ab = a.dot(b)  # a·b = |a||b|cosθ

expand_form = sp.expand(dot_cc)
print("\n(a+b)·(a+b) 展开：", expand_form)
print("a·a + 2a·b + b·b =", dot_aa + 2 * dot_ab + dot_bb)
print("等式验证：", sp.simplify(dot_cc - (dot_aa + 2 * dot_ab + dot_bb)) == 0)

# 4. 几何形式替换 a·b = |a||b|cosθ
geo_form = abs_a**2 + abs_b**2 + 2 * abs_a * abs_b * sp.cos(theta)
print("\n几何对应公式 |c|² =", geo_form)
```

#### 方案 2：纯几何符号（不拆分量，只用模长 + 夹角）

仅还原图中余弦形式，不使用坐标分量：

```python
import sympy as sp

abs_a, abs_b, theta = sp.symbols("|OA| |AB| A'", positive=True, real=True)
abs_c = sp.Symbol("|OB|", positive=True)

# 向量完全平方几何式
eq_geo = sp.Eq(abs_c**2, abs_a**2 + abs_b**2 + 2 * abs_a * abs_b * sp.cos(theta))
print("向量完全平方几何等式：")
sp.pprint(eq_geo)
```

输出：
$$|\overrightarrow{OB}|^{2} = |\overrightarrow{OA}|^{2} + |\overrightarrow{AB}|^{2} + 2 |\overrightarrow{OA}| |\overrightarrow{AB}| \cos(A')$$

#### 方案 3：数值向量实例（代入坐标验证）

贴合图中网格直观验算：

```python
import sympy as sp

# 设 OA=a=(2,0), AB=b=(2,3)
a = sp.Matrix([2, 0])
b = sp.Matrix([2, 3])
c = a + b

# 代数计算
mod_c_sq = c.dot(c)
mod_a_sq = a.dot(a)
mod_b_sq = b.dot(b)
dot_ab = a.dot(b)

print("|c|² =", mod_c_sq)
print("|a|² + 2a·b + |b|² =", mod_a_sq + 2 * dot_ab + mod_b_sq)

# 几何余弦验证
import math

abs_a = sp.sqrt(mod_a_sq)
abs_b = sp.sqrt(mod_b_sq)
cos_theta = dot_ab / (abs_a * abs_b)
geo_val = mod_a_sq + mod_b_sq + 2 * abs_a * abs_b * cos_theta
print("几何公式计算值：", sp.simplify(geo_val))
```

#### 关键对应图中概念

1. $\overrightarrow{OA} = \boldsymbol{a}$ → `a = Matrix([a1,a2])`
2. $\overrightarrow{AB} = \boldsymbol{b}$ → `b = Matrix([b1,b2])`
3. $\overrightarrow{OB} = \boldsymbol{c} = \boldsymbol{a}+\boldsymbol{b}$ → `c = a + b`
4. $(\boldsymbol{a}+\boldsymbol{b})^2 = |\boldsymbol{c}|^2 = \boldsymbol{c}\cdot\boldsymbol{c}$：向量平方等价于自身点积
5. $2\boldsymbol{a}\cdot\boldsymbol{b} = 2|\boldsymbol{a}||\boldsymbol{b}|\cos A'$：点积几何定义，$A'$ 是向量$\boldsymbol{a},\boldsymbol{b}$的夹角

#### 补充说明

- SymPy 中**向量平方不能直接写 `a**2`**，必须用 `a.dot(a)` 表示模长平方；
- `Matrix.dot()` 是标准内积运算，完美匹配线性代数向量点积；
- 若需要绘制网格向量图，可搭配 `matplotlib` 画出和图 1-1 一致的向量示意图。
