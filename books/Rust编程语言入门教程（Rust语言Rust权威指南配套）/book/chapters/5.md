# Struct

## 定义并实例化 struct

```rust
struct User {
    username: String,
    email: String,
}

let user_1 = User {
    username: String::from("qrkks"),
    email: String::from("qrkks3@gmail.com"),
};

dbg!(user_1.username);
```

```rust
fn build_user(username: String, email: String) -> User {
    User {
        username: username,
        email: email,
    }
}
let user_2 = build_user(String::from("hi"), String::from("34028312@qq.com"));
dbg!(user_2.username); // 如果不传借用，变量会被用掉；它会返回打印的值，如果不接自然就被用掉了
println!("{}", user_2.username); // 不会将变量用掉
```

## Struct 例子

```rust
{
    fn area(l: i32, w: i32) -> i32 {
        l * w
    }

    let l = 50;
    let w = 30;
    dbg!(area(l, w));
}
```

```rust
{
    fn area(rec: (i32, i32)) -> i32 {
        rec.0 * rec.1
    }

    dbg!(area((30, 50)));
}
```

```rust
:clear
```

```rust
#[derive(Debug)] // #[...] 叫 attribute，属性。它是写给编译器或宏看的附加信息。
struct Rectangle {
    width: i32,
    length: i32,
}
let rec = Rectangle {
    width: 30,
    length: 50,
}; // 实例化普通结构体使用花括号
dbg!(&rec);
```

```rust
impl Rectangle {
    fn area(&self) -> i32 {
        // self 不需要写类型标注。在 impl User 里，编译器已经知道 self 就是当前的 Rectangle 实例；你只需要声明是拥有它、借用它，还是可变借用它。
        self.width * self.length
    }

    fn general_area(width: i32, length: i32) -> i32 {
        width * length
    }
}
```
