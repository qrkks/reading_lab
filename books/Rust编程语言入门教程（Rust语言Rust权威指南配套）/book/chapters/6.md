### 定义方法

```rust
// 定义枚举 Message，四种不同变体（无参、结构体、单元组、多参数元组）
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}

// 给枚举 Message 实现关联方法
impl Message {
    fn call(&self) {
        match self {
            Message::Quit => println!("退出程序"),
            Message::Move { x, y } => println!("移动到 x:{}, y:{}", x, y),
            Message::Write(text) => println!("输出文本：{}", text),
            Message::ChangeColor(r, g, b) => println!("颜色 R{} G{} B{}", r, g, b),
        }
    }
}

fn main() {
    // 分别构造四种枚举实例
    let q = Message::Quit;
    let m = Message::Move { x: 12, y: 24 };
    let w = Message::Write(String::from("Hello"));
    let c = Message::ChangeColor(0, 255, 255);

    // 调用枚举实例的方法
    m.call();
}
```

## Option

### Rust 没有 Null

其它语言中：

- Null 是一个值，它表示“没有值”
- 一个变量可以处于两种状态：空值（null）、非空

Null 引用：Billion Dollar Mistake（十亿美元错误）

Null 的问题在于：当你尝试像使用非 Null 值那样使用 Null 值的时候，就会引起某种错误

Null 的概念还是有用的：因某种原因而变为无效或缺失的值

## Match

```rust
#[derive(Debug)]
// 定义硬币枚举，4 种单元变体
enum Coin {
    Penny,
    Nickel,
    Dime,
    Quarter,
}

// 匹配枚举，返回对应面值（单位：分）
fn value_in_cents(coin: &Coin) -> u8 {
    match coin {
        Coin::Penny => {
            println!("1 cent");
            1
        }
        Coin::Nickel => 5,
        Coin::Dime => 10,
        Coin::Quarter => 25,
    }
}

fn main() {
    value_in_cents(&Coin::Penny); // 如果不传借用
    dbg!(&Coin::Penny);
}
main();
```
