# 通用的编程概念

## 变量与可变性

```rust
fn main() {
    dbg!("Hello world");
}
main();
```

不可变变量

```rust
fn main() {
    let x = 1;
    dbg!(x);
    let x = 2; // Shadowing
    dbg!(x);
    let x = x + 1;
    dbg!(x);
}
main();
```

可变变量

```rust
fn main() {
    let mut x = 1;
    dbg!(x);
    x = 2;
    dbg!(x);
}
main();
```

常量

```rust
fn main() {
    // 常量必须声明类型
    const X: i32 = 1;
    println!("{}", X);
}
main();
```

## 数据类型：标题类型

默认：i32、f64

## 数据类型：复合类型

### 元组

```rust
let x = (1, 2);
dbg!(x);
// 索引
dbg!(x.0, x.1);
```

```rust
// 可以为元组成员分别设置类型
let tup: (i32, f64) = (1, 2.3);
dbg!(tup);
//解构
let (x, y) = tup;
dbg!(x, y);
```

### 数组

```rust
let arr = [1, 3, 4, 8];
dbg!(arr);
// 索引
dbg!(arr[0], arr[1]);
```

用处：

- 数组没有 Vector 灵活（以后再讲）。
  - Vector 和数组类似，它由标准库提供
  - Vector 的长度可以改变
  - 如果你不确定应该用数组还是 Vector，那么估计你应该用 Vector。

```rust
// 数组的类型声明
let arr: [i32; 2] = [1, 2];
dbg!(arr);
// 数组索引超出范围
dbg!(arr[2]); // 曾经允许编译通过，现版本编译也不允许通过了
// Error: this operation will panic at runtime
// dbg!(arr[2]);
//      ^^^^^^ index out of bounds: the length is 2 but the index is 2
// this operation will panic at runtime
```

```rust
// 数组的初始化
let arr = [3; 5];
dbg!(arr);
```

## 控制流：if else

```rust
let number = 3;

if number < 5 {
    println!("condition was true");
} else {
    println!("condition was false");
};
```

```rust
let number = 6;

if number % 4 == 0 {
    println!("number is divisible by 4");
} else if number % 3 == 0 {
    println!("number is divisible by 3");
} else if number % 2 == 0 {
    println!("number is divisible by 2");
} else {
    println!("number is not divisible by 4, 3 or 2");
}
```

```rust
// if 本身是表达式，天然可做 n 元运算符（多分支条件表达式）
// 别的语言可以显式的 return，但是外包函数才能实现
if true {
    dbg!("Hi");
    5
} else {
    6
}
```

```rust
let a = dbg!("Hi"); // dbg! 不只打印，还会返回打印的内容
dbg!(a);
```
