# 向量与线性方程组

1. 线性方程组 $Ax = b$ 的列图: 从列向量的视角解读线性方程组, 即矩阵 $A$ 的 $n$ 个列向量的线性组合生成向量 $b$.
2. $Ax = x_1\boldsymbol{a}_1 + x_2\boldsymbol{a}_2 + \dots + x_n\boldsymbol{a}_n = \boldsymbol{b}$ 是向量方程: $\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_n$ 是矩阵 $A$ 的列向量.
3. 当 $\boldsymbol{b} = \boldsymbol{0}$ 时, 矩阵 $A$ 的列向量组的线性组合 $Ax$ 是零向量: 一种可能的情况是 $\boldsymbol{x} = (0,0,\dots,0)$.
4. 线性方程组 $Ax = b$ 的行图: 从行向量的视角解读线性方程组, 即由 $m$ 行方程确定的 $m$ 个平面相交于点 $\boldsymbol{x}$.
5. 上述每个平面的方程都可由点积表示:
   (第 1 行)$\cdot \boldsymbol{x} = b_1$, (第 2 行)$\cdot \boldsymbol{x} = b_2$, $\dots$, (第 $m$ 行)$\cdot \boldsymbol{x} = b_m$.
6. 当 $\boldsymbol{b} = \boldsymbol{0}$ 时, 所有平面 (第 $i$ 行)$\cdot \boldsymbol{x} = 0$ 都经过原点 $\boldsymbol{x} = (0,0,\dots,0)$.

$$Ax=b$$

```python
import sympy as sp

A = sp.Matrix([[1, -2], [3, 2]])
b = sp.Matrix([1, 11])
x = A.solve(b)
# 数值计算库（SciPy、MATLAB）：全部用 solve(A,b)，贴合工程师、统计人解方程习惯；
# SymPy 符号矩阵：选用 A.solve(b)，偏向抽象代数、变换映射的 OO 设计；
# 两者只是设计取舍，不存在 “谁数学上更标准”。
print(x)
```
