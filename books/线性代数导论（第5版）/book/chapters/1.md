## 长度与点积

### 向量点积代码实现（分原生 Python、NumPy，贴合你线性代数/协方差场景）

#### 一、数学定义

两个同维度向量 $\boldsymbol a=[a_1,a_2,...,a_n],\boldsymbol b=[b_1,b_2,...,b_n]$
点积（内积）：

$$
\boldsymbol a\cdot\boldsymbol b = a_1b_1 + a_2b_2 + ... + a_nb_n
$$

#### 1. 纯 Python 手写（不用任何库，理解底层）

```python
def dot_product(a, b):
    ### 先校验维度一致
    if len(a) != len(b):
        raise ValueError("向量维度必须相同才能计算点积")
    total = 0
    for ai, bi in zip(a, b):
        total += ai * bi
    return total


### 测试二维向量
u = [1, 2]
v = [3, 4]
print(dot_product(u, v))  ### 1*3 + 2*4 = 11
```

#### 2. NumPy（你 Manim、统计建模最常用，高效）

##### 方式 1：`np.dot()` 标准点积函数

```python
import numpy as np

u = np.array([1, 2])
v = np.array([3, 4])
res = np.dot(u, v)
print(res)  ### 输出 11
```

##### 方式 2：`@` 运算符（推荐，更简洁）

```python
res = u @ v
print(res)
```

##### 方式 3：逐元素相乘后求和 `(a*b).sum()`

对应数学公式分步拆解，方便理解协方差计算逻辑：

```python
import numpy as np

res = (u * v).sum()
### 哈达玛积（Hadamard product）
print("u*v =", u * v)
print("np.multiply(u, v) =", np.multiply(u, v))
### 点积
print("(u*v).sum() =", res)
print("np.dot(u, v) =", np.dot(u, v))
```

#### 3. 拓展：矩阵×向量（本质多层点积）

矩阵每一行和向量做一次点积：

$$
\begin{bmatrix}1&2\\3&4\end{bmatrix}
\begin{bmatrix}5\\6\end{bmatrix}
=
\begin{bmatrix}1*5+2*6\\3*5+4*6\end{bmatrix}
$$

```python
M = np.array([[1, 2], [3, 4]])
x = np.array([5, 6])
print(M @ x)  ### [17 39]
```

#### 4. 和你之前知识联动（关键）

1. **向量模长 `np.linalg.norm(x)`**
   模长本质是向量自身与自身的点积开根号：

$$
\|\boldsymbol x\|=\sqrt{\boldsymbol x\cdot \boldsymbol x}
$$

```python
x = np.array([3, 4])
norm = np.sqrt(x @ x)
print(norm)  ### 5，等价 np.linalg.norm(x)
```

你截图 Manim 代码里 `norm = np.linalg.norm([x, y])` 底层就是这套点积逻辑。

2. **协方差矩阵底层全是点积**
   协方差 $Cov(X,Y) = \frac{1}{n-1}(X-\bar X) \cdot (Y-\bar Y)$，核心运算就是中心化向量的点积。

#### 补充区分易混点

- `np.dot(a,b)`：一维向量=点积；二维矩阵=矩阵乘法（行×列点积）
- `a * b`：仅逐元素相乘，**不是点积**，必须再加`.sum()`才是点积。

### 范数：Numpy 自带 `np.linalg.norm()` 直接算各类范数

#### 核心规则

函数：`np.linalg.norm(向量, ord=阶数)`

- `ord=None / ord=2`：默认 L2 范数（欧氏长度，对应公式 $\|v\|=\sqrt{v\cdot v}$）
- `ord=1`：L1 曼哈顿范数
- `ord=np.inf`：无穷（切比雪夫）范数

#### 完整代码示例

```python
import numpy as np

### 定义向量 v = [3, -4]
v = np.array([3, -4])

### 1. L2 范数（默认，不用写 ord=2）
l2 = np.linalg.norm(v)  ### linalg = linear algebra 的缩写
print("L2 范数：", l2)  ### 输出 5.0，等价 np.sqrt(v @ v)

### 2. L1 范数
l1 = np.linalg.norm(
    v, ord=1
)  ### order 在范数语境里翻译为阶、次数，正好对应你学的 Lp 里的下标 p
print("L1 范数：", l1)  ### 输出 7.0，|3|+|-4|

### 3. 无穷范数
linf = np.linalg.norm(v, ord=np.inf)
print("无穷范数：", linf)  ### 输出 4.0，max(|3|，|-4|)

### 4. 两点之间欧氏距离（差值向量的 L2 范数）
x = np.array([1, 2])
y = np.array([4, 6])
dist = np.linalg.norm(x - y)
print("两点欧氏距离：", dist)  ### 输出 5.0
print(np.linalg.norm(y - x))
### 范数自带绝对值效果；两点 A 到 B 的直线距离 = B 到 A 的直线距离，距离不存在方向，所以交换 x、y 差值不影响最终数值。
### 不光 L2，L1、无穷范数也全部满足这个对称性，因为都先取绝对值再计算。
```

#### 和手动计算对照（帮你对应之前的公式）

```python
### 手动实现 L2 范数：sqrt(v · v)
v_dot_v = v @ v  ### 自点积 3*3 + (-4)*(-4) =25
l2_manual = np.sqrt(v_dot_v)
print("手动 L2：", l2_manual)  ### 同样 5.0，和 np.linalg.norm(v) 完全相等
```

#### 补充 PyTorch（深度学习常用）

```python
import torch

v = torch.tensor([3, -4])
l2 = torch.norm(v)
l1 = torch.norm(v, p=1)
```

### MATLAB

```python
import numpy as np

# ====================== 1. 复刻 MATLAB：v = [2 3 4]'; w = [1 1 1]'; u = 2*v + 3*w ======================
# MATLAB 单引号'代表转置生成列向量
v = np.array([2, 3, 4]).reshape(-1, 1)  # 列向量 [2;3;4]
# reshape() 是 numpy 数组方法，只改变数组的行列形状，不修改数组里的数字、不重新排序。
# 语法：数组.reshape(行数，列数)
# 元素总数必须匹配：原数组总元素 = 新形状行数 × 列数
# -1 是自动填充占位符：让 numpy 自动计算这一维的长度
w = np.array([1, 1, 1]).reshape(-1, 1)  # 列向量 [1;1;1]
u = 2 * v + 3 * w
print("=== 向量线性组合 u = 2v + 3w ===")
print("v:\n", v)
print("w:\n", w)
print("u:\n", u)

# ====================== 2. 向量点积 v·w，对应 MATLAB v' * w ======================
# 测试向量 v=(1,2) w=(3,4)
vec_v = np.array([1, 2])
vec_w = np.array([3, 4])
# 方式 1：和 MATLAB v'*w 完全对应（转置行向量 × 列向量）
dot_result = vec_v.T @ vec_w
# .T 是 NumPy 数组的转置（transpose）属性，作用是交换矩阵 / 向量的行和列，完全等价 MATLAB 里的单引号 '。

# 方式 2：专用点积函数（日常更推荐）
dot_result2 = np.dot(vec_v, vec_w)
print("\n=== 向量点积 v·w ===")
print(f"点积结果（矩阵乘写法）: {dot_result}")
print(f"点积结果（np.dot 写法）: {dot_result2}")

# ====================== 3. 余弦相似度 + 向量夹角（原图核心公式） ======================
# 公式：cosθ = v'w / (||v|| * ||w||) ；角度 = arccos(cosθ)
dot_prod = vec_v.T @ vec_w
norm_v = np.linalg.norm(vec_v)  # norm(v) 向量长度/L2 范数
norm_w = np.linalg.norm(vec_w)

cosine_val = dot_prod / (norm_v * norm_w)
angle_radian = np.arccos(cosine_val)  # acos 反余弦，输出弧度
print("\n=== 余弦相似度 & 向量夹角 ===")
print(f"向量 v 长度 norm(v): {norm_v:.4f}")
print(f"向量 w 长度 norm(w): {norm_w:.4f}")
print(f"余弦值 cosθ: {cosine_val:.4f}")
print(f"向量夹角（弧度）: {angle_radian:.4f}")


# ====================== 4. 封装自定义余弦函数（对应 MATLAB .m 函数文件） ======================
def cosine(v, w):
    """计算两个向量的余弦相似度"""
    dot_p = np.dot(v, w)
    len_v = np.linalg.norm(v)
    len_w = np.linalg.norm(w)
    return dot_p / (len_v * len_w)


# 调用自定义函数
cos_func_result = cosine(vec_v, vec_w)
print("\n=== 自定义函数 cosine(v,w) 结果 ===")
print(f"余弦相似度函数输出：{cos_func_result:.4f}")
```
