# 线性方程组的求解

## 向量与线性方程组

1. 线性方程组 $Ax = b$ 的列图：从列向量的视角解读线性方程组，即矩阵 $A$ 的 $n$ 个列向量的线性组合生成向量 $b$.
2. $Ax = x_1\boldsymbol{a}_1 + x_2\boldsymbol{a}_2 + \dots + x_n\boldsymbol{a}_n = \boldsymbol{b}$ 是向量方程：$\boldsymbol{a}_1,\boldsymbol{a}_2,\dots,\boldsymbol{a}_n$ 是矩阵 $A$ 的列向量。
3. 当 $\boldsymbol{b} = \boldsymbol{0}$ 时，矩阵 $A$ 的列向量组的线性组合 $Ax$ 是零向量：一种可能的情况是 $\boldsymbol{x} = (0,0,\dots,0)$.
4. 线性方程组 $Ax = b$ 的行图：从行向量的视角解读线性方程组，即由 $m$ 行方程确定的 $m$ 个平面相交于点 $\boldsymbol{x}$.
5. 上述每个平面的方程都可由点积表示：
   (第 1 行)$\cdot \boldsymbol{x} = b_1$, (第 2 行)$\cdot \boldsymbol{x} = b_2$, $\dots$, (第 $m$ 行)$\cdot \boldsymbol{x} = b_m$.
6. 当 $\boldsymbol{b} = \boldsymbol{0}$ 时，所有平面 (第 $i$ 行)$\cdot \boldsymbol{x} = 0$ 都经过原点 $\boldsymbol{x} = (0,0,\dots,0)$.

$$Ax=b$$

```python
import sympy as sp

A = sp.Matrix([[1, -2], [3, 2]])
b = sp.Matrix([1, 11])
x = A.solve(b)
# 数值计算库（SciPy、MATLAB）：全部用 solve(A,b)，贴合工程师、统计人解方程习惯；
# SymPy 符号矩阵：选用 A.solve(b)，偏向抽象代数、变换映射的 OO 设计；
# 两者只是设计取舍，不存在“谁数学上更标准”。
print(x)
# print(sp.solve(A, b)) # 输出 []
```

## 逆矩阵

### Sympy 求逆矩阵完整方法

#### 1. 基础：`A**-1` 或 `A.inv()`

##### 示例代码

```python
import sympy as sp

# 构造方阵
A = sp.Matrix([[2, 1], [1, 1]])

# 两种求逆写法等价
A_inv1 = A**-1
A_inv2 = A.inv()

print(A_inv1)
print(A_inv2)
```

输出：
$$\begin{bmatrix} 1 & -1 \\ -1 & 2 \end{bmatrix}$$

##### 限制

仅对**可逆方阵**可用；奇异矩阵会直接报错。

#### 2. 高斯 - 若尔当消元法手动求逆（拼接增广阵 `[A|I]` + rref）

原理：把 `[A, I]` 做行最简，左边变单位阵，右边就是 $A^{-1}$

```python
import sympy as sp

A = sp.Matrix([[2, 1], [1, 1]])
n = A.shape[0]
# 拼接单位矩阵
aug = A.row_join(sp.eye(n))
# 高斯若尔当消元 rref
aug_rref = aug.rref()[0]
# 提取右侧逆矩阵
A_inv = aug_rref[:, n:]

print("A：")
sp.pprint(aug)
sp.pprint(aug_rref)
print("逆矩阵：")
sp.pprint(A_inv)
```

#### 3. 伴随矩阵法 `A.adjugate() / det(A)`

公式：$A^{-1} = \dfrac{1}{\det(A)} \cdot \text{adj}(A)$

```python
import sympy as sp

A = sp.Matrix([[2, 1], [1, 1]])
det_A = A.det()
adj_A = A.adjugate()
A_inv = adj_A / det_A
sp.pprint(A_inv)
```

#### 4. 含符号变量矩阵求逆

```python
import sympy as sp

a, b, c, d = sp.symbols("a b c d")
M = sp.Matrix([[a, b], [c, d]])
M_inv = M**-1

sp.pprint(M_inv)
```

输出：
$$\left[\begin{matrix}\dfrac{d}{a d - b c} & - \dfrac{b}{a d - b c}\\- \dfrac{c}{a d - b c} & \dfrac{a}{a d - b c}\end{matrix}\right]$$

#### 关键报错说明

若矩阵奇异（行列式=0），执行 `A.inv()` / `A**-1` 会抛出 `NonInvertibleMatrixError`，此时无逆，只能求伪逆 `A.pinv()`（长方形/奇异矩阵适用）。

#### 最简推荐写法

日常计算直接用：

```python
A = sp.Matrix(...)
A_inv = A.inv()
```

### Rank

#### Sympy 求矩阵秩两种方法

##### 方法 1：直接 `.rank()`（最简单）

```python
import sympy as sp

A = sp.Matrix([[1, 2], [2, 4]])
print(A.rank())  # 输出 1

B = sp.Matrix([[2, 1], [1, 1]])
print(B.rank())  # 输出 2
```

##### 方法 2：先转 rref，数非零行（手动理解原理）

`rref()` 返回 `(rref矩阵, 主元下标列表)`，主元列表长度就是秩：

```python
import sympy as sp

A = sp.Matrix([[1, 2], [2, 4]])
r, pivots = A.rref()
rank_A = len(pivots)
print(rank_A)  # 1
```

##### 符号矩阵也能算秩

```python
import sympy as sp

x = sp.Symbol("x")
M = sp.Matrix([[x, 1], [1, x]])
print(M.rank())
```

##### 配套判断满秩示例

```python
m, n = A.shape
if A.rank() == min(m, n):
    print("满秩")
else:
    print("降秩")
```
