#### 一、题目说明

连续随机变量 \(X\) 的概率密度：
\[
f(x)=
\begin{cases}
2x, & 0<x<1 \\
0, & \text{其他}
\end{cases}
\]
求数学期望 \(E(X)=\displaystyle\int\_{-\infty}^{+\infty}x\,f(x)\,dx\)。

##### 解析计算

\[
E(X)=\int_0^1 x\cdot 2x\,dx
=\int_0^1 2x^2 dx
= \left.\frac{2}{3}x^3\right|\_0^1
=\frac{2}{3}
\]

#### 二、代码实现（3种方式）

##### 1. Python 符号计算（SymPy，精确解析解）

```python
import sympy as sp
sp.init_printing()

x = sp.Symbol("x")
# 先限定 0<x<1，剩下所有情况为0
f = sp.Piecewise( # Piecewise 是 SymPy 库专门用来定义分段函数 的类，直译：分段、分块
    (2*x, (x > 0) & (x < 1)),  # 第一优先级：0<x<1
    (0, True)                   # 其余全部情况：x≤0 或 x≥1
)

EX = sp.integrate(x * f, (x, -sp.oo, sp.oo))
print("E(X) =", EX)
EX
```

输出：`E(X) = 2/3`

##### 2. Python 数值积分（Scipy，数值近似）

```python
from scipy import integrate

def pdf(x):
    if 0 < x < 1:
        return 2 * x
    else:
        return 0

#### 积分 x*pdf(x) 从0到1
EX, err = integrate.quad(lambda t: t * pdf(t), 0, 1)
print(f"E(X) ≈ {EX}")
```

输出：`E(X) ≈ 0.6666666666666666`

##### 3. 蒙特卡洛模拟（随机采样验证）

```python
import numpy as np

#### 逆变换采样：CDF F(x)=x² → 逆函数 x=√u, u~Uniform(0,1)
np.random.seed(42)
N = 1000000
u = np.random.rand(N)
samples = np.sqrt(u)
EX_mc = samples.mean()
print(f"蒙特卡洛估计 E(X) ≈ {EX_mc}")
```

输出会接近 `0.666...`，样本量越大越接近 \(\dfrac{2}{3}\)。

##### 4. R 语言版本

```r
#### 数值积分
f <- function(x) ifelse(x>0 & x<1, 2*x, 0)
EX <- integrate(function(t) t*f(t), 0, 1)
print(EX)
```

```python
from scipy import integrate

# 纯数值函数，可直接被quad调用
def f_num(x):
    return 2 * x if 0 < x < 1 else 0

# 计算 E(X) = ∫ t*f_num(t) dt
EX, err = integrate.quad(lambda t: t * f_num(t), 0, 1)
print(f"数值积分 E(X) = {EX}")
```

#### 三、补充说明

1. 数学期望本质是**加权积分**，权重为概率密度 \(f(x)\)；
2. 符号计算能直接得到精确分数 \(\dfrac{2}{3}\)，数值积分/蒙特卡洛是近似验证；
3. 该分布是 Beta 分布 \(\text{Beta}(2,1)\)，理论期望 \(\dfrac{\alpha}{\alpha+\beta}=\dfrac{2}{3}\)，和计算结果一致。
