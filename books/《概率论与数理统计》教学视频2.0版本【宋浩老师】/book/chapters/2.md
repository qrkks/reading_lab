# 随机变量及其分布

## 概率分布

构成要素：

1. 取值
2. 概率

【例1】

```python
import sympy as sp
from sympy.stats import FiniteRV, P

# 1. 定义符号c，解方程求概率
c = sp.Symbol("c")
eq = sp.Eq(0.2 + c + 0.1 + 0.4, 1)
c_val = sp.solve(eq, c)[0]  # 算出 c = 0.3

# 2. 此时 c_val 是确定数值，才能放进分布字典
X = FiniteRV('X', {0:0.2, 1:c_val, 2:0.1, 3:0.4})

# 求概率
print(P(X <= 2))
print(P(X < 0))
print(P(X>1, X<=2.5)) # 复合条件用逗号即可；数学上用^
print(P(X>=2))
```

## 0 1 分布 二项分布

### SymPy 伯努利（0-1分布）完整用法

SymPy 提供 **`sympy.stats.Bernoulli`** 表示0-1分布（伯努利分布），是标准0/1两点分布：

- 取 1 的概率：$p$
- 取 0 的概率：$1-p$

#### 1. 导入与基础创建

```python
from sympy import symbols, simplify
from sympy.stats import Bernoulli, density, E, cdf, P, variance

### 定义符号参数
p = symbols("p", positive=True)
### 构造伯努利随机变量 X ~ Bernoulli(p)
X = Bernoulli("X", p)
```

#### 2. 概率质量函数 PMF（分布律）

```python
pmf = density(X)
### 代入取值0、1查看概率
print(pmf(0))  # 1 - p
print(pmf(1))  # p
```

输出：

```
1 - p
p
```

#### 3. 常用数字特征

```python
### 期望 E[X] = p
print(E(X))
### 方差 variance(X) = p*(1-p)
print(simplify(variance(X)))
```

输出：

```
p
p*(1 - p)
```

#### 4. 累积分布函数 CDF

```python
F = cdf(X)
print(F(-1))  # x<0 → 0
print(F(0))   # 0≤x<1 → 1-p
print(F(1))   # x≥1 → 1
```

#### 5. 概率计算示例

```python
### 固定 p=0.7
X07 = Bernoulli("X07", 0.7)
print(P(X07 == 1))  # P(X=1)=0.7
print(P(X07 == 0))  # P(X=0)=0.3
print(P(X07 >= 0))  # 1
```

#### 6. 离散采样（随机取值0/1）

```python
from sympy.stats import sample
print(sample(X07))  # 随机输出 0 或 1
```

#### 补充区分

- **Bernoulli**：单次试验，输出 0/1 → 0-1分布
- **Binomial(n,p)**：n次独立伯努利试验，输出成功次数（0~n）

### 探索

```python
import math
import matplotlib.pyplot as plt

def binomial_term(n, k, p, q):
    """计算二项分布中 P(X=k) 的项"""
    # 组合数 C(n, k)
    comb = math.comb(n, k)
    return comb * (p**k) * (q**(n - k))

# 设定参数
n = 10
p = 0.3
q = 1 - p

# 1. 计算所有项的和
terms = []
for k in range(n + 1):
    term = binomial_term(n, k, p, q)
    terms.append(term)

total_sum = sum(terms)
print(f"参数 n={n}, p={p}")
print(f"各项概率之和 = {total_sum:.10f}")
print(f"(p+q)^n = {(p+q)**n:.10f}")

# 2. 画图
plt.figure(figsize=(8, 5))
plt.bar(range(n + 1), terms, color='skyblue')
plt.title(f'二项分布 B(n={n}, p={p}) 的概率质量函数')
plt.xlabel('成功次数 k')
plt.ylabel('概率 P(X=k)')
plt.xticks(range(n + 1))
plt.grid(axis='y', linestyle='--', alpha=0.7)
plt.show()
```

例子：抛硬币 \(n=5\) 次，正面为 “成功”，单次正面概率 \(p=0.5\)
求：恰好出现 2 次正面 的概率

```python
import sympy as sp

n = 5
p = 0.5
q = 1 - p
k = 2

# 计算 P(X=k)
prob = sp.binomial(n, k) * p**k * q**(n - k) # 对，sp.binomial(n, k) 就是专门计算二项式系数 \(\boldsymbol{\dbinom{n}{k}}\)（组合数）。
print(f"P(X={k}) = {prob:.4f}")
```

二项分布

```python
import sympy as sp

n, k = sp.symbols("n k", integer=True, nonnegative=True)
p, g = sp.symbols("p g")
g = 1 - p
expr = sp.binomial(n, k) * p**k * g**(n - k)
print(expr)

# 代数计算：代入数值：n=400, k=5, p=0.02
val = expr.subs({n: 400, k: 5, p: 0.02}).evalf(2)
print("计算结果：", val)

# 0,1分布
# 设单次概率 p=0.6
p_val = 0.6

# ① 求 P(X=1)：n=1, k=1
res1 = expr.subs({n:1, k:1, p:p_val}).evalf(2)
# result；
# evalf 是 Evaluate Floating-point 的缩写。
# Evaluate：求值、计算
# Floating-point：浮点数（日常小数）
# 合起来就是计算为浮点数，专门把 SymPy 符号表达式转成小数结果。
print("P(X=1) =", res1)

# ② 求 P(X=0)：n=1, k=0
res0 = expr.subs({n:1, k:0, p:p_val}).evalf(2)
print("P(X=0) =", res0)

print("抛5次硬币，恰好出现2次正面的概率：", expr.subs({n:5, k:2, p:0.5}).evalf(4))
print(expr.subs({n:5, k:2, p:0.5}).evalf(4)*expr.subs({n:5, k:1, p:0.5}).evalf(4))
```
