## 概率分布

构成要素：

1. 取值
2. 概率

【例 1】

```python
import sympy as sp
from sympy.stats import FiniteRV, P

# 1. 定义符号 c，解方程求概率
c = sp.Symbol("c")
eq = sp.Eq(0.2 + c + 0.1 + 0.4, 1)
c_val = sp.solve(eq, c)[0]  # 算出 c = 0.3

# 2. 此时 c_val 是确定数值，才能放进分布字典
X = FiniteRV('X', {0:0.2, 1:c_val, 2:0.1, 3:0.4})

# 求概率
print(P(X <= 2))
print(P(X < 0))
print(P(X>1, X<=2.5)) # 复合条件用逗号即可；数学上用^
print(P(X>=2))
```

## 0 1 分布 二项分布

### SymPy 伯努利（0-1 分布）完整用法

SymPy 提供 **`sympy.stats.Bernoulli`** 表示 0-1 分布（伯努利分布），是标准 0/1 两点分布：

- 取 1 的概率：$p$
- 取 0 的概率：$1-p$

#### 1. 导入与基础创建

```python
from sympy import symbols, simplify, Eq
from sympy.stats import Bernoulli, density, E, cdf, P, variance

### 定义符号参数
p = symbols("p", positive=True)
### 构造伯努利随机变量 X ~ Bernoulli(p)
X = Bernoulli("X", p)
```

#### 2. 概率质量函数 PMF（分布律）

```python
pmf = density(X)
### 代入取值 0、1 查看概率
print(pmf(0))  # 1 - p
print(pmf(1))  # p

print(P(Eq(X, 0)))  # 1 - p
print(P(Eq(X, 1)))  # p
# 只有 == 有坑，> < >= <= 全都能正常直接写，底层根源是 Python 语法对「等于」和「不等号」的处理逻辑完全分开。

print(P(X < 1))  # 1 - p
```

输出：

```
1 - p
p
```

#### 3. 常用数字特征

```python
### 期望 E[X] = p
print(E(X))
### 方差 variance(X) = p*(1-p)
print(variance(X))
print(simplify(variance(X)))
```

输出：

```
p
p*(1 - p)
```

#### 4. 累积分布函数 CDF

```python
F = cdf(X)
print(F)

# print(F[-1])  # x<0 → 0
print(F[0])   # 0≤x<1 → 1-p
print(F[1])   # x≥1 → 1
```

#### 5. 概率计算示例

```python
### 固定 p=0.7
X07 = Bernoulli("X07", 0.7)
print(P(X07 == 1))  # P(X=1)=0.7
print(P(X07 == 0))  # P(X=0)=0.3
print(P(X07 >= 0))  # 1
```

#### 6. 离散采样（随机取值 0/1）

```python
from sympy.stats import sample

```

### 探索

```python
import math
import matplotlib.pyplot as plt

def binomial_term(n, k, p, q):
    """计算二项分布中 P(X=k) 的项"""
    # 组合数 C(n, k)
    comb = math.comb(n, k)
    return comb * (p**k) * (q**(n - k))

# 设定参数
n = 10
p = 0.3
q = 1 - p

# 1. 计算所有项的和
terms = []
for k in range(n + 1):
    term = binomial_term(n, k, p, q)
    terms.append(term)

total_sum = sum(terms)
print(f"参数 n={n}, p={p}")
print(f"各项概率之和 = {total_sum:.10f}")
print(f"(p+q)^n = {(p+q)**n:.10f}")

# 2. 画图
plt.figure(figsize=(8, 5))
plt.bar(range(n + 1), terms, color='skyblue')
plt.title(f'二项分布 B(n={n}, p={p}) 的概率质量函数')
plt.xlabel('成功次数 k')
plt.ylabel('概率 P(X=k)')
plt.xticks(range(n + 1))
plt.grid(axis='y', linestyle='--', alpha=0.7)
plt.show()
```

例子：抛硬币 \(n=5\) 次，正面为“成功”，单次正面概率 \(p=0.5\)
求：恰好出现 2 次正面 的概率

```python
import sympy as sp

n = 5
p = 0.5
q = 1 - p
k = 2

# 计算 P(X=k)
prob = sp.binomial(n, k) * p**k * q**(n - k) # 对，sp.binomial(n, k) 就是专门计算二项式系数 \(\boldsymbol{\dbinom{n}{k}}\)（组合数）。
print(f"P(X={k}) = {prob:.4f}")
```

二项分布

```python
import sympy as sp

n, k = sp.symbols("n k", integer=True, nonnegative=True)
p, g = sp.symbols("p g")
g = 1 - p
expr = sp.binomial(n, k) * p**k * g**(n - k)
print(expr)

# 代数计算：代入数值：n=400, k=5, p=0.02
val = expr.subs({n: 400, k: 5, p: 0.02}).evalf(2)
print("计算结果：", val)

# 0,1 分布
# 设单次概率 p=0.6
p_val = 0.6

# ① 求 P(X=1)：n=1, k=1
res1 = expr.subs({n:1, k:1, p:p_val}).evalf(2)
# result；
# evalf 是 Evaluate Floating-point 的缩写。
# Evaluate：求值、计算
# Floating-point：浮点数（日常小数）
# 合起来就是计算为浮点数，专门把 SymPy 符号表达式转成小数结果。
print("P(X=1) =", res1)

# ② 求 P(X=0)：n=1, k=0
res0 = expr.subs({n:1, k:0, p:p_val}).evalf(2)
print("P(X=0) =", res0)

print("抛 5 次硬币，恰好出现 2 次正面的概率：", expr.subs({n:5, k:2, p:0.5}).evalf(4))
```


#### 二、核心分类清单

##### 1. 分布构造函数

`Binomial(var_name, n, p)`
作用：创建二项随机变量 X ~ B(n,p)
示例：`X = Binomial("X", n, p)`

##### 2. 分布核心函数（概率计算）

1. `density(X)`
   概率质量函数 PMF：$P(X=k)$
   使用：`density(X)(k)`
2. `cdf(X, x)`
   累积分布函数：$P(X \le x)$
3. `quantile(X, q)`
   分位数，返回最小 k 满足 $P(X\le k)\ge q$，q∈[0,1]

##### 3. 概率、期望算子（替代旧版 mean/variance/std）

1. `P(条件表达式)`
   计算区间/等式概率
   例：`P(X==3)、P(X>5)、P(2<=X<=7)`
2. `E(表达式)`
   期望算子，所有数字特征都靠它计算

##### 4. 随机采样函数

`sample(X)`
单次抽取一个服从二项分布的随机数；批量采样用列表推导

##### 5. 配套底层函数（二项分布专用组合数）

`binomial(a, b)`
计算组合数 $\binom{a}{b}$，PMF 的核心组成部分，从 sympy 顶层导入

#### 三、用 E 算子实现各类数字特征（替代废弃 mean/variance）

设 `mu = E(X)`，`var = E(X**2) - mu**2`

1. 期望（原 mean(X)）：`E(X)`
2. 方差（原 variance(X)）：`E(X**2) - E(X)**2`
3. 标准差（原 std(X)）：`sqrt(E(X**2) - E(X)**2)`
4. 偏度 skewness：`E((X - mu)**3) / var**(3/2)`
5. 超额峰度 kurtosis：`E((X - mu)**4) / var**2 - 3`

#### 四、完整可运行演示代码


### [7]

纯数学做

```python
import sympy as sp

# 定义概率符号 p
p = sp.Symbol('p', domain=(0,1))

# 已知条件 P(X>=1) = 5/9，X~B(2,p)
equation = sp.Eq(1 - (1 - p)**2, sp.Rational(5, 9))
# \(X\ge1\) 是「至少成功 1 次」，它的反面就是「一次都没成功」\(X=0\)
# P(X≥1)=1−P(一次都不成功)=1−(1−p)**2

# 解出 p
p_sol = sp.solve(equation, p)[0]
print(f"求得 p = {p_sol}")

# 计算 P(Y >= 1), Y~B(3,p)
prob_Y_ge1 = 1 - (1 - p_sol)**3
print(f"P(Y ≥ 1) = {prob_Y_ge1}")
```

用 Binomial 计算

```python
from sympy import Symbol, Eq, Rational, solve
from sympy.stats import Binomial, P

# 1. 定义参数 p
p = Symbol("p", positive=True)

# 2. 定义二项分布随机变量 X~B(2,p)，Y~B(3,p)
X = Binomial("X", 2, p)
Y = Binomial("Y", 3, p)

# 3. 列方程 P(X ≥ 1) = 5/9，直接写 X >= 1，不用 Ge
cond_eq = Eq(P(X >= 1), Rational(5, 9))
print("方程：", cond_eq)

# 4. 解方程求 p
p_sol = solve(cond_eq, p)[0]
print("解出 p =", p_sol)

# 5. 计算 P(Y ≥ 1)
prob_Y = P(Y >= 1).subs(p, p_sol)
print("P(Y ≥ 1) =", prob_Y)
```

## 泊松分布 几何分布 超几何分布

### 泊松分布 $X\sim P(\lambda)$ 全套核心函数（定义、公式、Sympy 代码一并给你）

参数 $\lambda>0$：单位区间内事件**平均发生次数**

#### 一、概率质量函数 PMF（单点概率）

描述恰好发生 $k$ 次的概率，离散分布，$k=0,1,2,\dots$

$$
P(X=k) = f(k;\lambda) = \frac{\lambda^k e^{-\lambda}}{k!}
$$

- $e$：自然常数；$k!$：$k$ 的阶乘
- 含义：比如 $\lambda=3$，恰好发生 2 次的概率就是 $\dfrac{3^2 e^{-3}}{2!}$

##### Sympy 计算示例

```python
from sympy import symbols, exp, factorial, Eq
from sympy.stats import Poisson, P

λ = symbols("λ", positive=True)
X = Poisson("X", λ)

### 两种方式求 P(X=2)
print(P(Eq(X, 2)))
### 手动展开 PMF 验证
pmf_k2 = λ**2 * exp(-λ) / factorial(2)
print(pmf_k2)
```

#### 二、累积分布函数 CDF（至多发生 k 次的概率）

$$
F(k;\lambda)=P(X\le k)=\sum_{i=0}^k \frac{\lambda^i e^{-\lambda}}{i!}
$$

- 含义：事件发生次数不超过 $k$ 的总概率
- 无简洁闭式，只能求和；Sympy 可直接计算：

```python
from sympy.stats import cdf
F = cdf(X)
print(F(3))  ### P(X ≤ 3)
```

#### 三、生存函数（互补累积，至少发生 k 次）

$$
S(k;\lambda)=P(X\ge k)=1-F(k-1)
$$

例：$P(X\ge1)=1-P(X=0)=1-e^{-\lambda}$

符号计算：

```python
from sympy import symbols, exp, factorial, Sum
from sympy.stats import Poisson, cdf

λ = symbols("λ", positive=True)
t = symbols("t", integer=True, nonnegative=True)
X = Poisson("X", λ)

# 先拿到 CDF 函数对象
F = cdf(X)
# 生存函数 S(t) = 1 - CDF(t)，纯代数运算，不触发概率不等式解析
S = 1 - F(t)
print("符号生存函数 S(t) =", S)

# 代入数值 t=0、t=2 验证
print("S(0) =", S.subs(t, 0))
print("S(2) =", S.subs(t, 2))
```

代数计算：

```python
from sympy import symbols, exp, factorial
from sympy.stats import Poisson, P, cdf

λ = symbols("λ", positive=True)
X = Poisson("X", λ)

# 阈值写数字，完全正常
print("S(0)=P(X>0) =", P(X > 0))
print("S(2)=P(X>2) =", P(X > 2))
```

#### 四、期望（均值函数）

$$
E(X) = \lambda
$$

泊松独有的特点：**均值 = λ**

```python
from sympy.stats import E
print(E(X)) ### 输出 λ
```

#### 五、方差函数

$$
D(X) = \lambda
$$

唯一一种「期望 = 方差」的常用离散分布

```python
from sympy.stats import variance
print(variance(X)) ### 输出 λ
```

#### 六、母函数系列（特征函数、矩母函数、概率生成函数）

##### 1. 概率生成函数 PGF（离散分布专用）

$$
G(t)=E(t^X)=\sum_{k=0}^\infty t^k \cdot \frac{\lambda^k e^{-\lambda}}{k!}=e^{\lambda(t-1)}
$$

##### 2. 矩母函数 MGF

$$
M(t)=E(e^{tX})=e^{\lambda(e^t -1)}
$$

##### 3. 特征函数 CF（傅里叶变换，统计理论常用）

$$
\phi(t)=E(e^{itX})=e^{\lambda(e^{it}-1)}
$$

#### 七、常用衍生计算函数（做题高频）

1. 一次都不发生：$P(X=0)=e^{-\lambda}$
2. 至少发生 1 次：$P(X\ge1)=1-e^{-\lambda}$
3. 恰好 1 次：$P(X=1)=\lambda e^{-\lambda}$

#### 八、和二项分布函数对比关键点

- 二项：2 参数 $n,p$，PMF 带组合数 $\mathrm C_n^k$，方差 $np(1-p)<$ 均值
- 泊松：仅 1 参数 $\lambda$，PMF 带阶乘 $k!$，方差 = 均值

#### 完整泊松符号计算代码（可直接运行）

```python
from sympy import symbols, exp, factorial, Eq
from sympy.stats import Poisson, P, E, variance, cdf

### 定义参数λ
λ = symbols("λ", positive=True)
### 泊松随机变量 X~Poisson(λ)
X = Poisson("X", λ)

### 1. PMF 单点概率 P(X=2)
print("P(X=2) =", P(Eq(X, 2)))

### 2. CDF P(X ≤ 3)
cdf_func = cdf(X)
print("P(X ≤ 3) =", cdf_func(3))

### 3. 期望、方差
print("E(X) =", E(X))
print("Var(X) =", variance(X))

### 4. 至少发生 1 次 P(X ≥ 1)
print("P(X ≥ 1) =", P(X >= 1))
```
