## 变异性与分数分布

### 表 3-1 对分数分布描述的样例

```python
u = [110, 120, 100, 90, 130, 110]

sum_u = sum(u)
print("总分：", sum_u)

mean_u = sum_u / len(u)
print("平均分：", mean_u)

deviation = [x - mean_u for x in u]
print("离均差：", deviation)

# 平方和
squared_deviation = [d**2 for d in deviation]
sum_squared_deviation = sum(squared_deviation)
print("离均差平方：", squared_deviation)
print("离均差平方和 (SS)：", sum_squared_deviation)

# 方差
variance = sum_squared_deviation / len(u)
print("方差：", variance)

# 无偏方差
unbiased_variance = sum_squared_deviation / (len(u) - 1)
print("无偏方差：", unbiased_variance)

# 标准差
std_dev = variance**0.5
print("标准差：", std_dev)

# 无偏标准差
unbiased_std_dev = unbiased_variance**0.5
print("无偏标准差：", unbiased_std_dev)
```

#### 一、代码逻辑正确性逐条核对

##### 1. 计算逻辑完全正确

###### 总体方差（有偏方差，除以 $n$）

```python
variance = sum_squared_deviation / len(u)
```

对应总体方差公式：
$$\sigma^2 = \frac{\sum(x-\mu)^2}{N}$$
适用于数据是**全部总体**的场景。

###### 样本无偏方差（除以 $n-1$）

```python
unbiased_variance = sum_squared_deviation / (len(u) - 1)
```

对应样本方差公式：
$$s^2 = \frac{\sum(x-\bar{x})^2}{n-1}$$
心理测量、实验统计中绝大多数情况用这个（抽样推断总体，校正自由度偏差）。

###### 标准差

- `std_dev`：总体标准差 $\sigma$
- `unbiased_std_dev`：样本标准差 $s$
  开平方操作写法 `**0.5` 数学等价，无错误。

###### 前置步骤

总分、均值、离均差、离均差平方、平方和 SS 推导链条完全标准，公式无错。

##### 2. 唯一小瑕疵：打印命名容易混淆

```python
print("方差：", variance)  # 准确叫法：总体方差 / 有偏方差
print("无偏方差：", unbiased_variance)  # 准确叫法：样本方差
print("标准差：", std_dev)  # 总体标准差
print("无偏标准差：", unbiased_std_dev)  # 样本标准差
```

建议修改打印文字，避免统计概念混淆（适配心理测量学术规范）。

##### 3. 代入数值验算（精确结果）

###### 计算结果对照

1. 总分：660
2. 平均分：110
3. 离均差：`[0, 10, -10, -20, 20, 0]`
4. 离均差平方：`[0, 100, 100, 400, 400, 0]`
5. 平方和 SS：1500
6. 总体方差（除以 6）：250
7. 样本无偏方差（除以 5）：300
8. 总体标准差：$\sqrt{250} \approx 15.8114$

#### 二、优化后的完整规范代码

```python
u = [110, 120, 100, 90, 130, 110]

sum_u = sum(u)
print("总分：", sum_u)

mean_u = sum_u / len(u)
print("平均分：", mean_u)

deviation = [x - mean_u for x in u]
print("离均差 (deviation score)：", deviation)

# 离均差平方、平方和 SS
squared_deviation = [d**2 for d in deviation]
sum_squared_deviation = sum(squared_deviation)
print("离均差平方 (squared deviation)：", squared_deviation)
print("离均差平方和 SS：", sum_squared_deviation)

# 1. 总体方差/标准差（数据为完整总体，除以 N）
pop_variance = sum_squared_deviation / len(u)
pop_std = pop_variance**0.5
print("\n【总体统计量】")
print("总体方差 σ²：", pop_variance)
print("总体标准差 σ：", pop_std)

# 2. 样本无偏方差/标准差（抽样数据，除以 n-1，心理测量常用）
sample_variance = sum_squared_deviation / (len(u) - 1)
sample_std = sample_variance**0.5
print("\n【样本无偏统计量（科研常用）】")
print("样本方差 s²：", sample_variance)
print("样本标准差 s：", sample_std)
```

#### 三、关键使用场景提醒（心理测量）

1. 若这 6 个分数是**全部被试总体**（只关心这 6 个人）：用 `pop_variance`、`pop_std`；
2. 若这 6 个是**抽样样本**，用来推断更大人群的离散程度（量表信效度、实验数据分析）：必须用除以 $n-1$ 的样本方差 `sample_variance`。

### 表 3-2 计算协方差和相关的样例

```python
# IQ、GPA 原始数据
X = [110, 120, 100, 90, 130, 110]
Y = [2.6, 3, 2.5, 1.5, 3.2, 3.4]

mean_X = sum(X) / len(X)
mean_Y = sum(Y) / len(Y)

# 计算两组离均差
deviation_X = [x - mean_X for x in X]
deviation_Y = [round(y - mean_Y, 1) for y in Y]

# 计算交叉乘积 cross product
cross_product = [dX * dY for dX, dY in zip(deviation_X, deviation_Y)]

# 修正打印代码
print("IQ 平均分：", round(mean_X, 2))
print("GPA 平均分：", round(mean_Y, 2))
print("IQ 离均差：", deviation_X)
print("GPA 离均差：", deviation_Y)
print("离均差交叉乘积：", list(map(int, cross_product)))

# X 离均差平方和 SS_X
SS_X = sum([d**2 for d in deviation_X])
print("IQ 离均差平方和 SS_X =", SS_X)
# Y 离均差平方和 SS_Y
SS_Y = sum([d**2 for d in deviation_Y])
print("GPA 离均差平方和 SS_Y =", SS_Y)

# 交叉乘积和 SP
SP = sum(cross_product)
print("交叉乘积和 SP =", SP)

# 总体协方差（完整全部观测，除以 n）
pop_cov = SP / len(X)
print("总体协方差 Cov_pop(X,Y) =", round(pop_cov, 2))

# 样本无偏协方差（抽样样本，自由度 n-1，统计分析通用）
sample_cov = SP / (len(X) - 1)
print("样本无偏协方差 Cov_sample(X,Y) =", round(sample_cov, 2))

# X 标准差
pop_std_X = (SS_X / len(X)) ** 0.5
sample_std_X = (SS_X / (len(X) - 1)) ** 0.5
print("IQ 总体标准差 σ_X =", round(pop_std_X, 2))
print("IQ 样本标准差 s_X =", round(sample_std_X, 2))

# Y 标准差
pop_std_Y = (SS_Y / len(Y)) ** 0.5
sample_std_Y = (SS_Y / (len(Y) - 1)) ** 0.5
print("GPA 总体标准差 σ_Y =", round(pop_std_Y, 2))
print("GPA 样本标准差 s_Y =", round(sample_std_Y, 2))

# 计算相关系数 r
# 总体相关系数
pop_r = pop_cov / (pop_std_X * pop_std_Y)
# 样本相关系数
sample_r = sample_cov / (sample_std_X * sample_std_Y)
print("总体相关系数 r =", round(pop_r, 4))
print("样本相关系数 r =", round(sample_r, 4))
```

### 计算协方差矩阵

```python
import numpy as np

X = np.array([110, 120, 100, 90, 130, 110])
Y = np.array([2.6, 3, 2.5, 1.5, 3.2, 3.4])
print(np.hstack([X, Y]))
# [110.  120.  100.   90.  130.  110.    2.6   3.    2.5   1.5   3.2   3.4]
print(np.hstack([X, Y]).shape)  # (12,)

# 1. 数学标准：列向量拼成 6 行 2 列
M = np.column_stack([X, Y])
print("标准样本矩阵 n×p：", M.shape)  # (6, 2)
print("样本矩阵 M：\n", M)

# 2. vstack 得到 p×n，适配 np.cov
data_v = np.vstack([X, Y])
print("vstack 结果 p×n：", data_v.shape)  # (2, 6)
print("vstack 结果 data_v：\n", data_v)

# 3. 两者转置等价
print(np.allclose(data_v, M.T))  # True

# 4. 计算协方差矩阵
# 默认 bias=False，除以 n-1，样本无偏协方差矩阵
cov_matrix = np.cov(data_v, bias=False)
# 老科学计算库（MATLAB、老 Numeric、np.cov 默认）
# 反着来：一行一个变量，一列一个观测。
print("样本无偏协方差矩阵：\n", cov_matrix)

print(
    "样本无偏协方差矩阵（竖向）计算结果：\n",
    np.cov(
        M,
        rowvar=False,  # 每列是一个变量，每行是一个观测
    ),
)
```
